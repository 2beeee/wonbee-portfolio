# 대수 함정 체크리스트 — 시험지 받기 직전 5분 스캔용

> 2022 개정 대수 범위(지수로그 · 지수로그함수 · 삼각함수)에서 **실제로 점수 깎이는 실수 패턴만** 모았다. 시험 시작 전 2분 + 풀면서 의심스러울 때 한 번 더.

---

## 🚨 최다 실점 Top 10 (반드시 체크)

- [ ] **1. 밑 조건 무시.** `log_a N`에서 a > 0, a ≠ 1. 진수 N > 0.
  - 특히 밑이 문자 (a² − 3a + 3 꼴)일 때 **a > 0 ∧ a ≠ 1** 두 조건 **동시에** 점검.

- [ ] **2. 0 < a < 1 에서 부등호 안 뒤집음.**
  - 지수부등식: aˣ > aʸ → a > 1이면 x > y, **0 < a < 1이면 x < y**.
  - 로그부등식도 동일.

- [ ] **3. 로그방정식 풀고 정의역 검산 누락.**
  - log_a X = log_a Y → X = Y 풀이 후, **X > 0 ∧ Y > 0** 다시 확인.
  - 놓치면 외부해 섞여 들어감.

- [ ] **4. √(a²) = a 실수.** 정답은 **|a|**. 부호 모르면 절댓값으로.

- [ ] **5. 치환 조건 망각.**
  - aˣ = t → **t > 0**.
  - log x = t → t는 모든 실수 가능하지만 **x > 0** 조건 별개.
  - 치환 후 이차식 해 중 조건 위배하는 것 기각.

- [ ] **6. 사분면 부호 혼동.**
  - 2사분면: sin **+**, cos **−**, tan **−**.
  - 3사분면: sin **−**, cos **−**, tan **+**.
  - 4사분면: sin **−**, cos **+**, tan **−**.
  - **sin²+cos²=1** 에서 cos = ±√(1−sin²) — 부호는 **사분면에서** 가져와라.

- [ ] **7. 주기 공식에서 |b| 놓침.**
  - y = a·sin(bx + c) + d 의 주기는 **2π/|b|**. b = −3 이어도 주기는 2π/3.
  - tan 주기는 **π/|b|** (2π 아님).

- [ ] **8. 평행이동 부호 반대로.**
  - y = sin(bx + c) = sin(b(x + c/b)) → **x축으로 −c/b** 만큼 이동.
  - 괄호 밖으로 뺄 때 부호 잊지 마라.

- [ ] **9. 지표·가수 혼동.**
  - log N 의 **가수는 항상 0 ≤ α < 1**.
  - log 0.003 = −3 + 0.4771, 지표는 **−3** (−2.5229 아님).

- [ ] **10. 지수법칙 — 밑이 음수일 때.**
  - (−2)^(1/2) 은 실수에서 **정의 안 됨**.
  - (−2)³·(−2)² = (−2)⁵ 는 OK, 하지만 (−2)^(3.5) 는 금지.

## ⚠️ 놓치기 쉬운 경계/예외

- [ ] **0 < a < 1 분기 처리.** 지수·로그함수 증감 반대.
- [ ] **a > 1 vs a > 0 구분.** a > 0 은 a < 1도 포함 → 분기 필요.
- [ ] **지수함수는 y > 0** — y = 0 해 절대 없음. y = −1 같은 방정식이면 해 없음.
- [ ] **로그함수는 x > 0** — 정의역 먼저.
- [ ] **tan θ 는 π/2 + nπ 에서 정의 안 됨.**
- [ ] **부채꼴 공식 l = rθ** — θ가 **라디안**일 때만. 도 단위 혼용 금지.
- [ ] **일반각 변환** — 1080° = 360° × 3 + 0° → 기본각 0°. 부호 방향 확인.

## ⚠️ 서논술 감점 포인트

서논술 25점은 부분점수 전략. 다음을 반드시 써야 감점 안 됨:

- [ ] **밑 조건 명시** ("단, a > 0, a ≠ 1 이고 x > 0").
- [ ] **경우 나누기** — "a > 1 일 때"와 "0 < a < 1 일 때" **별도 서술**.
- [ ] **치환 조건 명시** — "t = aˣ (t > 0) 로 놓으면".
- [ ] **정의역 확인 단계** — 최종 해에 "이 중 x = ... 는 진수 조건을 만족" 한 줄.
- [ ] **사분면 명시** — "θ는 제2사분면 각이므로 cos θ < 0" 한 줄.
- [ ] **마지막 답** — 원문항이 요구하는 형태 ("최댓값", "해의 개수", "가능한 x의 합") 정확히.

## ⚠️ 문제 문장 읽기 함정

- [ ] **"모든 실수 x에 대해"** vs **"어떤 실수 x에 대해"** — ∀ vs ∃. 푸는 방향 정반대.
- [ ] **"최솟값이 존재할 필요충분조건"** 같은 메타 질문 — 경계값 대입해 불연속 체크.
- [ ] **"해가 두 개"** vs **"서로 다른 해가 두 개"** — 중근 포함 여부.
- [ ] **"모든 근의 합"** vs **"곱"** — 근과 계수 관계인지 직접 계산인지.
- [ ] **그림과 글 동시 제공** — 글로 주어진 조건에 **그림에 없는 조건 숨어있음**.
- [ ] **단위** — 문제에서 ° 쓰는데 답 요구할 때 rad 인지 확인. 대부분 라디안.

## ⚠️ 계산 실수 체크

- [ ] **log 2 + log 3 = log 6**. log 2 · log 3 ≠ log 6.
- [ ] **log(a/b) = log a − log b**. log(b/a) 로 뒤집어 쓰지 마라.
- [ ] **aˣ⁺ʸ = aˣ · aʸ** (곱). **a^(xy)** 와 구분.
- [ ] **(aˣ)ʸ = a^(xy)** (곱), **aˣ⁺ʸ 아님**.
- [ ] **ⁿ√aⁿ** — n 짝수 → |a|, n 홀수 → a.
- [ ] **a⁻ˣ = 1/aˣ** — 음수 지수 제거할 때 분모로.
- [ ] **sin(π/6) = 1/2** (cos 아님), **cos(π/6) = √3/2**. 순서 헷갈리지 마라.
- [ ] **sin 2π = 0** (1 아님), **cos 2π = 1**.

## 🎯 시험장 체크 루틴 (문제 풀기 전)

시험지 받고 **첫 2분**에 할 것:

1. 전체 문제 훑어보며 **킬러로 보이는 문제 별표** (보통 18~20, 서논술 25점 중 마지막).
2. 쉬운 문제부터 (보통 1~10번) **15분 안에** 끝낸다는 계획.
3. 각 문제 옆에 유형 번호 (`02_pattern_playbook.md` 참조) 작게 표기해두면 다시 볼 때 빠름.
4. **시계 보고 목표 시간 설정** — 35분 지나도 서논술 안 건드렸으면 일단 빈 칸 → 아무거나 부분점수 확보.
