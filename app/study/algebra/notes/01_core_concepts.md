# 대수 · 핵심 개념·공식 총정리

> 교과서(미래엔 황선욱) p.10–97 + 유인물 기준. 각 항목마다 **정의 → 공식 → 사용조건 → 자주 틀리는 포인트** 4박자.

---

## I-1. 지수와 로그

### ① 거듭제곱근 (교과서 p.12–17)

**정의.** 실수 *a*와 자연수 *n*(≥2)에 대해, *xⁿ = a*를 만족하는 *x*를 *a*의 *n*제곱근이라 한다.

**핵심 결과 (반드시 암기):**

| *n* 의 성질 | *a* > 0 | *a* = 0 | *a* < 0 |
|---|---|---|---|
| *n*이 **홀수** | 실수근 1개 | 0 | 실수근 1개 (음수) |
| *n*이 **짝수** | 실수근 2개 (±) | 0 | **실수근 없음** |

실수 범위에서 존재할 때, *a*의 양의 *n*제곱근 하나를 **ⁿ√a** 로 쓴다.

**연산 공식 (a, b > 0, m·n은 자연수):**
- ⁿ√a · ⁿ√b = ⁿ√(a·b)
- ⁿ√a / ⁿ√b = ⁿ√(a/b)
- (ⁿ√a)ᵐ = ⁿ√(aᵐ)
- ᵐ√(ⁿ√a) = ᵐⁿ√a
- ⁿᵖ√(aᵐᵖ) = ⁿ√(aᵐ)   ← **약분 가능 규칙**

**사용조건.** 지수 공식을 자유롭게 쓰려면 *a* > 0 가정이 편함. *a* < 0일 때는 *n*이 홀수인지 확인 후 부호 따로.

**자주 틀리는 포인트.**
- **√(a²) = |a|** 이지 *a*가 아니다. 부호 확인 필수.
- *a* < 0, *n* 짝수 → ⁿ√a 는 실수에서 **정의 안 됨**.
- ⁿ√((-3)²) ≠ -3. 제곱 먼저 → √9 = 3.

---

### ② 지수의 확장 (교과서 p.18–25)

**정의.** *a* > 0이고 *m*이 정수, *n*이 자연수일 때
- a⁰ = 1
- a⁻ⁿ = 1/aⁿ
- a^(m/n) = ⁿ√(aᵐ)
- 실수 지수: a^x (a > 0)는 모든 실수 x에 대해 정의됨.

**지수법칙 (a > 0, b > 0, x, y 실수):**
1. aˣ · aʸ = a^(x+y)
2. aˣ / aʸ = a^(x−y)
3. (aˣ)ʸ = a^(xy)
4. (ab)ˣ = aˣ · bˣ
5. (a/b)ˣ = aˣ/bˣ

**사용조건.** **a, b > 0** 일 때만 지수법칙을 자유롭게 쓸 수 있다. a ≤ 0이면 유리수 지수에 한해 신중히.

**자주 틀리는 포인트.**
- **밑이 음수**이면 지수법칙 함부로 쓰지 마라. 예: (-2)^(1/2) 은 실수 아님.
- a^(m/n) = (ⁿ√a)ᵐ = ⁿ√(aᵐ) — 두 해석이 같으려면 **a > 0** 필수.
- a^0 = 1 인데 **a ≠ 0** 조건 잊기 쉬움.

---

### ③ 로그 (교과서 p.26–32)

**정의.** *a* > 0, *a* ≠ 1, *N* > 0 일 때, aˣ = N 의 유일한 해 *x*를 **log_a N** 이라 한다.

즉 **log_a N = x ⇔ aˣ = N**.

**밑 조건 (무조건):**
- 밑 *a*: **a > 0 그리고 a ≠ 1**
- 진수 *N*: **N > 0**

**로그 성질 (a > 0, a ≠ 1, M, N > 0, k 실수):**
1. log_a 1 = 0
2. log_a a = 1
3. log_a (MN) = log_a M + log_a N
4. log_a (M/N) = log_a M − log_a N
5. log_a Mᵏ = k · log_a M
6. **밑 변환 공식:** log_a N = log_c N / log_c a   (c > 0, c ≠ 1)
7. log_(aᵏ) N = (1/k) · log_a N
8. log_a N · log_N a = 1  (log_a N = 1/log_N a)
9. a^(log_a N) = N
10. a^(log_b c) = c^(log_b a)  ← **지수·로그 교환 공식**

**자주 틀리는 포인트.**
- log_a M² ≠ (log_a M)² — 거듭제곱이 **진수에 붙었는지 전체에 붙었는지** 구분.
- log_a (M+N) ≠ log_a M + log_a N — 합의 로그는 풀리지 않는다.
- **밑 조건**을 반드시 확인. 문제에서 a를 변수로 주면 "a > 0, a ≠ 1"을 별도로 명시·검토.
- log_a Mᵏ = k log_a M 은 **M > 0** 조건 필요. M < 0일 때 제곱해서 양수 만들고 쓰는 함정 있음.

---

### ④ 상용로그 (교과서 p.33–41)

**정의.** 밑이 10인 로그. **log N = log₁₀ N**.

**상용로그의 분해:**
- log N = 정수부분 (**지표** n) + 소수부분 (**가수** α, 0 ≤ α < 1)
- N이 정수부분 k자리수 ⇔ 지표 = k − 1
- 0 < N < 1 이고 소수 k째 자리에서 처음으로 0이 아닌 숫자가 나오면 지표 = −k

**자주 쓰이는 값 (암기):**
- log 2 ≈ 0.3010
- log 3 ≈ 0.4771
- log 7 ≈ 0.8451
- log 5 = log(10/2) = 1 − log 2 ≈ 0.6990

**자주 틀리는 포인트.**
- 지표·가수 정의에서 **가수는 항상 0 이상 1 미만**. 음수 가수 금지.
- log 0.003 의 지표는 −3 (가수는 양수). "−2.52..." 식으로 쪼개면 틀린다.

## I-2. 지수함수와 로그함수

### ⑤ 지수함수 y = aˣ (a > 0, a ≠ 1) (교과서 p.44–50)

**그래프 특성:**

| 성질 | a > 1 | 0 < a < 1 |
|---|---|---|
| 증감 | **증가** | **감소** |
| 점근선 | x축 (y=0) | x축 (y=0) |
| 반드시 지나는 점 | (0, 1), (1, a) | (0, 1), (1, a) |
| 치역 | y > 0 | y > 0 |
| 정의역 | 모든 실수 | 모든 실수 |

**평행·대칭 변환 (y = aˣ 기준):**
- y = aˣ⁻ᵖ + q : x축으로 p, y축으로 q 평행이동
- y = a⁻ˣ : y축 대칭 (= (1/a)ˣ)
- y = −aˣ : x축 대칭
- y = −a⁻ˣ : 원점 대칭

**자주 틀리는 포인트.**
- 지수함수는 **항상 양수**. y ≤ 0 해는 절대 없음.
- (0, 1) 통과는 **a^0 = 1** 원리. 평행이동 하면 달라짐.

---

### ⑥ 로그함수 y = log_a x (a > 0, a ≠ 1) (교과서 p.47–51)

**그래프 특성:**

| 성질 | a > 1 | 0 < a < 1 |
|---|---|---|
| 증감 | **증가** | **감소** |
| 점근선 | y축 (x=0) | y축 (x=0) |
| 반드시 지나는 점 | (1, 0), (a, 1) | (1, 0), (a, 1) |
| 정의역 | x > 0 | x > 0 |
| 치역 | 모든 실수 | 모든 실수 |

**지수함수와의 관계:** y = aˣ 와 y = log_a x 는 **직선 y = x 에 대해 대칭**.

**변환:**
- y = log_a (x − p) + q : x, y축 평행이동
- y = log_(1/a) x = −log_a x : x축 대칭

**자주 틀리는 포인트.**
- 로그함수는 **x > 0** 에서만 정의. 정의역 확인 필수.
- y = log_a (x−3) 의 정의역은 **x > 3**, 점근선은 **x = 3**.
- (1, 0) 통과는 평행이동 하면 사라진다 — 헷갈리지 마라.

---

### ⑦ 지수·로그 방정식과 부등식 (교과서 p.52–61)

**지수방정식 해법 4가지:**
1. **밑을 같게.** aˣ = aʸ ⇒ x = y (a > 0, a ≠ 1)
2. **지수를 같게.** aˣ = bˣ ⇒ a = b (지수가 0 아닐 때) 또는 x = 0.
3. **aˣ = t 치환.** 이차·삼차식으로 환원. **t > 0** 조건 필수.
4. **양변에 로그.** 적절한 밑의 로그를 씌워 일차식으로.

**지수부등식:**
- aˣ > aʸ
  - a > 1 ⇒ x > y   (부등호 유지)
  - 0 < a < 1 ⇒ x < y   (**부등호 뒤집힘**)

**로그방정식 해법:**
1. **진수 비교.** log_a X = log_a Y ⇒ X = Y, **X > 0, Y > 0** 동시에 확인.
2. **로그 = 상수.** log_a X = k ⇒ X = aᵏ.
3. **log X = t 치환.** 이차식화.
4. **밑을 통일.** 밑 변환 공식으로 정리.

**로그부등식:**
- log_a X > log_a Y
  - a > 1 ⇒ X > Y (X, Y > 0)
  - 0 < a < 1 ⇒ X < Y (X, Y > 0) **부등호 뒤집힘**
- 진수 양수 조건 **반드시** 교집합.

**자주 틀리는 포인트.**
- 0 < a < 1 일 때 **부등호 뒤집기** 까먹는 게 1순위 실수.
- 로그방정식 풀고 **진수 양수 조건**으로 해 걸러내기 잊음.
- aˣ = t 치환 후 **t > 0** 조건 빠뜨림.

## II-1. 삼각함수

### ⑧ 일반각과 호도법 (교과서 p.72–76)

**일반각.** 시초선(양의 x축)을 기준으로 반시계 방향 회전각을 양의 각으로 정의. 일반각은 **360°·n + θ** (n은 정수).

**사분면 판정.**
- 제1사분면: 0° < θ < 90°
- 제2사분면: 90° < θ < 180°
- 제3사분면: 180° < θ < 270°
- 제4사분면: 270° < θ < 360°
(일반각은 360°n 더한 뒤 판정.)

**호도법 (radian).**
- **π [rad] = 180°**
- 1 [rad] = (180/π)°
- 1° = (π/180) [rad]

**주요 변환 (암기):**
| 각도 | 호도 |
|---|---|
| 30° | π/6 |
| 45° | π/4 |
| 60° | π/3 |
| 90° | π/2 |
| 120° | 2π/3 |
| 135° | 3π/4 |
| 150° | 5π/6 |
| 180° | π |
| 270° | 3π/2 |
| 360° | 2π |

**부채꼴 공식 (반지름 r, 중심각 θ [rad]):**
- 호의 길이: **l = rθ**
- 넓이: **S = (1/2) r² θ = (1/2) r l**

**자주 틀리는 포인트.**
- θ 단위가 **라디안인지 도인지** 확인. l=rθ 공식은 **라디안 전용**.
- 일반각 부호: 시계 방향은 음수.

---

### ⑨ 삼각함수의 정의 (교과서 p.77–82)

**단위원 정의 (반지름 r, 각 θ의 동경 위 점 P(x, y)):**
- sin θ = y/r
- cos θ = x/r
- tan θ = y/x (x ≠ 0)
- csc θ = r/y, sec θ = r/x, cot θ = x/y (고교 범위 일부)

**사분면별 부호 (ASTC 규칙 — "All Students Take Calculus"):**

| 사분면 | sin | cos | tan |
|---|---|---|---|
| 1사분면 | + | + | + |
| 2사분면 | + | − | − |
| 3사분면 | − | − | + |
| 4사분면 | − | + | − |

**대표값 표 (반드시 암기):**

| θ | 0 | π/6 | π/4 | π/3 | π/2 |
|---|---|---|---|---|---|
| sin | 0 | 1/2 | √2/2 | √3/2 | 1 |
| cos | 1 | √3/2 | √2/2 | 1/2 | 0 |
| tan | 0 | √3/3 | 1 | √3 | ∄ |

**기본 항등식:**
- **sin²θ + cos²θ = 1**
- tan θ = sin θ / cos θ
- 1 + tan²θ = sec²θ
- 1 + cot²θ = csc²θ

**연관각 공식 (꼭 외울 것):**
- sin(−θ) = −sin θ, cos(−θ) = cos θ, tan(−θ) = −tan θ  (**우기함수**)
- sin(π − θ) = sin θ, cos(π − θ) = −cos θ, tan(π − θ) = −tan θ
- sin(π + θ) = −sin θ, cos(π + θ) = −cos θ, tan(π + θ) = tan θ
- sin(π/2 − θ) = cos θ, cos(π/2 − θ) = sin θ  (**여각 공식**)
- sin(π/2 + θ) = cos θ, cos(π/2 + θ) = −sin θ

**자주 틀리는 포인트.**
- sin²θ + cos²θ = 1 에서 **sin²θ = (sin θ)²** — 괄호 잊지 말 것.
- 3사분면 값 구할 때 cos·sin **둘 다 음수**.
- tan(π/2) 는 **정의 안 됨** — 답에 나오면 오답.

### ⑩ 삼각함수의 그래프 (교과서 p.83–95)

**y = sin x :**
- 정의역: 모든 실수
- 치역: [−1, 1]
- 주기: **2π**
- 원점 대칭 (홀함수)
- x = 0, π, 2π, ... 에서 0

**y = cos x :**
- 정의역: 모든 실수
- 치역: [−1, 1]
- 주기: **2π**
- y축 대칭 (짝함수)
- cos x = sin(x + π/2)

**y = tan x :**
- 정의역: x ≠ π/2 + nπ (n은 정수)
- 치역: 모든 실수
- 주기: **π**
- 원점 대칭 (홀함수)
- 점근선: x = π/2 + nπ

**일반형 y = a·sin(bx + c) + d :**
- **진폭**: |a|
- **주기**: 2π/|b|   (tan은 **π/|b|**)
- **평행이동**: x축으로 −c/b, y축으로 d
- **최댓값**: d + |a|, **최솟값**: d − |a|

| 일반형 | 진폭 | 주기 | 최대 | 최소 |
|---|---|---|---|---|
| y = a·sin(bx+c) + d | \|a\| | 2π/\|b\| | d+\|a\| | d−\|a\| |
| y = a·cos(bx+c) + d | \|a\| | 2π/\|b\| | d+\|a\| | d−\|a\| |
| y = a·tan(bx+c) + d | — | π/\|b\| | 없음 | 없음 |

**자주 틀리는 포인트.**
- 주기 공식에서 **|b|** — b가 음수여도 주기는 양수.
- 평행이동 값: y = sin(bx + c) 는 **x축 방향으로 −c/b 만큼** 이동 (부호 주의).
- tan 그래프는 **진폭 개념 없음**. |a|는 그래프의 기울기 스케일.
- 최댓값 문제: d + |a|. **a가 음수라도 |a|** 사용.

## 부록 · 자주 안 외워지는 공식 빠른 복습

- **log의 밑 변환**: log_a N = (log_c N)/(log_c a) — **분자가 진수, 분모가 밑**.
- **a^(log_a x) = x** — 지수·로그 역함수 관계.
- **부채꼴 넓이**: S = (1/2) r² θ — r² 잊지 말 것.
- **삼각함수 대표값**: 30°-60°-90° 직각삼각형 (1 : √3 : 2), 45°-45°-90° (1 : 1 : √2).
- **sin² + cos² = 1** 은 모든 θ에 대해 성립, 단 tan² + 1 = sec² 는 cos θ ≠ 0 조건.
- **주기 공식**: sin·cos는 2π/|b|, tan은 π/|b|.
