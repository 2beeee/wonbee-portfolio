## 3. 출제 패턴 추정 (보충자료 기반)

제공받은 **「평가원·수능 기출문제」 보충자료 3부**(지수·로그 27문항, 지수/로그함수 23문항, 삼각함수 27문항)가 출제 경향의 직접 단서. 보충자료는 **2012 ~ 2026학년도 평가원/수능·학평**에서 해당 범위만 뽑은 것. 유형 밀도가 높은 쪽:

| 영역 | 출제 비중 | 킬러 가능 지점 |
|---|---|---|
| 지수·로그 계산 | 높음 (소문제 3–5문항 예상) | 실생활 활용 (흑체복사형) |
| 지수·로그 방정식/부등식 | 매우 높음 | log의 밑 조건 분기 |
| 지수·로그함수 그래프 | 중 (그래프 해석 1–2) | 대칭성·교점 개수 |
| 일반각·호도법 변환 | 낮음 (객관식 1) | 부호 실수 |
| 삼각함수 값 (제2~4사분면) | 높음 | sinθ·cosθ 부호 분기 |
| 삼각함수 그래프 | 매우 높음 | 주기·최댓값·평행이동 킬러 |
| 단답/서논술 25점 | **합답형 3–4문항** | 경우 나누기 서술 |

> **핵심 판단**: 75점 객관식 + 25점 단답·서논술 구조. 객관식은 유형 인식 속도 싸움. 서논술 25점은 **경우 나누기 + 과정 쓰기**가 관건.

## 4. 50분 내 시간 배분 (목표)

```
00:00–02:00   시험지 전체 스캔 → 킬러 표시
02:00–22:00   객관식 쉬움~중 (1~12번) 20분 안에 끝내기
22:00–35:00   객관식 중상·그래프·방정식 (13~20번) 13분
35:00–45:00   단답·서논술 (25점) 10분 — 풀이 과정 필수
45:00–50:00   재검토 (부호·정의역·밑 조건 체크)
```

목표 득점: **78~88점** (100점은 욕심, 90 근방이 현실 최대).
우선 전략:
1. **확실한 75점부터 먼저 확보** — 기본 공식·그래프 암기로 객관식 60점+ 고정.
2. 단답·서논술은 부분점수 전략 — 백지는 절대 내지 않는다.
3. 킬러 (삼각함수 합답형, 로그 밑 분기)는 시간이 남으면 도전.
