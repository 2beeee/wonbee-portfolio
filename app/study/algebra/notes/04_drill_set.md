# 대수 드릴셋 — 유형별 대표문제 20題

> 2022 개정 대수 범위 × 평가원/수능 빈출 유형을 동등 난이도로 재구성. 각 문항에 **유형 번호**, **난이도**, **목표 시간**, 접은 풀이를 부착. 답만 보지 말고 **과정을 따라 손으로 쓰며** 풀어라.
>
> 표기: 🟢 쉬움 (1~2분) · 🟡 중간 (3~5분) · 🔴 킬러 (5분+)

## 드릴 1 — 거듭제곱근 값 계산 [유형1 · 🟢 · 2분]

다음 식의 값을 구하여라.
$$\sqrt[3]{8} \cdot \sqrt[4]{16} \cdot \sqrt[6]{64}$$

<details>
<summary>풀이 보기</summary>

모두 2의 거듭제곱으로 환원.
- ³√8 = 2
- ⁴√16 = 2
- ⁶√64 = 2

정답 = 2 · 2 · 2 = **8**
</details>

## 드릴 2 — 지수식 간단히 [유형1 · 🟢 · 2분]

$$\sqrt[3]{a^2 \sqrt{a}}$$ 를 aᵏ 꼴로 나타낼 때, k의 값은? (a > 0)

<details>
<summary>풀이 보기</summary>

루트 안부터 분수 지수로:
- a² · √a = a² · a^(1/2) = a^(5/2)
- ³√(a^(5/2)) = a^(5/6)

답: **k = 5/6**
</details>

## 드릴 3 — a^x + a^(−x) 제곱 활용 [유형2 · 🟡 · 3분]

실수 x에 대해 2ˣ + 2⁻ˣ = 3 일 때, 4ˣ + 4⁻ˣ 의 값을 구하여라.

<details>
<summary>풀이 보기</summary>

(2ˣ + 2⁻ˣ)² = 2²ˣ + 2·2ˣ·2⁻ˣ + 2⁻²ˣ = 4ˣ + 2 + 4⁻ˣ

3² = 4ˣ + 4⁻ˣ + 2
9 − 2 = 4ˣ + 4⁻ˣ

답: **7**
</details>

## 드릴 4 — 로그 성질 활용 [유형3 · 🟢 · 2분]

log₂ 12 − log₂ 3 + log₂ (1/2) 의 값을 구하여라.

<details>
<summary>풀이 보기</summary>

log₂ (12/3) − log₂ 2 = log₂ 4 − 1 = 2 − 1 = **1**
</details>

## 드릴 5 — 로그 계산 (소인수분해 응용) [유형3 · 🟡 · 4분]

log 2 = a, log 3 = b 일 때, log 15 를 a, b로 나타내면?

<details>
<summary>풀이 보기</summary>

15 = 30/2 = (3 · 10)/2
log 15 = log 3 + log 10 − log 2 = b + 1 − a

답: **1 − a + b**

(또는 log 15 = log(3·5) = log 3 + log 5 = b + (1−a))
</details>

---

## 드릴 6 — log_a b + log_b a 형 [유형4 · 🟡 · 5분]

a > 0, a ≠ 1, b > 0, b ≠ 1 이고 log_a b + 6·log_b a = 5 일 때, log_a b 의 모든 값의 합은?

<details>
<summary>풀이 보기</summary>

t = log_a b 치환 (t ≠ 0, log_b a = 1/t).
t + 6/t = 5
t² − 5t + 6 = 0
(t − 2)(t − 3) = 0
t = 2 또는 3

두 값의 합: **5**

(보정 체크: a, b > 0, ≠ 1 이므로 t ≠ 0, 두 해 다 유효.)
</details>

## 드릴 7 — 상용로그 자릿수 [유형5 · 🟡 · 4분]

log 2 = 0.3010 을 이용하여 2³⁰ 의 **자리수**를 구하여라.

<details>
<summary>풀이 보기</summary>

log(2³⁰) = 30 · log 2 = 30 × 0.3010 = 9.030

지표 = 9, 즉 자리수 = 지표 + 1 = **10자리**

(가수 = 0.030 → 2³⁰의 첫 숫자는 10^0.030 ≈ 1.07, 즉 1....)
</details>

## 드릴 8 — 지수방정식 치환 [유형6 · 🟡 · 4분]

방정식 4ˣ − 5·2ˣ + 4 = 0 의 모든 실수해의 **합**을 구하여라.

<details>
<summary>풀이 보기</summary>

t = 2ˣ (t > 0) 치환. 4ˣ = t².
t² − 5t + 4 = 0
(t − 1)(t − 4) = 0
t = 1 또는 4.

- t = 1 → 2ˣ = 1 → x = 0
- t = 4 → 2ˣ = 4 → x = 2

해의 합: 0 + 2 = **2**
</details>

---

## 드릴 9 — 로그방정식 진수 조건 [유형7 · 🟡 · 5분]

방정식 log₂(x − 1) + log₂(x + 1) = 3 의 해를 구하여라.

<details>
<summary>풀이 보기</summary>

정의역: x − 1 > 0 ∧ x + 1 > 0 → **x > 1**.

log₂{(x−1)(x+1)} = 3
(x−1)(x+1) = 2³ = 8
x² − 1 = 8
x² = 9
x = ±3

정의역에서 **x = 3** 만 유효.

답: **x = 3**

(함정: x = −3 은 x+1 > 0 조건 위배로 기각.)
</details>

---

## 드릴 10 — 로그방정식 치환 [유형7 · 🔴 · 6분]

방정식 (log x)² − 3·log x + 2 = 0 의 모든 양의 실수해의 **곱**을 구하여라.

<details>
<summary>풀이 보기</summary>

t = log x (상용로그, x > 0).
t² − 3t + 2 = 0
(t − 1)(t − 2) = 0
t = 1 또는 2.

- t = 1 → log x = 1 → x = 10
- t = 2 → log x = 2 → x = 100

해의 곱: 10 × 100 = **1000**

(log의 관점에서, 두 근의 합 = 3 → log(x₁·x₂) = 3 → x₁x₂ = 10³ = 1000.)
</details>

---

## 드릴 11 — 로그부등식 [유형8 · 🟡 · 4분]

부등식 log_(1/3)(x − 2) ≥ −2 의 해를 구하여라.

<details>
<summary>풀이 보기</summary>

정의역: x − 2 > 0 → **x > 2**.

밑 **1/3 < 1** 이므로 부등호 **뒤집기**.
log_(1/3)(x − 2) ≥ log_(1/3)(1/3)⁻² = log_(1/3) 9

→ (밑 < 1이라서 부등호 반전) x − 2 **≤** 9
→ x ≤ 11

정의역과 교집합: **2 < x ≤ 11**
</details>

## 드릴 12 — 지수함수 그래프 [유형9 · 🟡 · 3분]

함수 y = 2ˣ⁻¹ + 3 의 그래프의 점근선의 방정식과 y절편을 구하여라.

<details>
<summary>풀이 보기</summary>

- 점근선: 원래 y = 2ˣ 의 점근선 y = 0 을 **y축으로 +3** 이동. → **y = 3**
- y절편: x = 0 대입 → y = 2⁻¹ + 3 = 1/2 + 3 = **7/2**
</details>

## 드릴 13 — 로그함수 최댓값·최솟값 [유형10 · 🔴 · 6분]

2 ≤ x ≤ 8 에서 함수 y = (log₂ x)² − 2·log₂ x + 3 의 **최댓값과 최솟값**을 구하여라.

<details>
<summary>풀이 보기</summary>

t = log₂ x 치환. x ∈ [2, 8] → t ∈ [log₂ 2, log₂ 8] = **[1, 3]**.

y = t² − 2t + 3 = (t − 1)² + 2

t ∈ [1, 3] 에서 꼭짓점 t = 1 → 최솟값 2.
t = 3 에서 y = 9 − 6 + 3 = 6 → 최댓값 6.

답: 최댓값 **6** (x = 8 일 때), 최솟값 **2** (x = 2 일 때).
</details>

## 드릴 14 — 지수·로그 실생활 활용 [유형11 · 🟡 · 5분]

어떤 물체의 방사능 세기 N은 시간 t (년)에 대해 N = N₀ · (1/2)^(t/T) 로 감소한다. N₀ 는 초기값, T = 20 (년, 반감기).

t = 60년 후 **남아있는 비율** N/N₀ 를 구하여라.

<details>
<summary>풀이 보기</summary>

N/N₀ = (1/2)^(60/20) = (1/2)³ = **1/8**

(즉 60년 후 원래의 1/8, **12.5%** 만 남음.)

실생활 문제는 공식을 그대로 대입하고 **비를 취하는** 전략이 핵심.
</details>

## 드릴 15 — 호도법 변환과 부채꼴 [유형12 · 🟢 · 3분]

반지름이 4, 중심각이 **3π/4** (라디안)인 부채꼴의 **호의 길이**와 **넓이**를 구하여라.

<details>
<summary>풀이 보기</summary>

- 호의 길이: l = rθ = 4 × (3π/4) = **3π**
- 넓이: S = (1/2) r² θ = (1/2) × 16 × (3π/4) = **6π**

(또는 S = (1/2) r l = (1/2) × 4 × 3π = 6π. 동일.)
</details>

## 드릴 16 — 삼각함수 값 (사분면 판정) [유형13 · 🟡 · 4분]

sin θ = 4/5 이고 θ는 제2사분면의 각일 때, cos θ + tan θ 의 값을 구하여라.

<details>
<summary>풀이 보기</summary>

제2사분면: sin **+**, cos **−**, tan **−**.

sin²θ + cos²θ = 1
(4/5)² + cos²θ = 1
cos²θ = 1 − 16/25 = 9/25
cos θ = **−3/5** (제2사분면이므로 음수)

tan θ = sin θ / cos θ = (4/5) / (−3/5) = **−4/3**

답: cos θ + tan θ = −3/5 + (−4/3) = −9/15 − 20/15 = **−29/15**
</details>

## 드릴 17 — 삼각함수 관계식 [유형13 · 🔴 · 6분]

π < θ < 3π/2 이고 sin θ + cos θ = −1/2 일 때, sin θ · cos θ 와 sin θ − cos θ 를 구하여라.

<details>
<summary>풀이 보기</summary>

제3사분면: sin **−**, cos **−**, 따라서 sin · cos **+**.

(sin θ + cos θ)² = sin²θ + 2 sin θ cos θ + cos²θ = 1 + 2 sin θ cos θ
(−1/2)² = 1 + 2 sin θ cos θ
1/4 − 1 = 2 sin θ cos θ
sin θ cos θ = **−3/8**

(sin θ − cos θ)² = 1 − 2 sin θ cos θ = 1 − 2·(−3/8) = 1 + 3/4 = 7/4
sin θ − cos θ = ±√7 / 2

부호 판정: 제3사분면에서 sin θ − cos θ 는?
- sin θ < 0, cos θ < 0. 크기 비교는 각도에 따라 다르지만, π < θ < 5π/4 구간에서 |cos θ| > |sin θ| (음수지만 cos이 더 음수) → sin θ − cos θ > 0.
- 5π/4 < θ < 3π/2 구간에서 |sin θ| > |cos θ| → sin θ − cos θ < 0.

sin θ + cos θ = −1/2 (구체적 수치 제약) → 둘의 합이 음수이면서 둘 다 음수. 판별은 **추가 조건 필요** — 보통 시험에서는 이런 킬러는 객관식 선지에서 판별 가능하거나, ± 두 답 모두 인정.

답: sin θ cos θ = **−3/8**, sin θ − cos θ = **±√7 / 2**.

(킬러 팁: 부호 판별이 필요하면 추가 조건 검토. 서논술이면 경우 나누기 서술.)
</details>

## 드릴 18 — 삼각함수 그래프 주기·최대 [유형14 · 🟡 · 3분]

함수 y = 3 sin(2x − π/3) + 1 의 **주기**, **최댓값**, **최솟값**을 구하여라.

<details>
<summary>풀이 보기</summary>

일반형 y = a sin(bx + c) + d 에서 a = 3, b = 2, c = −π/3, d = 1.

- 주기: 2π / |b| = 2π / 2 = **π**
- 최댓값: d + |a| = 1 + 3 = **4**
- 최솟값: d − |a| = 1 − 3 = **−2**
</details>

## 드릴 19 — 삼각함수 그래프에서 식 복원 [유형14 · 🔴 · 6분]

다음 조건을 모두 만족하는 함수 y = a·cos(bx + c) + d 에서 a + b + d 를 구하여라. (단, a > 0, b > 0, 0 ≤ c < 2π.)

조건:
- 최댓값 5, 최솟값 −1
- 주기 π
- x = 0 일 때 y = 5 (즉, 최댓값을 지남)

<details>
<summary>풀이 보기</summary>

- 최대 = d + a = 5, 최소 = d − a = −1 → a = **3**, d = **2**.
- 주기 2π/b = π → b = **2**.
- x = 0 대입: y = 3·cos(c) + 2 = 5 → cos c = 1 → c = 0 (0 ≤ c < 2π).

a + b + d = 3 + 2 + 2 = **7**
</details>

## 드릴 20 — 삼각함수 방정식 해의 개수 [유형15 · 🔴 · 7분]

0 ≤ x ≤ 2π 에서 방정식 2 sin²x − 3 sin x + 1 = 0 의 서로 다른 실근의 **개수**를 구하여라.

<details>
<summary>풀이 보기</summary>

t = sin x 치환 (−1 ≤ t ≤ 1).
2t² − 3t + 1 = 0
(2t − 1)(t − 1) = 0
t = 1/2 또는 t = 1.

- **sin x = 1/2** : [0, 2π] 에서 x = π/6 또는 5π/6 → **2개**
- **sin x = 1** : [0, 2π] 에서 x = π/2 → **1개**

서로 다른 근: 2 + 1 = **3개**

(함정: sin x = 1 의 해는 1개. sin x = 1/2 은 2개. 합답형은 주기마다 몇 번 나오는지 꼼꼼히.)
</details>
