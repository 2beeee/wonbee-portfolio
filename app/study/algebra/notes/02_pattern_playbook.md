# ★ 대수 유형 플레이북 — 시험장 즉석 해법

> **이 파일이 내일 새벽 가장 많이 보게 될 파일이다.** 유형 인식 → 기계적 해법. 출처: 교과서(미래엔 황선욱) 처음~97p + 평가원/수능 기출 보충자료 3부. 15유형, 출제 빈도순 정렬.

## 유형 1. 거듭제곱근 값 계산 / 식 정리 [★★]

- **식별 신호**: ⁿ√a 기호, (ⁿ√a)ᵐ, ᵐ√(ⁿ√a) 가 식에 포함. 답이 단순한 분수 지수나 자연수.
- **해법 루틴**:
  1. 모든 ⁿ√ 를 **분수 지수 a^(m/n) 꼴로 환원**.
  2. 지수법칙으로 하나의 지수식 만들기 (aᵖ 꼴).
  3. 필요하면 a^(p/q) = ᵠ√(aᵖ) 로 되돌림.
- **공식/결과**: a^(m/n) · a^(p/q) = a^(m/n + p/q). 공통분모로 통분.
- **함정**:
  - 밑이 다른 거듭제곱근 (√2 · ³√3) → 그대로 두기. 억지로 하나로 묶지 마라.
  - 밑이 음수면 지수법칙 불가 → 우선 부호 분리.
- **대표 문제**: `04_drill_set.md` → 드릴 1, 2.

---

## 유형 2. 지수식에서 특정 값 구하기 (a^x + a^(-x) 꼴) [★★★]

- **식별 신호**: "aˣ + a⁻ˣ = k 일 때 a²ˣ + a⁻²ˣ 를 구하라" 또는 조건식이 aˣ 대칭형.
- **해법 루틴**:
  1. **t = aˣ + a⁻ˣ** (또는 aˣ − a⁻ˣ)로 치환.
  2. **제곱하면** (aˣ ± a⁻ˣ)² = a²ˣ + a⁻²ˣ ± 2.
  3. 세제곱식: (aˣ + a⁻ˣ)³ = a³ˣ + a⁻³ˣ + 3(aˣ + a⁻ˣ).
- **공식/결과**: 합의 대칭식은 **제곱하여 2 ±**, **세제곱하여 3(원식) ±**.
- **함정**:
  - aˣ > 0 이므로 aˣ + a⁻ˣ ≥ 2 (AM-GM, aˣ = a⁻ˣ = 1 일 때 등호).
  - aˣ − a⁻ˣ 의 부호는 x의 부호에 따라 다름.
- **대표 문제**: 드릴 3.

## 유형 3. 로그 값 계산 (성질 활용) [★★★]

- **식별 신호**: log_a M ± log_a N, log_a Mᵏ, log 변환 공식이 필요한 식.
- **해법 루틴**:
  1. 로그 4성질 (합, 차, 거듭제곱, 밑 변환) 중 어떤 게 적용될지 스캔.
  2. 진수를 **소인수분해** → log_a (2ᵖ · 3ᵖ) → p·log_a 2 + q·log_a 3.
  3. log 2, log 3, log 7 같은 문자값이 주어지면 표로 대입.
- **공식/결과**:
  - log_a MN = log_a M + log_a N
  - log_a (M/N) = log_a M − log_a N
  - log_a Mᵏ = k · log_a M
  - log_a b = log_c b / log_c a
- **함정**:
  - log(M+N) ≠ log M + log N **절대** 아님.
  - log_a Mᵏ 쓰려면 M > 0. Mᵏ = (−3)² = 9 같은 경우 k log_a M 으로 분해 금지.
  - log_(aᵏ) N = (1/k) log_a N **(역수)**.
- **대표 문제**: 드릴 4, 5.

---

## 유형 4. log_a b + log_b a 형 식 (역수 관계) [★★]

- **식별 신호**: 문제식에 log_a b 와 log_b a 가 **동시에** 등장. 또는 log_(a²)b + log_b √a 같은 형.
- **해법 루틴**:
  1. **log_b a = 1 / log_a b** 적용해서 **t = log_a b** 치환.
  2. 모든 로그를 t로 환산.
  3. 이차식 풀이 → 유효성 검증 (밑 조건).
- **공식/결과**: log_(aᵖ) bᵠ = (q/p) · log_a b.
- **함정**:
  - log_a b · log_b a = 1 (항상 성립, a, b > 0, ≠ 1).
  - 이차식 해 중 a=1, b=1 나오면 기각.
- **대표 문제**: 드릴 6.

## 유형 5. 상용로그 · 지표·가수 [★★]

- **식별 신호**: log N 의 정수부분·소수부분 언급, "N의 자리수는?", "N의 소수 몇째 자리에 처음 0 아닌 숫자?"
- **해법 루틴**:
  1. log N 을 계산 또는 주어진 log 2, log 3 값으로 표현.
  2. **지표(정수부분)** → 자리수·소수자리 판정.
  3. **가수(소수부분)** → 유효숫자 패턴 (같은 가수 → 같은 숫자 배열).
- **공식/결과**:
  - N의 자리수 = 지표 + 1 (N ≥ 1일 때)
  - 0 < N < 1, 소수 k째 자리에서 처음 0 아닌 숫자 → 지표 = −k
- **함정**:
  - 가수는 **항상 0 이상 1 미만**. log 0.003 = −3 + 0.4771 (가수 = 0.4771).
  - 같은 가수라고 같은 숫자 **아니다** — 0과 자릿수가 다를 수 있음.
- **대표 문제**: 드릴 7.

---

## 유형 6. 지수방정식 (밑 통일 / 치환) [★★★]

- **식별 신호**: aˣ = bˣ, a^(2x) + a^x = k, 지수에 식이 복잡.
- **해법 루틴**:
  1. **모든 밑을 통일** 시도. 2ˣ·4ˣ·8ˣ → 2ˣ·2²ˣ·2³ˣ.
  2. 안 되면 **t = aˣ 치환** (t > 0 조건).
  3. 이차·삼차식 → 해 구하고 **t > 0** 으로 걸러냄.
  4. x = log_a t.
- **공식/결과**: aˣ = aʸ ⇔ x = y (a ≠ 1, a > 0).
- **함정**:
  - **t > 0** 조건 필수. 음수 해 버리기.
  - aˣ = bˣ → a = b 또는 x = 0 — 두 경우 모두 확인.
- **대표 문제**: 드릴 8.

---

## 유형 7. 로그방정식 (진수 비교 / 치환) [★★★]

- **식별 신호**: log_a X = log_a Y, (log x)² 꼴, 로그가 여러 개 등장.
- **해법 루틴**:
  1. **정의역 (진수 > 0) 먼저 체크.**
  2. log_a X = log_a Y → X = Y.
  3. (log x)² − k·log x + m = 0 형이면 **t = log x 치환**.
  4. 푼 해를 **진수 > 0** 조건에 대입해 검산.
- **공식/결과**: X, Y > 0 하에서 log_a X = log_a Y ⇔ X = Y.
- **함정**:
  - 정의역 누락 → 오답 직행 구간.
  - log x² = 2 log x 는 **x > 0** 일 때만. x < 0 이면 log x² = 2 log |x|.
- **대표 문제**: 드릴 9, 10.

---

## 유형 8. 지수·로그 부등식 (밑 분기) [★★★]

- **식별 신호**: 부등호 + aˣ 또는 log_a x. 특히 밑 *a*가 문자일 때.
- **해법 루틴**:
  1. 밑 *a*의 범위 확인: **a > 1** 인가 **0 < a < 1** 인가?
  2. a > 1 → 부등호 **그대로**.
  3. 0 < a < 1 → 부등호 **뒤집기**.
  4. 로그 부등식이면 **진수 > 0** 조건 교집합.
- **공식/결과**:
  - aˣ > aʸ, a > 1 ⇒ x > y
  - aˣ > aʸ, 0 < a < 1 ⇒ x < y
- **함정**:
  - 밑이 문자 (예: 밑 = a²−1)면 a² − 1 > 1 vs 0 < a² − 1 < 1 **경우 나누기**.
  - 로그 부등식 → **정의역** (진수>0) 안 잊기.
- **대표 문제**: 드릴 11.

---

## 유형 9. 지수함수·로그함수 그래프 해석 [★★]

- **식별 신호**: 그래프가 주어지고 함수식을 맞추거나 교점 좌표·대소 비교.
- **해법 루틴**:
  1. **점근선** 확인 → 평행이동 값 찾기.
  2. **반드시 지나는 점** (지수함수: (0,1) 평행이동 후, 로그함수: (1,0) 평행이동 후) 이용.
  3. 증감 방향으로 a > 1 / 0 < a < 1 판별.
  4. 교점 문제는 **두 함수 같다고 놓고** 방정식화.
- **공식/결과**: y = aˣ 와 y = log_a x 는 y = x 대칭.
- **함정**:
  - 평행이동 후 (0, 1) 지나지 않을 수 있음 — 대입 실험.
  - 지수함수 점근선은 x축이지만 평행이동 후 y = q 로 변함.
- **대표 문제**: 드릴 12.

## 유형 10. 지수·로그함수의 최댓값/최솟값 [★★★]

- **식별 신호**: 정의역 [m, M] 이 주어지고 함수의 최대/최소 구하기.
- **해법 루틴**:
  1. **증감 판별**: 밑이 1보다 크냐 작냐.
  2. 증가함수면 x 최대 → y 최대, x 최소 → y 최소.
  3. 감소함수면 반대.
  4. 치환이 필요한 식(예: y = (log x)² + log x + 1)은 t = log x 로 이차식 만들고 꼭짓점 확인.
  5. **치환 후 t의 범위 재계산** — 이게 핵심.
- **공식/결과**: 이차식 y = a(t−h)² + k 는 꼭짓점에서 최대/최소, 단 **정의역 확인**.
- **함정**:
  - t 범위를 x 범위와 혼동. x ∈ [2, 8] 이고 t = log₂ x 면 **t ∈ [1, 3]**.
  - 밑이 0 < a < 1 이면 t의 범위 방향이 뒤집힘.
- **대표 문제**: 드릴 13.

## 유형 11. 지수·로그 실생활 활용 (흑체복사·음압·pH 형) [★★]

- **식별 신호**: "V = V₀ · 10^(something)", "T² ∝ ...", 공식이 문제에 **직접 제시**.
- **해법 루틴**:
  1. 주어진 공식을 **정확히 그대로** 대입.
  2. 대입값이 두 개 주어지면 **비(ratio)로 나눠** 모르는 변수 소거.
  3. 비를 취했으므로 **로그 취하기** → 선형식화.
- **공식/결과**: A/B = 10^(x−y) ⇒ log(A/B) = x − y.
- **함정**:
  - 로그 밑이 10인지 e인지 확인.
  - 공식 변수 의미 (입력이 x인지 x+1인지) 꼼꼼히.
- **대표 문제**: 드릴 14.

## 유형 12. 일반각·호도법 변환 [★]

- **식별 신호**: 각을 도 → 라디안 또는 라디안 → 도로 변환.
- **해법 루틴**:
  1. **180° = π [rad]** 기준 비례식.
  2. π/6 = 30°, π/4 = 45°, π/3 = 60° 즉시 인식.
  3. 음각·360°+α 는 먼저 **기본 범위 [0°, 360°)** 로 환원.
- **공식/결과**: 도 × (π/180) = 라디안, 라디안 × (180/π) = 도.
- **함정**:
  - 부채꼴 공식 l = rθ, S = (1/2)r²θ 는 **θ가 라디안일 때만**.
- **대표 문제**: 드릴 15.

## 유형 13. 삼각함수 값 계산 (사분면 판정) [★★★]

- **식별 신호**: "sin θ = 3/5 이고 π/2 < θ < π 일 때 cos θ를 구하라" 형태. 또는 tan θ가 주어지고 sin + cos 구하기.
- **해법 루틴**:
  1. **사분면 확인** → sin, cos, tan 부호 결정.
  2. **sin²θ + cos²θ = 1** 로 나머지 값 구하기.
  3. 부호는 **사분면에서 가져옴** — 무조건 양수로 쓰지 말 것.
- **공식/결과**:
  - tan θ = sin θ / cos θ
  - sin²θ + cos²θ = 1
  - 1 + tan²θ = 1/cos²θ
- **함정**:
  - √ 씌울 때 **부호** — 사분면에 따라 ±.
  - 제2사분면: sin +, cos −, tan −.
  - 제3사분면: sin −, cos −, tan +.
- **대표 문제**: 드릴 16, 17.

## 유형 14. 삼각함수 그래프 — 주기·최대·평행이동 [★★★]

- **식별 신호**: y = a·sin(bx + c) + d 꼴 함수의 주기·최댓값·평행이동 찾기. 또는 그래프에서 식 복원.
- **해법 루틴**:
  1. **최대·최소** → d + |a| 와 d − |a| 의 평균 = d (중심선), 차의 반 = |a|.
  2. **주기**: sin·cos은 2π/|b|, tan은 π/|b|.
  3. **평행이동**: bx + c = b(x + c/b) → **x축으로 −c/b**, y축으로 d.
  4. 그래프 점 하나 대입해 c 결정.
- **공식/결과**:

| 양 | sin/cos | tan |
|---|---|---|
| 진폭 | \|a\| | — |
| 주기 | 2π/\|b\| | π/\|b\| |
| 최대 | d+\|a\| | ∞ |
| 최소 | d−\|a\| | −∞ |

- **함정**:
  - 주기에 **|b|** — b가 음수여도 주기는 양수.
  - c의 부호 — bx + c vs b(x+c) vs b(x−c). 괄호 밖으로 뺄 때 부호 주의.
- **대표 문제**: 드릴 18, 19.

## 유형 15. 삼각함수 방정식·부등식 (범위 내 해의 개수) [★★★]

- **식별 신호**: "0 ≤ x ≤ 2π 에서 sin x = 1/2 의 해의 개수", sin·cos 복합식, **그래프 해석**.
- **해법 루틴**:
  1. 기본 해 1~2개 구하기 (단위원 또는 대표값).
  2. 일반해: sin x = k → x = sin⁻¹k + 2nπ 또는 π − sin⁻¹k + 2nπ.
  3. 주어진 범위 안에서 **해의 개수 카운팅**.
  4. 합답형은 각 해 대입 → 개수 세기.
- **공식/결과**:
  - sin x = k (−1 ≤ k ≤ 1): 해의 개수는 [0, 2π)에서 보통 0, 1 또는 2개.
  - sin²x + cos²x = 1 활용해 **한 함수로 통일**.
- **함정**:
  - sin x = 1 이나 −1은 해가 **1개** (나머지는 2개).
  - 주기 고려해 범위 안 해 전부 세기.
  - sin x · cos x = (1/2)sin 2x 활용 가능 (**공식 유인물에서 나올 수 있음**).
- **대표 문제**: 드릴 20.

## ★ 시험 직전 30초 리마인더

시험지 받자마자 머리 속에 박아야 할 4가지:

1. **밑 조건** — a > 0, a ≠ 1. 진수 N > 0.
2. **부등호 뒤집기** — 0 < a < 1 일 때.
3. **사분면 부호** — sin²+cos²=1 쓸 때 √의 ±.
4. **주기 공식** — sin/cos: 2π/|b|, tan: π/|b|.
