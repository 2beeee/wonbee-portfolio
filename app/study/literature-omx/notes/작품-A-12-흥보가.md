# 흥보가

- 갈래: 판소리
- 성격/제재: 해학적 · 풍자적 / 궁핍한 흥보의 매품팔이
- 주제: 조선 후기 궁핍한 양반과 신분 질서의 약화를 해학과 풍자로 드러낸다.
- 작가: 작자 미상
- 화자/시점: 판소리 창자/아니리 전개

## 핵심 시어 / 소재
| 시어 / 소재 | 의미 |
|---|---|
| 아니리 | 장단 없이 말하듯 이어 가는 사설 |
| 자진모리 | 빠른 장단. 흥보의 남루한 행색을 생동감 있게 전달 |
| 진양조 | 느린 장단. 흥보 아내의 만류와 슬픔을 길게 부각 |
| 매품팔이 | 화폐경제 중시·신분 질서 약화·부패한 사회 |

## ⭐ 체크 포인트
- 판소리 = 창 + 아니리 조합
- 조선 후기 사회상 3개: 궁핍한 양반 / 화폐경제 / 신분 질서 약화
