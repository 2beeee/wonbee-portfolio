## 화자 정서 / 태도 파악 ⭐
<a id="attitude"></a>

- 출제 빈도: ★★★★★
- 정의: 시·고전시가에서 화자가 무엇을 어떻게 느끼는지, 대상에게 어떤 태도를 취하는지 묻는 유형.

### 풀이 5단계
1. 화자가 누구인지 먼저 확정한다.
2. 긍정·부정 시어와 정서 전환 지점을 밑줄 친다.
3. 마지막 연/종장에서 정서가 유지되는지 뒤집히는지 확인한다.
4. 정서를 한 단어로 먼저 축약한다.
5. 선지의 과장·축소·반전을 제거한다.

### 함정 패턴
- 대상을 화자로 착각하기
- 한 구절만 보고 전체 정서를 단정하기
- 작가의 감정 = 화자의 감정으로 곧바로 치환하기

### 이번 시험 적용 예시
- <마음의 달>: 위로의 정서
- <찬기파랑가>: 추모 + 예찬
- <서경별곡>: 사랑 + 원망

## 시어 / 소재의 상징 의미 ⭐
<a id="symbol"></a>

- 출제 빈도: ★★★★★
- 정의: 핵심 시어·소재가 작품 안에서 무엇을 뜻하는지 연결하는 유형.

### 풀이 5단계
1. 반복되거나 제목에 걸린 시어를 먼저 고른다.
2. 그 시어가 긍정/부정 어느 맥락인지 본다.
3. 다른 시어와의 관계를 본다.
4. 작가 맥락이 붙으면 역사적 의미도 본다.
5. 대응표를 만든 뒤 안 맞는 선지를 버린다.

### 함정 패턴
- 비유된 대상과 비유하는 대상을 뒤집기
- 역사적 상징을 놓치기
- 다른 작품의 상징을 그대로 가져오기

### 이번 시험 적용 예시
- <자경>: 명경/명덕, 제세주/사공
- <도산십이곡>: 피미일인, 교교백구
- <꽃>: 꽃, 꽃 맹아리, 꽃성

## 표현상 특징과 효과 ⭐
<a id="device"></a>

- 출제 빈도: ★★★★☆
- 정의: 대조, 반복, 설의, 영탄, 장단, 삽입시 등 표현 장치의 효과를 묻는 유형.

### 풀이 5단계
1. 표현법 이름보다 효과를 먼저 본다.
2. 반복이면 운율/강조를 본다.
3. 형식 변화가 내용 변화와 맞물리는지 본다.
4. 보기의 효과가 작품 맥락과 맞는지 확인한다.
5. 효과가 과장되면 오답 처리한다.

### 함정 패턴
- 설의법을 단순 의문문으로 보기
- 삽입시를 장식용으로만 보기
- 장단 차이를 분위기와 연결하지 못하기

### 이번 시험 적용 예시
- <마음의 달>: 대비 + 역설
- <꽃>: 4행 연 + 영탄
- <흥보가>: 자진모리 vs 진양조

## 갈래 특징 / 문학사 의의 ⭐
<a id="genre"></a>

- 출제 빈도: ★★★★★
- 정의: 향가·고려 속요·시조·가사·판소리·전기소설 등의 형식과 문학사적 위치를 묻는 유형.

### 풀이 5단계
1. 먼저 갈래를 확정한다.
2. 형식 키워드 2~3개를 떠올린다.
3. 같은 갈래끼리만 비교한다.
4. 문학사 의의를 짧게 정리한다.
5. 작품 내용보다 갈래 특징을 먼저 요구하는지 본다.

### 함정 패턴
- 향가/속요/시조를 전부 옛 시로 뭉개기
- 판소리의 창과 아니리를 섞기
- 가사와 시조의 형식 차이를 놓치기

### 이번 시험 적용 예시
- <찬기파랑가>: 향찰, 10구체
- <서경별곡>: 후렴구, 연장체, 3음보
- <상춘곡>: 4음보 연속체의 정격 가사

## 인물 심리 / 시점 / 공간 의미 ⭐
<a id="narrative"></a>

- 출제 빈도: ★★★★★
- 정의: 소설에서 인물 심리, 시점, 공간의 상징 기능을 묻는 유형.

### 풀이 5단계
1. 누가 보고 말하는지부터 확인한다.
2. 공간이 그냥 배경인지 상징인지 판단한다.
3. 주요 인물을 둘씩 대비해 본다.
4. 결말 직전 장치가 무슨 역할인지 본다.
5. 인물의 말과 실제 선택이 어긋나는지 확인한다.

### 함정 패턴
- 무진을 단순 고향 배경으로만 보기
- 하인숙을 구원자로 단정하기
- 환상성을 사실성 결여로만 치부하기

### 이번 시험 적용 예시
- <무진기행>: 무진/안개/전보/통금 사이렌
- <달밤>: 1인칭 관찰자 시점
- <만복사저포기>: 전기성 + 실제 지명 + 삽입시

## 작품 간 비교 ⭐
<a id="compare"></a>

- 출제 빈도: ★★★★☆
- 정의: 같은 갈래나 비슷한 소재 작품을 비교하는 유형.

### 풀이 5단계
1. 두 작품의 갈래와 시대를 먼저 적는다.
2. 공통점 1개, 차이점 1개를 메모한다.
3. 화자의 태도와 표현 방식이 같은지 다른지 본다.
4. 비교 기준이 무엇인지 먼저 읽는다.
5. 둘 중 하나에만 해당하는 선지를 제거한다.

### 함정 패턴
- 소재가 비슷하다고 주제가 같다고 단정하기
- 같은 갈래라서 태도도 같다고 착각하기
- 비교 기준을 안 읽고 작품별 감상으로만 풀기

### 이번 시험 적용 예시
- <찬기파랑가> vs <제망매가>
- <서경별곡> vs <송인>
- <상춘곡> vs <누항사>

## 고전 시가 현대어 풀이 ⭐
<a id="classical"></a>

- 출제 빈도: ★★★★☆
- 정의: 낯선 고전 어휘를 현대어로 바꿔 맥락을 이해하는 유형.

### 풀이 5단계
1. 모르는 단어만 따로 적는다.
2. 해설란 대표 어휘부터 외운다.
3. 직역보다 핵심 의미어를 잡는다.
4. 후렴/감탄사는 의미보다 기능을 본다.
5. 현대어 풀이 뒤에 정서까지 붙인다.

### 함정 패턴
- 낯선 단어 때문에 감정 흐름을 놓치기
- 후렴을 내용어처럼 해석하기
- 하나의 어휘 뜻을 다른 작품에 그대로 옮기기

### 이번 시험 적용 예시
- <서경별곡>: 질삼 뵈, 긴힛끈, 사공
- <상춘곡>: 홍진, 풍월 주인, 봉두
- <누항사>: 누항, 안빈 관련 어휘

## 보기 활용 / 작가·사회 맥락
<a id="context"></a>

- 출제 빈도: ★★★★☆
- 정의: 보기나 작가·시대 정보를 바탕으로 작품을 해석하는 유형.

### 풀이 5단계
1. 보기 핵심 문장을 한 줄로 요약한다.
2. 작품 속 대응 요소를 1:1로 연결한다.
3. 보기에만 있고 작품에 없는 말을 경계한다.
4. 작가 맥락이 실제로 필요한지 확인한다.
5. 보기와 작품의 공통 핵심만 남긴다.

### 함정 패턴
- 보기 내용을 작품에 과도하게 덧씌우기
- 작가 정보만 외우고 작품 내부 단서를 안 보기
- 역사 맥락을 단순 배경 묘사와 혼동하기

### 이번 시험 적용 예시
- <자경> 3월 학평 보기 대응
- <꽃>의 이육사 맥락
- <흥보가>의 조선 후기 사회상
