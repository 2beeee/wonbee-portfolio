# 우선순위 분류

## TIER S
- **자경**: 가정통신문에 3월 학평 포함이 명시됨. 실제 handout에 문제와 해설이 붙어 있음.
- **도산십이곡**: 사진 3장 + PDF 1개가 전부 이 작품 필기. 선생님 강조 밀도 최고.
- **무진기행**: 활동지 2장 + 발췌 handout 4쪽. 시점·공간·인물 문제 재료가 가장 많음.

## TIER A
- **누항사**: 교과서 비교 작품인데 별도 handout까지 있음.
- **마음의 달 / 꽃 / 이옥설 / 찬기파랑가 / 서경별곡 / 시조 3편 / 상춘곡 / 만복사저포기 / 흥보가**
  - 교과서 메인 작품
  - 활동·OX·정리 문제가 이미 붙어 있음
  - 갈래/상징/문학사 의의형 객관식으로 출제 가능성 높음

## TIER B
- 비교 읽기, 확장 읽기, 보조 작품들
- 범위 안이더라도 **지금은 작품명 + 주제 + 핵심 장치 1개**만 확보

## 시간 배분
- **밤 120분**
  - 10분: 범위/티어 확인
  - 50분: S 3작품 + A 빠른 스캔
  - 50분: 유형북
  - 10분: 아침용 핀
- **아침 30분**
  - 15분: S 작품만
  - 10분: 함정 선지
  - 5분: 밤에 핀한 것
