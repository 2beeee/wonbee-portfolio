# 화학 실수 방지 체크리스트

> **사용법**: 문제 푼 직후 5초 안에 스캔. 매 문제 끝에 "이 체크리스트 항목에 걸리나?" 셀프 검증.

---

## 🔴 4대 킬러 실수 (무조건 확인)

### 1. 몰질량(M) 계산 실수
- [ ] **원자수 곱하기** 제대로? ($\ce{H2SO4} = 1 \times 2 + 32 + 16 \times 4 = 98$)
- [ ] $\ce{Cl} = 35.5$ (반올림 금지)
- [ ] 숨겨진 원자: $\ce{CaCO3}$는 산소 **3개** (C 1 + O 3)
- [ ] $\ce{NH4Cl}$ 같은 복합 이온화합물: 모든 원자 포함

**실수 예**: $\ce{Al2(SO4)3}$의 몰질량? → Al×2 + (S+O×4)×3 = 27×2 + 96×3 = 54+288 = **342**

---

### 2. 계수(coefficient) 실수
- [ ] 반응식 **계수 맞춤** 먼저 ( 계수 틀리면 양적관계 다 틀림)
- [ ] 계수비 = 몰비 = **부피비(기체)** = **분자수비** ≠ **질량비**
- [ ] 분수 계수 나오면 전체 × 2
- [ ] 최소 정수비로 약분

**실수 예**: $\ce{C2H6} + 7/2\ce{O2} \to 2\ce{CO2} + 3\ce{H2O}$ → 전체 ×2 필요

---

### 3. 상태 기호 누락
- [ ] (s) 고체 / (l) 액체 / (g) 기체 / (aq) 수용액
- [ ] **22.4 L 법칙은 (g)만** 적용. (l), (s)는 적용 **금지**
- [ ] "표준상태" = 0℃, 1기압
- [ ] 액체 $\ce{H2O}$ 질량 묻는데 22.4 L 쓰면 오답

---

### 4. 부분전하 방향 (δ+/δ−)
- [ ] **작은 전기음성도 → 큰 전기음성도** (화살표 머리가 −)
- [ ] **한국 교과서 관례**: 화살표는 +에서 −로 (IUPAC과 반대)
- [ ] H–O: $\overset{\delta+}{H} \to \overset{\delta-}{O}$ (O가 δ−)
- [ ] H–Cl: H쪽 δ+, Cl쪽 δ−

## ⚠️ 개념 혼동 함정 (빈출 TOP 10)

### ① Cl vs N 전기음성도
- **Cl(3.2) > N(3.0)** — 순서 뒤집히기 쉬움
- N이 Cl보다 위(2주기 vs 3주기)라 "N이 클 것 같다"는 함정

### ② 결합 극성 ≠ 분자 극성
| 분자 | 결합 극성 | 분자 극성 |
|---|---|---|
| $\ce{CO2}$ | 극성 (C=O) | **무극성** (대칭) |
| $\ce{CH4}$ | 거의 무극성 | 무극성 |
| $\ce{H2O}$ | 극성 (O-H) | **극성** (굽은형) |
| $\ce{CCl4}$ | 극성 (C-Cl) | **무극성** (대칭) |

### ③ 비공유 전자쌍 개수 ≠ 결합쌍 개수
- $\ce{H2O}$: 결합쌍 2, 비공유쌍 **2 (O에)**
- $\ce{NH3}$: 결합쌍 3, 비공유쌍 **1 (N에)**
- $\ce{CH4}$: 결합쌍 4, 비공유쌍 **0**

### ④ VSEPR 결합각: 정확한 숫자
- 정사면체 = **109.5°** (≠ 110°, 120°, 109°)
- 삼각뿔 = **107°**
- 굽은형 = **104.5°** (물에서)

### ⑤ H의 옥텟 예외
- H는 **2개**로 안정 (He처럼)
- 옥텟 규칙을 H에 적용하면 오답

### ⑥ 계수비 vs 질량비
- $\ce{2H2 + O2 -> 2H2O}$에서 H₂:O₂ = 2:1 (몰비·부피비)
- 질량비는 4g : 32g = **1:8**

### ⑦ 표준상태(STP) 조건 주의
- 22.4 L는 **0℃(=273K), 1기압** 조건만
- 25℃는 **22.4 L 아님** (다른 온도)
- 문제에 "25℃, 1기압"이라고 나오면 비례식 필요

### ⑧ 용융 vs 수용액 전기분해
| 종류 | (−)극 | (+)극 |
|---|---|---|
| NaCl 용융(l) | **Na 금속** | Cl₂ |
| NaCl 수용액(aq) | **H₂ 기체** (물이 먼저 환원) | Cl₂ |
| 물(H₂SO₄ 첨가) | H₂ | O₂ |

### ⑨ 극성 공유 결합 vs 이온 결합 경계
- Δχ = 1.7 주변은 모호
- 일반적으로 **금속+비금속 = 이온**, **비금속+비금속 = 공유**
- NaCl(2.3, 이온), HCl(1.0, 극성공유), HF(1.8, 경계 → 극성공유로 분류)

### ⑩ "극성 유유상통"
- 극성-극성 ✓ (H₂O + 에탄올)
- 무극성-무극성 ✓ (헥세인 + 기름)
- 극성-무극성 ✗ (물-기름)
- "같은 끼리 녹는다" 기억

## 🧪 단위·숫자 실수 체크

### 단위 변환
- [ ] 1 L = 1000 mL = 1000 cm³
- [ ] 1 g = 1000 mg
- [ ] 0℃ = 273 K (273.15이지만 시험에서 273)
- [ ] 1 atm = 760 mmHg = 101.3 kPa
- [ ] 22.4 L = 22400 mL = 22400 cm³

### 유효숫자
- [ ] 아보가드로수 $6.02 \times 10^{23}$ — 그냥 **6 × 10²³으로 근사 금지**
- [ ] 몰질량 Cl = **35.5** (반올림 금지)
- [ ] 최종 답은 **유효숫자 2~3자리**로 맞추기

### 지수 계산 (10의 거듭제곱)
- [ ] $\frac{6 \times 10^{23}}{6.02 \times 10^{23}} = \frac{6}{6.02} \approx 1$ — 지수 상쇄 주의
- [ ] 곱셈: 밑은 곱, 지수는 합 ($10^a \times 10^b = 10^{a+b}$)
- [ ] 나눗셈: 지수 **빼기** ($10^a / 10^b = 10^{a-b}$)

## 📝 서논술형 답안 체크

### 채점 기준 포인트 (부분점수)
- [ ] **핵심 용어** 포함 (몰, 전기음성도, 쌍극자, 비공유 전자쌍 등)
- [ ] **단위** 반드시 적기 (g, L, mol, °)
- [ ] **이유/근거** 서술: "~때문에", "~이므로"
- [ ] **그림·화살표** 요구시 방향 정확하게

### 답안 구조 템플릿
```
[결론] A는 B이다.
[근거] 왜냐하면 C이기 때문이다.
[추가 설명/예시] 예를 들어, D의 경우 E가 성립한다.
```

### 잘 쓰는 표현
- "전기음성도 차가 ~이므로 **극성 공유 결합**이다"
- "중심원자의 비공유 전자쌍 ~개가 결합쌍을 밀어 **결합각이 감소**한다"
- "분자 구조가 **대칭이므로 쌍극자 모멘트가 상쇄**되어 무극성이다"

## ⏰ 시험 중 시간 관리 함정

### 시간 낭비 함정
- [ ] **한 문제 3분 이상**: 다음 문제로 일단 넘어가기
- [ ] **자신 없는 문제 고쳐 쓰기**: 첫 선택이 70% 맞음 (직감 믿기)
- [ ] **OMR 계속 확인**: 10문제마다 한 번씩만 일괄 마킹

### 킬러 문제 대처
- [ ] 선지 5개 중 **명백히 틀린 2~3개 소거**
- [ ] 남은 것 중 **가장 가능성 높은** 걸로 찍기
- [ ] 별표 ★ 표시하고 **나중에 다시**

---

## 🎯 문제 풀고 나서 자가 검증 3초 루틴

**매 문제마다 답 쓰고 3초 확인**:
1. 단위 맞나? (g / mol / L / °)
2. 자릿수 그럴듯한가? (10⁻² ~ 10² 범위?)
3. 조건 빠뜨린 것 없나? (표준상태, 기체, 수용액?)

## 🚫 이번 시험 범위 **밖** 주제 (혼동 주의)

- [ ] **산-염기 중화반응 계산** — 2학기 범위
- [ ] **산화환원 반응 계수 맞추기** — 2학기 범위
- [ ] **기체의 용해도·헨리 법칙** — 2학기 범위
- [ ] **몰농도(M) 계산** — 범위 외
- [ ] **pH 계산** — 2학기
- [ ] **원자 모형 · 전자 배치 예외 (Cr, Cu)** — 1학년
- [ ] **양자수·궤도** — 1학년

> 💡 시험에 이런 거 보이면 → **시험 범위 외? 선지 재확인** → 아마 계산 안 시킴
