## 유형 1. 몰 ↔ 질량 ↔ 부피 ↔ 개수 환산

### 식별 신호
- "몇 mol인가?" / "몇 g인가?" / "몇 L인가?" / "분자 몇 개?"
- 0℃·1기압 표시 → 22.4 L 쓸 준비
- 아보가드로수 6.02×10²³ 명시

### 해법 루틴 (5초 안에)
```
[ 주어진 양 ]           [ 묻는 양 ]
    │                        ▲
    ▼                        │
    몰(mol) ──────────────────┘
    (중계 허브: 항상 mol로 먼저!)
```

### 핵심 공식
$$n \, (\text{mol}) = \frac{w \, (\text{g})}{M \, (\text{g/mol})} = \frac{N}{6.02 \times 10^{23}} = \frac{V \, (\text{L})}{22.4}$$

### 대표 문제
> $\ce{H2O}$ 3.6 g 속 분자 수는? (H=1, O=16)

**풀이 루틴**:
1. 몰질량: $M(\ce{H2O}) = 18$ g/mol
2. 몰: $n = 3.6/18 = 0.2$ mol
3. 분자 수: $N = 0.2 \times 6.02 \times 10^{23} = 1.204 \times 10^{23}$ 개

### 함정
- ⚠️ 22.4 L는 **0℃·1atm만**. 다른 조건이면 비례식
- ⚠️ $\ce{H2O}$는 액체에서 22.4 L 적용 **금지** (기체 상태에서만)
- ⚠️ 원자 수 ≠ 분자 수: $\ce{H2O}$ 1몰 = 분자 1몰 = 원자 3몰

## 유형 2. 화학 반응식 계수 맞추기

### 식별 신호
- 식에 빈 칸 (□, a, b, c) 있음
- "가장 간단한 정수비" 언급

### 해법 루틴 (원소별 밸런스)
1. 반응물·생성물 원자 종류 나열
2. **금속 → 비금속 → 수소 → 산소** 순서로 맞춤
3. 제일 마지막에 O, H 맞추기
4. 분수 나오면 전체에 2 곱하기

### 대표 문제
> $a\ce{C2H6} + b\ce{O2} \to c\ce{CO2} + d\ce{H2O}$의 계수는?

**풀이**:
1. C: 2a = c
2. H: 6a = 2d → d = 3a
3. O: 2b = 2c + d = 4a + 3a = 7a → b = 7a/2
4. a=2 대입: a=2, b=7, c=4, d=6
5. **$2\ce{C2H6} + 7\ce{O2} \to 4\ce{CO2} + 6\ce{H2O}$**

### 함정
- ⚠️ 전체 계수를 약분해서 **최소 정수비**로 만들기
- ⚠️ 생성물/반응물 **상태 기호** (s, l, g, aq) 빠뜨리지 말기

## 유형 3. 양적 관계 (몰 · 질량 · 부피)

### 식별 신호
- 반응식 주어짐 + 한쪽 양 주어짐 → 다른 쪽 양 계산

### 해법 루틴 (3-step)
```
1️⃣ 주어진 것 → 몰로 변환
2️⃣ 계수비로 묻는 물질의 몰 구하기
3️⃣ 묻는 몰 → 원하는 단위 (g, L, 개수)
```

### 핵심 공식
**계수비 = 몰비 = 분자수비 = 부피비 (기체)**

### 대표 문제
> $\ce{N2 + 3H2 -> 2NH3}$ 반응에서 $\ce{H2}$ 6 L가 반응할 때 생성되는 $\ce{NH3}$의 부피는? (같은 조건, 기체)

**풀이**:
- 계수비 H₂ : NH₃ = 3 : 2
- 기체 부피비 = 계수비
- $V(\ce{NH3}) = 6 \times \frac{2}{3} = 4$ L

### 함정
- ⚠️ **질량 묻는 문제**는 계수비로 바로 못 구함. 몰로 바꾼 후 몰질량 곱하기
- ⚠️ 기체 부피비는 **같은 온도·압력** 조건에서만
- ⚠️ 고체·액체는 부피비 적용 금지

## 유형 4. 한계반응물 판별 (킬러 유형)

### 식별 신호
- 반응물 **2개 모두** 양이 주어짐
- "완전히 반응" / "남는 물질" / "최대로 생성"

### 해법 루틴 (4-step)
```
1️⃣ 두 반응물 각각의 몰 계산
2️⃣ 반응식 계수로 나누기 (n/계수)
3️⃣ 작은 쪽 = 한계반응물 (다 소모)
4️⃣ 생성물 양 = 한계반응물 × (생성물 계수/한계 계수)
```

### 대표 문제
> $\ce{2H2 + O2 -> 2H2O}$에서 H₂ 4 mol과 O₂ 1 mol이 반응. 생성 H₂O와 남는 물질?

**풀이**:
1. H₂: 4/2 = **2**
2. O₂: 1/1 = **1** ← 더 작음 → O₂가 한계반응물
3. 생성 H₂O = 1 × 2/1 = **2 mol**
4. 소모된 H₂ = 1 × 2/1 = 2 mol → **H₂ 2 mol 남음**

### 함정
- ⚠️ "반응물이 적은 쪽"이 한계반응물이라 **단순 생각 금지** — 계수로 나눈 값으로 비교
- ⚠️ 남는 물질 계산할 때 **원래 양 - 반응량**
- ⚠️ 한계반응물은 **100% 소모**, 과량 반응물은 **일부 남음**

## 유형 5. 전기분해 생성물

### 식별 신호
- "(+)극 / (−)극에 생성되는 물질"
- "부피비 / 몰비" 묻기
- "용융 / 수용액" 구분

### 해법 루틴
```
물 전기분해 (H₂SO₄/Na₂SO₄ 소량):
  (−)극: H₂  ⏎  부피비 H₂ : O₂ = 2 : 1
  (+)극: O₂  ⏎  (몰비도 2 : 1)

NaCl(l) 용융:
  (−)극: Na (금속 석출)
  (+)극: Cl₂ (기체)

NaCl(aq) 수용액:
  (−)극: H₂ (물이 먼저 환원, Na⁺는 안 나옴)
  (+)극: Cl₂
```

### 대표 문제
> 물에 소량의 황산을 넣고 전기분해했을 때 양극과 음극에서 생성되는 기체의 부피비는?

**풀이**:
- 양극(+) = O₂, 음극(−) = H₂
- H₂ : O₂ = **2 : 1**
- (암기: "음수소양산소" + "수소가 2배")

### 함정
- ⚠️ **(+)극 = 양극 = 산화**, **(−)극 = 음극 = 환원**
- ⚠️ NaCl 수용액은 (−)극에서 Na 안 나옴 (물이 더 쉽게 환원)
- ⚠️ 순수한 물은 전기 안 통함 → 황산나트륨/황산 소량 첨가 필수

## 유형 6. 전기음성도 크기 비교

### 식별 신호
- "전기음성도가 가장 큰 것" / "작은 순서"
- 원소 여러 개 주어짐

### 해법 루틴 (암기 순위 + 주기율표)
```
1️⃣ 기본 순위: F > O > Cl > N > C > H
2️⃣ 같은 주기: 오른쪽 > 왼쪽
3️⃣ 같은 족: 위쪽 > 아래쪽
4️⃣ 18족은 제외 (정의 안 함)
```

### 대표 문제
> 다음 원소의 전기음성도를 큰 순서대로: **N, O, F, Cl**

**풀이**:
- F=4.0, O=3.4, Cl=3.2, N=3.0
- **F > O > Cl > N**

### 함정
- ⚠️ **Cl > N** (3.2 > 3.0) — 시험 함정 1위!
- ⚠️ 주기가 같으면 오른쪽, 족이 같으면 위쪽 — 두 원칙 **동시 적용 주의**
- ⚠️ **He, Ne, Ar**은 전기음성도 정의 안 함

## 유형 7. 결합 종류 판별 (공유/이온/극성)

### 식별 신호
- "이온 결합인 것" / "극성 공유 결합인 것"
- 화합물 여러 개 주어짐

### 해법 루틴 (Δχ 표 암기)
```
1️⃣ 원소 2개 전기음성도 차 |Δχ| 계산
2️⃣ Δχ = 0     → 무극성 공유 (같은 원자)
   Δχ ~ 0.4   → 거의 무극성 (C-H 등)
   Δχ 0.4~1.7 → 극성 공유
   Δχ > 1.7   → 이온 결합
3️⃣ 금속+비금속 = 보통 이온 결합
   비금속+비금속 = 공유 결합
```

### 대표 문제
> **$\ce{NaCl}$, $\ce{H2O}$, $\ce{Cl2}$, $\ce{HCl}$** 중 결합 종류는?

**풀이**:
| 화합물 | 결합 원자 | Δχ | 결합 종류 |
|---|---|---|---|
| $\ce{NaCl}$ | Na(0.9), Cl(3.2) | 2.3 | **이온** |
| $\ce{H2O}$ | O(3.4), H(2.2) | 1.2 | **극성 공유** |
| $\ce{Cl2}$ | Cl(3.2), Cl(3.2) | 0 | **무극성 공유** |
| $\ce{HCl}$ | H(2.2), Cl(3.2) | 1.0 | **극성 공유** |

### 함정
- ⚠️ **이온 결합 화합물도 공유 결합 포함 가능** ($\ce{NaOH}$, $\ce{NH4Cl}$)
- ⚠️ Δχ=1.7은 애매 → 문제에 명시된 값 따르기
- ⚠️ **결합의 극성 ≠ 분자의 극성** (유형 10 참고)

## 유형 8. 루이스 전자점식 그리기

### 식별 신호
- "루이스 전자점식을 그려라"
- 공유 전자쌍 / 비공유 전자쌍 개수 묻기

### 해법 루틴 (4-step)
```
1️⃣ 전체 원자가 전자 수 합산
2️⃣ 중심원자 정하기 (덜 전기음성, 주변 원자 연결)
3️⃣ 단일결합으로 연결 (결합쌍)
4️⃣ 남은 전자를 비공유 전자쌍으로 배치 → 옥텟 완성
   (필요하면 이중/삼중 결합으로 조정)
```

### 대표 문제
> $\ce{CO2}$의 루이스 구조와 공유/비공유 전자쌍 수?

**풀이**:
1. 원자가 전자: C(4) + O(6)×2 = 16
2. 중심원자: C (O보다 전기음성도 작음)
3. O=C=O 이중결합 2개 → 결합쌍 4개 (8 전자)
4. 남은 전자 16-8 = 8을 O 2개에 나눠 → 각 O에 비공유 2쌍
5. **공유 전자쌍: 4쌍 / 비공유 전자쌍: 4쌍**

### 분자별 공유/비공유 전자쌍 치트표
| 분자 | 공유 | 비공유 (분자 전체) |
|---|---|---|
| $\ce{H2O}$ | 2 | 2 (O에) |
| $\ce{NH3}$ | 3 | 1 (N에) |
| $\ce{CH4}$ | 4 | 0 |
| $\ce{CO2}$ | 4 | 4 (O 2개에 각 2쌍) |
| $\ce{N2}$ | 3 | 2 (N 2개에 각 1쌍) |
| $\ce{O2}$ | 2 | 4 (O 2개에 각 2쌍) |
| $\ce{HCN}$ | 4 | 1 (N에) |
| $\ce{HCl}$ | 1 | 3 (Cl에) |

### 함정
- ⚠️ H는 **2전자**로 안정 (옥텟 아님)
- ⚠️ **공유 전자쌍 개수** 묻는 문제는 단일/이중/삼결 모두 합쳐서
- ⚠️ **중심원자 비공유 전자쌍**이 VSEPR 모양 결정 (유형 9와 연결)

## 유형 9. VSEPR 모양 · 결합각

### 식별 신호
- "분자 모양" / "결합각"
- "가장 큰/작은 결합각"

### 해법 루틴 (3-step)
```
1️⃣ 중심원자 주변 전자쌍 수 세기 (결합+비공유)
2️⃣ 전자쌍 기하: 2→선형, 3→평면삼각, 4→정사면체
3️⃣ 비공유 전자쌍이 결합쌍 밀기 → 결합각 감소
   정사면체(109.5°) → 삼각뿔(107°) → 굽은(104.5°)
```

### 치트표 ⭐⭐⭐
| 중심원자 주변 | 비공유쌍 수 | 모양 | 결합각 | 예시 |
|---|---|---|---|---|
| 2쌍 | 0 | **선형** | **180°** | $\ce{CO2, BeCl2, HCN}$ |
| 3쌍 | 0 | **평면삼각** | **120°** | $\ce{BF3, BCl3}$ |
| 3쌍 | 1 | **굽은형** | <120° | $\ce{SO2, O3}$ |
| 4쌍 | 0 | **정사면체** | **109.5°** | $\ce{CH4, CCl4, NH4+}$ |
| 4쌍 | 1 | **삼각뿔** | **107°** | $\ce{NH3}$ |
| 4쌍 | 2 | **굽은형** | **104.5°** | $\ce{H2O}$ |

### 대표 문제
> 다음 분자의 결합각 큰 순서: $\ce{H2O}, \ce{CH4}, \ce{NH3}$

**풀이**:
- $\ce{CH4}$ = 109.5° (비공유 0)
- $\ce{NH3}$ = 107° (비공유 1)
- $\ce{H2O}$ = 104.5° (비공유 2)
- **$\ce{CH4} > \ce{NH3} > \ce{H2O}$**

### 함정
- ⚠️ **이중·삼중 결합도 1쌍**으로 취급 (VSEPR에서)
- ⚠️ **비공유 전자쌍 > 결합쌍** 밀어내는 힘
- ⚠️ $\ce{H2O}$와 $\ce{SO2}$ 둘 다 "굽은형"이지만 전자쌍 개수 다름

## 유형 10. 분자 극성 판정

### 식별 신호
- "극성 분자" / "무극성 분자"
- "쌍극자 모멘트가 0인 분자"

### 해법 루틴 (3-step)
```
1️⃣ 결합이 모두 무극성? → Yes → 무극성 분자
2️⃣ 결합에 극성 있음 → 분자 구조 확인
3️⃣ 대칭 구조 (쌍극자 상쇄)? → Yes → 무극성
                              → No  → 극성
```

### 대칭 → 무극성 분자 (쌍극자 상쇄)
| 분자 | 구조 |
|---|---|
| $\ce{CO2}$ | 선형 (O=C=O) |
| $\ce{BeCl2}$ | 선형 |
| $\ce{BF3}$ | 평면삼각 |
| $\ce{CH4}, \ce{CCl4}$ | 정사면체 |
| $\ce{PCl5}$ | 삼각쌍뿔 (이번 시험 외) |
| $\ce{SF6}$ | 정팔면체 (이번 시험 외) |

### 비대칭 → 극성 분자
| 분자 | 구조 | 쌍극자 방향 |
|---|---|---|
| $\ce{H2O}$ | 굽은형 | H ↑ O (2개 중첩) |
| $\ce{NH3}$ | 삼각뿔 | 3개 H → N 합산 |
| $\ce{HCl}$ | 이중원자 | H → Cl |
| $\ce{HCN}$ | 선형 비대칭 | H → C, C → N (방향 같아 상쇄 안 됨) |
| $\ce{SO2}$ | 굽은형 | 2개 O 방향 같아 상쇄 안 됨 |
| $\ce{CH3Cl}$ | 정사면체 치환 | Cl 쪽으로 치우침 |

### 대표 문제
> **$\ce{CH4}$와 $\ce{CH3Cl}$**의 극성 비교

**풀이**:
- $\ce{CH4}$: 정사면체, C-H 결합 4개 **대칭** → **무극성**
- $\ce{CH3Cl}$: 정사면체, **C-Cl 결합이 다름** (대칭 깨짐) → **극성**

### 함정
- ⚠️ **결합 극성과 분자 극성은 다름**. 예: $\ce{CO2}$는 결합(C=O) 극성이지만 분자는 무극성
- ⚠️ 삼각뿔($\ce{NH3}$), 굽은형($\ce{H2O, SO2}$)은 무조건 **극성**
- ⚠️ 정사면체라도 **치환된 화합물**은 극성일 수 있음 ($\ce{CHCl3}$, $\ce{CH2Cl2}$)

## 유형 11. 대전체 · 용해성 실험

### 식별 신호
- "대전체로 물줄기 휘는" / "용매에 녹는"
- "극성/무극성" 액체 구분 실험

### 해법 루틴 (극성/무극성 분류)
```
극성 용액: 물 (H₂O), 에탄올 (C₂H₅OH)
무극성 용액: 헥세인 (C₆H₁₄), 사염화탄소 (CCl₄), 벤젠 (C₆H₆), 기름

대전체 가까이 → 극성 액체만 휘어짐 ✓
용해: 극성-극성 ✓, 무극성-무극성 ✓, 극성-무극성 ✗
```

### 대표 문제
> 물과 헥세인에 소금($\ce{NaCl}$)과 요오드($\ce{I2}$)를 넣었을 때 어디에 녹나?

**풀이**:
- $\ce{NaCl}$ = 이온 결합 (극성) → **물에 녹음**, 헥세인 ✗
- $\ce{I2}$ = 무극성 분자 → **헥세인에 녹음**, 물 ✗
- 분리하면: 물 층 아래(밀도 큼), 헥세인 층 위

### 함정
- ⚠️ 사염화탄소($\ce{CCl4}$)는 **정사면체 대칭 → 무극성**
- ⚠️ 에탄올은 **-OH 때문에 극성** (하이드록실기)
- ⚠️ 밀도 순서: 물(1.0) < 헥세인(0.66) → 물이 아래

## 유형 12. 복합형 (분자 비교 매트릭스) — 킬러 유형

### 식별 신호
- 표에 분자 4~5개, 속성 여러 개
- "옳은/옳지 않은 것은?"
- 종합 판단 필요

### 해법 루틴 (체크리스트)
```
분자 하나씩 다음 속성 정리:
  ✓ 루이스 구조 (결합쌍/비공유쌍)
  ✓ VSEPR 모양 & 결합각
  ✓ 분자 극성
  ✓ 결합 극성 방향
```

### 대표 문제 (5분자 매트릭스)

| 속성 | $\ce{CH4}$ | $\ce{NH3}$ | $\ce{H2O}$ | $\ce{CO2}$ | $\ce{HCl}$ |
|---|---|---|---|---|---|
| **모양** | 정사면체 | 삼각뿔 | 굽은형 | 선형 | 선형(2원) |
| **결합각** | 109.5° | 107° | 104.5° | 180° | - |
| **비공유쌍 (중심)** | 0 | 1 | 2 | 0 | - |
| **결합 극성** | 무극성 (거의) | 극성 | 극성 | 극성 | 극성 |
| **분자 극성** | 무극성 | 극성 | 극성 | **무극성** | 극성 |
| **물에 녹음** | ✗ | ✓ | - | 약간 | ✓ |

### 선지별 판단 예시
- "CH₄는 극성 분자이다" → ❌ (무극성)
- "NH₃와 H₂O는 결합각이 같다" → ❌ (107° vs 104.5°)
- "CO₂는 극성 결합이지만 무극성 분자이다" → ✓
- "비공유 전자쌍이 많을수록 결합각 작다" → ✓
- "HCl는 분자 내 전기음성도 차 때문에 극성" → ✓

### 함정
- ⚠️ 선지 5개 중 하나만 찾기 → **가장 헷갈리는 2~3개 남기고** 소거
- ⚠️ "결합 극성 ≠ 분자 극성" 명제 **반복 출제**
- ⚠️ 숫자 값(결합각, 전기음성도) **정확히 암기해서 선지 매칭**
