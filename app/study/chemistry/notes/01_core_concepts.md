# 01. 핵심 개념 정리

> 각 개념은 **정의 → 법칙/공식 → 적용 조건 → 예외/오개념** 4박자 구조.
> $\ce{}$ 표기는 mhchem 형식. 수식은 KaTeX.

## Ⅰ-1-02. 몰(mol)과 물질의 양

### 1. 몰(mole)
- **정의**: 원자·분자 등 입자 $6.02\times10^{23}$개를 나타내는 **묶음 단위**. 기호 **mol**.
- **법칙**: 이 수를 **아보가드로수**($N_A$)라 한다. 정확값 $6.02214076\times10^{23}$.
- **적용**: 입자의 종류 무관 — 원자든 분자든 이온이든 1 mol = $N_A$개.
- **오개념**: "1몰 = 1g" (×). 1몰 = $N_A$개라는 **개수 단위**일 뿐, 질량은 물질마다 다르다.

### 2. 몰질량 (molar mass)
- **정의**: 물질 1 mol의 질량. 단위 **g/mol**.
- **공식**:
  $$\text{몰질량}(\text{g/mol}) = \sum (\text{각 원자의 몰질량} \times \text{원자 수})$$
- **자주 쓰는 몰질량 (외워야 함)**

| 물질 | 식 | 몰질량 |
|---|---|---|
| 물 | $\ce{H2O}$ | 18 |
| 이산화 탄소 | $\ce{CO2}$ | 44 |
| 염화 나트륨 | $\ce{NaCl}$ | 58.5 |
| 메테인 | $\ce{CH4}$ | 16 |
| 암모니아 | $\ce{NH3}$ | 17 |
| 산소 | $\ce{O2}$ | 32 |
| 질소 | $\ce{N2}$ | 28 |
| 수소 | $\ce{H2}$ | 2 |
| 에탄올 | $\ce{C2H5OH}$ | 46 |
| 포도당 | $\ce{C6H12O6}$ | 180 |
| 탄산수소 나트륨 | $\ce{NaHCO3}$ | 84 |

- **원자 몰질량**: H=1, C=12, N=14, O=16, Na=23, Mg=24.3, S=32, Cl=35.5
- **적용 조건**: 이온 결합 물질(NaCl 등)은 분자가 없지만 **화학식 단위당 몰질량**으로 계산.

### 3. 몰과 기체의 부피 — 아보가드로 법칙
- **법칙**: 같은 온도·압력에서 **기체의 종류와 관계없이** 같은 부피에 포함된 분자 수가 같다.
- **핵심 수치**: $0\ ^\circ\text{C}$, $1\ \text{atm}$에서 **기체 1 mol = 22.4 L**.
- **적용 조건**: **기체만** 해당. 액체·고체에는 적용 안 됨.
- **예외/주의**:
  - 온도·압력이 달라지면 22.4 L ≠ 1몰. (예: $t\ ^\circ\text{C}$, 1기압에서 1몰 = 24 L라는 조건이 흔함)
  - 문제에 명시된 조건을 먼저 확인.

### 4. 몰 · 질량 · 입자수 · 부피 통합 관계식 (0℃, 1기압)
$$\boxed{\;n(\text{mol}) = \frac{\text{입자 수}}{6.02\times10^{23}} = \frac{\text{질량}(g)}{\text{몰질량}(g/\text{mol})} = \frac{\text{부피}(L)}{22.4\ L/\text{mol}}\;}$$

## Ⅰ-2. 화학 반응식과 양적 관계

### 5. 화학 반응식
- **정의**: 반응물·생성물을 화학식과 상태기호로 나타내고 계수로 원자 수를 맞춘 식.
- **표기**: $\ce{H2(g) + O2(g) -> H2O(l)}$ 꼴.
- **상태 기호**: $(s)$ 고체, $(l)$ 액체, $(g)$ 기체, $(aq)$ 수용액.
- **작성 순서**:
  1. 화살표 왼쪽=반응물, 오른쪽=생성물, `+`로 연결
  2. 양변 원자 종류·개수 같게 **계수** 조정 (가장 간단한 정수비, 1은 생략)
  3. 상태 기호 기입
- **대표 균형식**:
  - $\ce{2H2(g) + O2(g) -> 2H2O(l)}$
  - $\ce{N2(g) + 3H2(g) -> 2NH3(g)}$
  - $\ce{CH4(g) + 2O2(g) -> CO2(g) + 2H2O(l)}$
  - $\ce{Mg(s) + 2HCl(aq) -> MgCl2(aq) + H2(g)}$
  - $\ce{CaCO3(s) + 2HCl(aq) -> CaCl2(aq) + CO2(g) + H2O(l)}$
  - $\ce{2C(s) + O2(g) -> 2CO(g)}$
  - $\ce{Fe2O3(s) + 3CO(g) -> 2Fe(s) + 3CO2(g)}$

### 6. 화학 반응식이 알려주는 것
- **계수비** = **몰비** = **분자 수 비** = **기체의 부피비** (온도·압력 일정일 때)
- **계수비 ≠ 질량비**. 질량비는 "몰비 × 각 물질 몰질량"으로 계산.
- 예시: $\ce{N2 + 3H2 -> 2NH3}$
  - 몰비 = 1 : 3 : 2
  - 부피비(0℃, 1atm) = 1 : 3 : 2
  - 질량비 = 28 : 6 : 34 = 14 : 3 : 17

### 7. 화학 반응의 양적 관계 — 한계반응물
- **정의**: 반응식 계수비에 따라 먼저 모두 소진되는 반응물. 이 반응물이 생성물 양을 결정한다.
- **풀이 루틴** (4단계):
  1. 균형식 작성
  2. 주어진 양을 **mol로 환산** (질량÷몰질량, 부피÷22.4)
  3. 계수비로 나눠 작은 쪽이 **한계반응물**
  4. 한계반응물 mol × 생성물 계수/반응물 계수 = 생성물 mol
- **대표 예제** (교과서 p.37):
  $\ce{Mg(s) + 2HCl(aq) -> MgCl2(aq) + H2(g)}$
  - Mg 1.2 g = 0.05 mol, HCl 0.2 mol
  - 계수비 1:2 → Mg : HCl 필요비 0.05 : 0.1
  - HCl 0.2 mol 중 0.1만 쓰임 → **Mg이 한계**, HCl 0.1 mol 남음
  - H₂ = 0.05 mol → 0℃ 1atm에서 $0.05 \times 22.4 = 1.12\ \text{L}$

## Ⅱ-1. 화학 결합과 결합의 극성

### 8. 화학 결합의 전기적 성질
- **핵심 실험 1 — 물의 전기분해**:
  $\ce{2H2O(l) -> 2H2(g) + O2(g)}$
  - (+)극 → $\ce{O2}$ 기체
  - (-)극 → $\ce{H2}$ 기체
  - **부피비 (+):(-) = 1 : 2** (즉 $\ce{O2:H2 = 1:2}$)
  - **전해질($\ce{Na2SO4}$) 녹이는 이유**: 순수한 물은 전류가 잘 안 흐르므로 이온을 공급.
- **핵심 실험 2 — NaCl 용융액 전기분해**:
  $\ce{2NaCl(l) -> 2Na(l) + Cl2(g)}$
  - (+)극 → $\ce{Cl2}$ 기체
  - (-)극 → $\ce{Na}$
  - **NaCl 고체**는 전류 ×, **액체(용융액)**는 전류 ○ (이온이 자유 이동)
- **결론**: 화학 결합(공유/이온 모두)에는 **전자가 관여**한다 = 화학 결합에 전기적 성질이 있다.

### 9. 전기음성도 (electronegativity)
- **정의**: 공유 결합을 이루는 원자가 **공유 전자쌍을 끌어당기는 정도**를 상대적 수치로 나타낸 것.
- **폴링 척도**: **F = 4.0** 기준.
- **주기적 경향**:
  - 같은 주기: 원자 번호 ↑ → 전기음성도 ↑ (F에 가까울수록 큼)
  - 같은 족: 원자 번호 ↑ → 전기음성도 ↓
  - **대체로 주기율표 우측 상단(F 근처)이 최대**, 좌측 하단이 최소.
- **1~3주기 주요 값** (교과서 p.89):
  | 원자 | H | Li | Be | B | C | N | O | F |
  |---|---|---|---|---|---|---|---|---|
  | χ | 2.2 | 1.0 | 1.6 | 2.0 | 2.6 | 3.0 | 3.4 | **4.0** |
  | 원자 | Na | Mg | Al | Si | P | S | Cl | |
  | χ | 0.9 | 1.3 | 1.6 | 1.9 | 2.2 | 2.6 | 3.2 | |

### 10. 결합의 극성
- **무극성 공유 결합**: 같은 원자끼리 결합 → 공유 전자쌍 치우침 없음. $\ce{H2, O2, N2, F2}$ 등.
- **극성 공유 결합**: 다른 원자끼리 결합, 전기음성도 차이 > 0 → 공유 전자쌍이 전기음성도 큰 쪽으로 치우침.
  - 전기음성도 큰 쪽 = **부분적 음전하 $\delta^-$**
  - 전기음성도 작은 쪽 = **부분적 양전하 $\delta^+$**
- **이온 결합**: 전기음성도 차이가 매우 클 때 전자가 완전히 이동. $\ce{NaCl}$ 등.
- **연속적 스펙트럼**: 무극성 공유결합 → 극성 공유결합 → 이온 결합 (전기음성도 차이 증가 방향)

### 11. 결합의 쌍극자 모멘트
- **정의**: 극성 공유 결합에서 **극성의 크기**를 나타내는 벡터량.
- **표기**: 화살표 ($\delta^+ \longrightarrow \delta^-$, 즉 **전기음성도 작은 원자 → 큰 원자** 방향).
- **크기**: 전기음성도 차이 클수록 쌍극자 모멘트 ↑.
- **무극성 공유 결합의 결합 쌍극자 모멘트 = 0**.

## Ⅱ-2. 분자의 구조와 성질

### 12. 루이스 전자점식 / 루이스 구조
- **루이스 전자점식**: 원소 기호 주위 상하좌우에 **원자가 전자**를 점으로 표시.
  - 전자 4개까지는 상하좌우 하나씩, 5번째부터 쌍 이루게.
- **루이스 구조**: 공유 전자쌍을 **결합선**으로 표시. 비공유 전자쌍은 점 쌍 또는 생략.
- **옥텟 규칙**: 결합 후 원자가 가장 바깥 껍질에 **전자 8개**(H, He는 2개)를 갖는 안정한 배치.
- **결합 종류**:
  - 단일 결합: 공유 전자쌍 1개 (예: $\ce{H-H}$, $\ce{H-F}$)
  - 이중 결합: 공유 전자쌍 2개 (예: $\ce{O=O}$, $\ce{C=O}$)
  - 삼중 결합: 공유 전자쌍 3개 (예: $\ce{N#N}$, $\ce{C#N}$ in HCN)
- **주요 분자의 전자쌍 구조** (암기):

| 분자 | 공유 전자쌍 | 비공유 전자쌍 (중심원자) |
|---|---|---|
| $\ce{H2}$ | 1 | 0 |
| $\ce{HF}$ | 1 | 3 (F) |
| $\ce{H2O}$ | 2 | **2** (O) |
| $\ce{NH3}$ | 3 | **1** (N) |
| $\ce{CH4}$ | 4 | 0 |
| $\ce{BeCl2}$ | 2 | 0 (Be) |
| $\ce{BCl3}$ | 3 | 0 (B) |
| $\ce{CO2}$ | 4 (이중 결합 2개) | 0 (C) |
| $\ce{HCN}$ | 4 (단일 1 + 삼중 1) | 0 (C) |
| $\ce{N2}$ | 3 (삼중) | 1 + 1 |

### 13. 전자쌍 반발 이론 (VSEPR)
- **정의**: 중심 원자 주위의 전자쌍은 **정전기적 반발을 최소화**하기 위해 서로 최대한 멀리 배치.
- **반발력 크기**: 비공유–비공유 > 비공유–공유 > 공유–공유
- **이중/삼중 결합 = 단일 결합과 똑같이 "1개의 전자쌍 방향"**으로 취급.
- **전자쌍 수 → 배치 → 결합각**:

| 공유 전자쌍 수 | 비공유 전자쌍 수 | 분자 구조 | 결합각 | 대표 분자 |
|---|---|---|---|---|
| 2 | 0 | 선형 (linear) | **180°** | $\ce{BeCl2}$, $\ce{CO2}$, $\ce{HCN}$, $\ce{CS2}$ |
| 3 | 0 | 평면 삼각형 | **120°** | $\ce{BCl3}$, $\ce{BF3}$, $\ce{CH2O}$ |
| 4 | 0 | 정사면체 | **109.5°** | $\ce{CH4}$, $\ce{CCl4}$, $\ce{CF4}$ |
| 3 | 1 | 삼각뿔 | **≈107°** | $\ce{NH3}$, $\ce{NCl3}$, $\ce{PH3}$ |
| 2 | 2 | 굽은 형 (V자) | **≈104.5°** | $\ce{H2O}$, $\ce{H2S}$, $\ce{OF2}$ |

- **결합각 크기 순서** (같은 전자쌍 수 4일 때): $\ce{CH4}\ (109.5°) > \ce{NH3}\ (107°) > \ce{H2O}\ (104.5°)$
  - 이유: **비공유 전자쌍이 많을수록 공유 전자쌍을 더 세게 밀어서 결합각 작아짐**.

### 14. 분자의 극성
- **무극성 분자**: 분자의 쌍극자 모멘트 **합 = 0**.
- **극성 분자**: 분자의 쌍극자 모멘트 **합 ≠ 0**.
- **판단 순서**:
  1. 이원자 분자 → 같은 원자면 무극성, 다른 원자면 극성
  2. 삼원자 이상 → **결합의 쌍극자 모멘트 벡터 합**으로 판단
    - 선형+대칭 (예: $\ce{CO2}$): 서로 반대 방향 상쇄 → 무극성
    - 평면삼각+대칭 (예: $\ce{BCl3}$): 120° 세 벡터 합 0 → 무극성
    - 정사면체+대칭 (예: $\ce{CH4}, \ce{CCl4}$): 합 0 → 무극성
    - **굽은 형**, **삼각뿔** → 비공유 전자쌍 방향으로 합이 남음 → **극성**
- **대표 무극성**: $\ce{H2, O2, N2, F2, CO2, BCl3, CH4, CCl4}$
- **대표 극성**: $\ce{HF, HCl, H2O, NH3, H2S, CH3Cl, HCN, CH2O}$

### 15. 분자 구조 → 물질의 성질
- **전기장 속 배열**: 극성 분자 → 일정 방향으로 정렬, 무극성 분자 → 무질서.
- **대전체 가까이 했을 때**: 극성 분자(예: 물줄기) → 대전체 쪽으로 **휜다** (대전체의 +/- 부호 무관).
- **용해성 "Like dissolves like"**:
  - 극성 용매(물) ↔ 극성 물질·이온 결합 물질 잘 녹음 ($\ce{NaCl, HCl, C2H5OH, CuCl2}$)
  - 무극성 용매(헥세인 $\ce{C6H14}$) ↔ 무극성 물질 잘 녹음 ($\ce{I2}$)
- **섞이지 않는 조합**: 물 ↔ 기름, 물 ↔ 헥세인 (층 분리)
