## 유형 1. 몰 환산 (3문제)

### 1-A (쉬움 ⭐)
**$\ce{CH4}$ 4.8 g은 몇 mol인가? (C=12, H=1)**

<details>
<summary>해설 보기</summary>

$M(\ce{CH4}) = 12 + 1 \times 4 = 16$ g/mol
$n = w/M = 4.8/16 = 0.3$ mol
**답: 0.3 mol**
</details>

### 1-B (중간 ⭐⭐)
**$\ce{O2}$ $3.01 \times 10^{23}$개는 0℃·1기압에서 몇 L를 차지하는가?**

<details>
<summary>해설 보기</summary>

$n = N/N_A = 3.01 \times 10^{23} / 6.02 \times 10^{23} = 0.5$ mol
$V = n \times 22.4 = 0.5 \times 22.4 = 11.2$ L
**답: 11.2 L**
</details>

### 1-C (킬러 ⭐⭐⭐)
**0℃·1기압에서 어떤 기체 11.2 L의 질량이 22 g이다. 이 기체의 분자량은?**

<details>
<summary>해설 보기</summary>

$n = V/22.4 = 11.2/22.4 = 0.5$ mol
$M = w/n = 22/0.5 = 44$ g/mol
→ **$\ce{CO2}$** (몰질량 44)
**답: 44**
</details>

## 유형 2. 계수 맞추기 (3문제)

### 2-A (쉬움)
**$a\ce{H2} + b\ce{O2} \to c\ce{H2O}$의 $a:b:c$는?**

<details>
<summary>해설 보기</summary>

H: 2a = 2c → a = c
O: 2b = c → c = 2b
a=2, b=1, c=2
**답: 2 : 1 : 2**
</details>

### 2-B (중간)
**$a\ce{Al} + b\ce{O2} \to c\ce{Al2O3}$의 최소 정수 계수는?**

<details>
<summary>해설 보기</summary>

Al: a = 2c
O: 2b = 3c
c=2 대입: a=4, b=3, c=2
**답: 4, 3, 2 → $4\ce{Al} + 3\ce{O2} \to 2\ce{Al2O3}$**
</details>

### 2-C (킬러)
**$a\ce{C3H8} + b\ce{O2} \to c\ce{CO2} + d\ce{H2O}$의 $a+b+c+d$는?**

<details>
<summary>해설 보기</summary>

C: 3a = c
H: 8a = 2d → d = 4a
O: 2b = 2c + d = 6a + 4a = 10a → b = 5a
a=1: a=1, b=5, c=3, d=4
$a+b+c+d = 1+5+3+4 = 13$
**답: 13** (식: $\ce{C3H8} + 5\ce{O2} \to 3\ce{CO2} + 4\ce{H2O}$)
</details>

## 유형 3. 양적 관계 (3문제)

### 3-A (쉬움)
**$\ce{CaCO3 -> CaO + CO2}$ 반응에서 $\ce{CaCO3}$ 10 g이 완전히 분해되면 생기는 $\ce{CO2}$ 몰수는? (Ca=40, C=12, O=16)**

<details>
<summary>해설 보기</summary>

$M(\ce{CaCO3}) = 40 + 12 + 48 = 100$
$n(\ce{CaCO3}) = 10/100 = 0.1$ mol
계수비 1:1이므로 $n(\ce{CO2}) = 0.1$ mol
**답: 0.1 mol**
</details>

### 3-B (중간)
**$\ce{N2 + 3H2 -> 2NH3}$에서 $\ce{H2}$ 0.3 mol이 완전히 반응하면 생성되는 $\ce{NH3}$의 질량은? (N=14, H=1)**

<details>
<summary>해설 보기</summary>

계수비 H₂ : NH₃ = 3 : 2
$n(\ce{NH3}) = 0.3 \times 2/3 = 0.2$ mol
$M(\ce{NH3}) = 14 + 3 = 17$
$w = 0.2 \times 17 = 3.4$ g
**답: 3.4 g**
</details>

### 3-C (킬러)
**$\ce{Mg + 2HCl -> MgCl2 + H2}$에서 Mg 6 g이 완전히 반응할 때 0℃·1기압에서 생성되는 $\ce{H2}$ 기체의 부피는? (Mg=24)**

<details>
<summary>해설 보기</summary>

$n(\ce{Mg}) = 6/24 = 0.25$ mol
계수비 Mg : H₂ = 1 : 1
$n(\ce{H2}) = 0.25$ mol
$V = 0.25 \times 22.4 = 5.6$ L
**답: 5.6 L**
</details>

## 유형 4. 한계반응물 (3문제)

### 4-A (쉬움)
**$\ce{2H2 + O2 -> 2H2O}$에서 H₂ 0.2 mol, O₂ 0.2 mol 반응. 한계반응물은?**

<details>
<summary>해설 보기</summary>

H₂: 0.2/2 = 0.1
O₂: 0.2/1 = 0.2
작은 쪽 → **H₂가 한계반응물** (O₂ 남음)

생성 H₂O = 0.2 × 2/2 = 0.2 mol
소모 O₂ = 0.2 × 1/2 = 0.1 mol → 남은 O₂ = 0.1 mol
**답: H₂**
</details>

### 4-B (중간)
**$\ce{N2 + 3H2 -> 2NH3}$에서 N₂ 2 mol, H₂ 3 mol 반응. 생성되는 NH₃ 몰수는?**

<details>
<summary>해설 보기</summary>

N₂: 2/1 = 2
H₂: 3/3 = 1 ← 작음 → H₂ 한계
생성 NH₃ = 3 × 2/3 = **2 mol**
**답: 2 mol**
</details>

### 4-C (킬러)
**Mg 4.8 g과 HCl 0.3 mol이 반응 ($\ce{Mg + 2HCl -> MgCl2 + H2}$). 생성되는 $\ce{H2}$의 몰수는? (Mg=24)**

<details>
<summary>해설 보기</summary>

$n(\ce{Mg}) = 4.8/24 = 0.2$ mol
Mg: 0.2/1 = 0.2
HCl: 0.3/2 = 0.15 ← 작음 → **HCl 한계**
$n(\ce{H2}) = 0.3 \times 1/2 = 0.15$ mol
**답: 0.15 mol**

(Mg 소모 = 0.15 mol, 남은 Mg = 0.05 mol = 1.2 g)
</details>

## 유형 5. 전기분해 (3문제)

### 5-A (쉬움)
**물에 $\ce{H2SO4}$를 소량 첨가하여 전기분해할 때, (−)극에서 생기는 기체는?**

<details>
<summary>해설 보기</summary>

(−)극: 환원 → $2\ce{H+} + 2e^- \to \ce{H2}$
**답: 수소 ($\ce{H2}$)**
</details>

### 5-B (중간)
**물 전기분해에서 같은 시간 동안 생성된 기체 중 $\ce{O2}$의 부피가 50 mL였다. $\ce{H2}$의 부피는?**

<details>
<summary>해설 보기</summary>

H₂ : O₂ = **2 : 1** (부피비)
$V(\ce{H2}) = 50 \times 2 = 100$ mL
**답: 100 mL**
</details>

### 5-C (킬러)
**NaCl을 녹여(용융) 전기분해했을 때 (+)극에서 0℃·1기압 Cl₂ 기체 2.24 L 생성. 반응한 NaCl의 몰수는? (Na=23, Cl=35.5)**

<details>
<summary>해설 보기</summary>

$n(\ce{Cl2}) = 2.24/22.4 = 0.1$ mol
(+)극 반응: $2\ce{Cl-} \to \ce{Cl2} + 2e^-$
→ Cl⁻ 0.2 mol 필요 → NaCl 0.2 mol
**답: 0.2 mol**
</details>

## 유형 6. 전기음성도 비교 (3문제)

### 6-A (쉬움)
**$\ce{F, O, N, Cl}$ 중 전기음성도가 가장 큰 원소는?**

<details>
<summary>해설 보기</summary>

F=4.0 (가장 큼)
**답: F**
</details>

### 6-B (중간)
**다음을 전기음성도 큰 순서로 배열: $\ce{H, C, N, O}$**

<details>
<summary>해설 보기</summary>

H(2.2), C(2.6), N(3.0), O(3.4)
**답: O > N > C > H**
</details>

### 6-C (킬러)
**다음 결합 중 **결합의 극성(Δχ)** 이 가장 큰 것은? $\ce{H-F, H-O, N-H, C-H, C-O}$**

<details>
<summary>해설 보기</summary>

|Δχ|:
- H-F: 4.0-2.2 = **1.8**
- H-O: 3.4-2.2 = 1.2
- N-H: 3.0-2.2 = 0.8
- C-H: 2.6-2.2 = 0.4
- C-O: 3.4-2.6 = 0.8

**답: H-F (Δχ=1.8)**
</details>

## 유형 7. 결합 종류 (3문제)

### 7-A (쉬움)
**$\ce{Cl2}$의 결합 종류는?**

<details>
<summary>해설 보기</summary>

Cl-Cl, Δχ=0 → **무극성 공유 결합**
</details>

### 7-B (중간)
**다음 중 **이온 결합** 화합물은? $\ce{NaCl, HCl, Cl2, H2O, NH3}$**

<details>
<summary>해설 보기</summary>

- NaCl: Na(0.9), Cl(3.2), Δχ=2.3 → **이온**
- HCl: Δχ=1.0 → 극성 공유
- Cl₂: Δχ=0 → 무극성 공유
- H₂O: Δχ=1.2 → 극성 공유
- NH₃: Δχ=0.8 → 극성 공유

**답: NaCl**
</details>

### 7-C (킬러)
**다음 화합물을 결합 종류별로 분류: $\ce{MgO, CO2, H2, NaOH, CH4, KCl}$**

<details>
<summary>해설 보기</summary>

| 화합물 | Δχ | 분류 |
|---|---|---|
| MgO | 3.4-1.3=2.1 | 이온 |
| CO₂ | 3.4-2.6=0.8 | 극성 공유 |
| H₂ | 0 | 무극성 공유 |
| NaOH | Na-O 이온 + O-H 공유 | **이온+극성공유 모두** |
| CH₄ | 2.6-2.2=0.4 | 거의 무극성 |
| KCl | 3.2-0.8=2.4 | 이온 |

**답: 이온=MgO, KCl / NaOH는 둘 다 / 극성공유=CO₂ / 무극성공유=H₂, CH₄**
</details>

## 유형 8. 루이스 전자점식 (3문제)

### 8-A (쉬움)
**$\ce{HCl}$의 공유 전자쌍과 비공유 전자쌍 수는?**

<details>
<summary>해설 보기</summary>

H-Cl (단일결합), Cl에 비공유 3쌍
- 공유: **1쌍**
- 비공유: **3쌍** (Cl에)
</details>

### 8-B (중간)
**$\ce{CO2}$의 공유 전자쌍 총 수는?**

<details>
<summary>해설 보기</summary>

O=C=O (이중결합 2개)
이중결합 = 2쌍, 전체 = **4쌍**
**답: 4쌍**
</details>

### 8-C (킬러)
**$\ce{HCN}$의 루이스 구조에서 공유/비공유 전자쌍 수는?**

<details>
<summary>해설 보기</summary>

H-C≡N
- H-C: 단일 (1쌍)
- C≡N: 삼중 (3쌍)
- 공유 전자쌍 총: **4쌍**
- 비공유: N에 **1쌍** (5 원자가전자 - 3쌍 공유 = 2전자 = 1쌍)
- C에 0쌍 (4 전자가전자 전부 사용)

**답: 공유 4쌍, 비공유 1쌍**
</details>

## 유형 9. VSEPR · 결합각 (3문제)

### 9-A (쉬움)
**$\ce{CO2}$의 분자 모양은?**

<details>
<summary>해설 보기</summary>

중심 C 주변 전자쌍 2 (이중결합 2개도 각 1쌍으로 셈), 비공유 0
→ **선형**, 결합각 **180°**
</details>

### 9-B (중간)
**$\ce{NH3}$와 $\ce{H2O}$의 결합각을 비교하라**

<details>
<summary>해설 보기</summary>

$\ce{NH3}$: 비공유 1쌍 → **107°**
$\ce{H2O}$: 비공유 2쌍 → **104.5°**
**NH₃ > H₂O** (비공유 많을수록 결합각 ↓)
</details>

### 9-C (킬러)
**$\ce{BF3}$와 $\ce{NH3}$의 모양과 결합각을 비교. 왜 다른가?**

<details>
<summary>해설 보기</summary>

- $\ce{BF3}$: B 주변 전자쌍 3 (비공유 0) → **평면삼각, 120°**
- $\ce{NH3}$: N 주변 전자쌍 4 (결합 3 + 비공유 1) → **삼각뿔, 107°**

**이유**: B는 원자가 전자 3개 (옥텟 미충족, 6전자로 안정), N은 5개로 비공유 1쌍 가지기 때문
→ 전자쌍 개수가 달라 배치가 달라짐
</details>

## 유형 10. 분자 극성 판정 (3문제)

### 10-A (쉬움)
**$\ce{H2O}$는 극성인가 무극성인가?**

<details>
<summary>해설 보기</summary>

굽은형 구조 + O-H 극성 결합, 쌍극자 상쇄 안 됨
→ **극성 분자**
</details>

### 10-B (중간)
**다음 중 무극성 분자: $\ce{CO2, H2O, NH3, CH4, HCl}$**

<details>
<summary>해설 보기</summary>

- CO₂: 선형 대칭 → **무극성**
- H₂O: 굽은 → 극성
- NH₃: 삼각뿔 → 극성
- CH₄: 정사면체 대칭 → **무극성**
- HCl: 이중원자 극성결합 → 극성

**답: CO₂, CH₄**
</details>

### 10-C (킬러)
**$\ce{CH4}$와 $\ce{CH3Cl}$의 극성을 비교하라. 왜 다른가?**

<details>
<summary>해설 보기</summary>

- $\ce{CH4}$: 정사면체, 4개 C-H 결합 **대칭** → **무극성**
- $\ce{CH3Cl}$: 정사면체, 3개 C-H + 1개 C-Cl (**비대칭**)
  - C-Cl 결합이 C-H보다 강한 극성
  - 쌍극자 상쇄 불가 → **극성**

**답: CH₄는 무극성, CH₃Cl은 극성. 대칭성 차이 때문**
</details>

## 유형 11. 대전체 · 용해 (3문제)

### 11-A (쉬움)
**물줄기에 대전체를 가까이 대면 어떻게 되는가?**

<details>
<summary>해설 보기</summary>

물 = 극성 분자 → 쌍극자가 대전체와 반대 방향으로 배열
→ **물줄기가 휘어진다**
</details>

### 11-B (중간)
**설탕, 소금, 기름, 요오드 중 물에 잘 녹는 것과 헥세인에 잘 녹는 것을 분류하라**

<details>
<summary>해설 보기</summary>

| 물질 | 극성? | 물 | 헥세인 |
|---|---|---|---|
| 설탕 | 극성 (-OH 다수) | ✓ | ✗ |
| 소금 (NaCl) | 이온 (극성에 잘녹) | ✓ | ✗ |
| 기름 | 무극성 | ✗ | ✓ |
| I₂ | 무극성 | ✗ | ✓ |

**답: 물 → 설탕, 소금 / 헥세인 → 기름, I₂**
</details>

### 11-C (킬러)
**헥세인, 에탄올, 물, 사염화탄소 4가지 액체를 대전체에 가까이 댔을 때 휘어지는 것을 **모두** 고르시오**

<details>
<summary>해설 보기</summary>

| 액체 | 극성? | 휘어짐? |
|---|---|---|
| 헥세인 ($\ce{C6H14}$) | 무극성 | ✗ |
| 에탄올 ($\ce{C2H5OH}$) | 극성 (-OH) | ✓ |
| 물 ($\ce{H2O}$) | 극성 | ✓ |
| 사염화탄소 ($\ce{CCl4}$) | **무극성** (정사면체 대칭) | ✗ |

**답: 에탄올, 물**

⚠️ CCl₄는 C-Cl이 극성이지만 **정사면체 대칭**이라 분자 전체는 무극성 → 헷갈리지 말 것
</details>

## 유형 12. 복합형 (3문제)

### 12-A (쉬움)
**다음 설명 중 옳은 것은?**

① CH₄는 극성 분자이다
② 물의 결합각은 109.5°이다
③ NH₃의 중심원자 비공유 전자쌍은 1쌍이다
④ CO₂의 결합은 무극성이다
⑤ F의 전기음성도는 3.4이다

<details>
<summary>해설 보기</summary>

① 무극성 ✗
② 104.5° ✗
③ **정답** ✓
④ C=O 극성 결합 ✗
⑤ F=4.0, O=3.4 ✗

**답: ③**
</details>

### 12-B (중간)
**$\ce{CO2, CH4, NH3, H2O}$를 다음 기준으로 분류하라:**
**(a) 분자 모양 (b) 결합각 (c) 극성 여부**

<details>
<summary>해설 보기</summary>

| 분자 | 모양 | 결합각 | 극성 |
|---|---|---|---|
| CO₂ | 선형 | 180° | 무극성 |
| CH₄ | 정사면체 | 109.5° | 무극성 |
| NH₃ | 삼각뿔 | 107° | 극성 |
| H₂O | 굽은형 | 104.5° | 극성 |

**결합각 순**: CO₂ > CH₄ > NH₃ > H₂O
**극성**: NH₃, H₂O만
</details>

### 12-C (킬러)
**다음 표에서 **모두 옳은** 분자는?**

| 분자 | 모양 | 결합각 | 분자 극성 | 결합 극성 |
|---|---|---|---|---|
| ① CH₄ | 정사면체 | 109.5° | 무극성 | 무극성 |
| ② NH₃ | 삼각뿔 | 104.5° | 극성 | 극성 |
| ③ H₂O | 굽은형 | 104.5° | 극성 | 극성 |
| ④ CO₂ | 선형 | 180° | 극성 | 극성 |
| ⑤ HCl | 선형 | 180° | 극성 | 극성 |

<details>
<summary>해설 보기</summary>

① CH₄: 모양/각 ✓, 분자 무극성 ✓, 결합 C-H는 "거의 무극성"(Δχ=0.4)이라 애매. 보통 **무극성으로 봄** → 대부분 ✓로 처리
② NH₃: 결합각 **107°** (104.5° ✗) → **오답**
③ H₂O: 모두 맞음 ✓ → **정답 유력**
④ CO₂: 분자 **무극성** (극성 아님) → **오답**
⑤ HCl: 이중원자이므로 "선형"이라 부르지 않고 결합각 개념이 없음. 극성은 맞음 → 모양/결합각 기재가 어색

**답: ③ (+ ① 가능)**

⚠️ 문제의 정확한 선지 표현에 따라 답이 ① 또는 ③
⚠️ CH₄의 "C-H 결합 극성" 서술은 교과서 관례에 따라 달라질 수 있음
</details>
