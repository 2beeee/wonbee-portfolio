# 화학 중간고사 개요 (2026-1학기)

## 시험 정보
- **일시**: 2026-04-23(목) 2교시 **09:30 ~ 10:20** (50분)
- **학년/과목**: 2학년 화학 (2022 개정 교육과정)
- **시험범위**:
  - **교과서**: 처음 ~ **93p** (Ⅰ. 화학의 언어 전체 + Ⅱ. 물질의 구조와 성질 전체 — 공유 결합, 분자 구조까지)
  - **유인물**: 1회 ~ 3회 (몰 · 양적관계 · 물질의 구조와 성질)
- **배점**: 객관식 85점 + 단답형·서논술형 15점 = **100점**
- **평가**: 5등급 상대평가

|  | 02. 몰과 물질의 양 | 18-25 | **아보가드로수 6.02×10²³**, 몰질량(g/mol), 기체 1몰=**22.4 L**(0℃, 1기압) |
| **2. 화학 반응식** | 01. 화학 반응식의 의미 | 30-35 | 계수비 = 몰비 = 분자수비 = 부피비 (≠질량비) |
|  | 02. 화학 반응의 양적 관계 | 36-39 | **한계반응물**, Mg+HCl 대표 예제 |

## 시험범위 한 줄 요약
> **몰(22.4 L · 6.02×10²³ · g/mol) → 화학 반응식 계수비 → 양적 관계 → 전기음성도 · 결합 극성 → 루이스 구조 → VSEPR 분자 모양 → 분자 극성 · 용해성**

## 50분 시간 배분 전략 (100점 만점, 문항 수 가정: 객관식 18~20 + 서논술 2~3)
| 시간 | 할 일 |
|---|---|
| **0분** | 문제지 받자마자 서논술형 페이지 먼저 확인 → 머릿속에서 문제 틀 잡기 |
| **0~3분** | 1번부터 쉬운 문제 빠르게 쓸어담기 (10~12문항, 1문제당 10~15초) |
| **3~30분** | 중간 난이도 객관식 정밀 소거법 (1문제당 1~1.5분) |
| **30~40분** | **서논술형 집중 공략** — 부분점수 최대 확보 |
| **40~47분** | 킬러 객관식 (몰 비율 계산, 분자 구조 분류) |
| **47~50분** | OMR 마킹 확인, 계수 맞춤·단위 재확인 |

## 학생 현황 반영 전략
- **내신 3등급 / 모의 1등급대** → 개념 이해력은 충분, **속도·계산 정확도**가 변수
- **목표**: 중상 난이도 문제 **전부 정답** → **85~90점 안정권**
- **킬러 전략**: 모든 걸 맞추려 하지 말고, 모르는 문제는 **가장 근접한 선지로 찍고 다음 문제로** (시간 손실 방지)
- **실수 방지**: 몰질량/계수/상태기호/부분전하 방향 — 4대 실수 체크

## 시험 직전 30초 루틴 (교실에서)
1. 아보가드로수 = **6.02 × 10²³**
2. 0℃ 1기압 기체 1몰 = **22.4 L** (단, 조건 달리 나오면 바꿔 계산)
3. H=1, C=12, N=14, O=16, Na=23, Mg=24, S=32, Cl=35.5
4. 전기음성도: **F > O > N ≈ Cl > C > H**
5. VSEPR 결합각: 180° / 120° / 109.5° / **107°(NH₃)** / **104.5°(H₂O)**
6. "극성 끼리 잘 녹는다" — 물-에탄올 ○, 물-헥세인 ×
