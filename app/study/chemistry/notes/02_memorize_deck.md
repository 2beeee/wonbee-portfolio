> **사용법**: 문제 풀면서 헷갈리면 Ctrl+F로 검색. 시험 전날 저녁 + 당일 아침 2회 암송.
> **우선순위**: ⭐⭐⭐ 무조건 암기 / ⭐⭐ 시험에 자주 / ⭐ 여유 있으면

## 2. 몰 통합 환산 치트시트 ⭐⭐⭐

```
         ÷ 몰질량(g/mol)              × 22.4 L/mol
 질량(g) ──────────────→ 몰(n) ──────────────→ 부피(L, STP)
         ← × 몰질량 ──                ← ÷ 22.4 ──
                         │
                         │ × 6.02×10²³
                         ▼
                      입자 수
```

**한 줄 공식** ⭐⭐⭐
$$n = \frac{w}{M} = \frac{N}{N_A} = \frac{V}{22.4} \ (\text{STP})$$

| 변환 | 식 |
|---|---|
| g → mol | $n = w/M$ |
| mol → g | $w = n \times M$ |
| mol → 개수 | $N = n \times 6.02 \times 10^{23}$ |
| 개수 → mol | $n = N / 6.02 \times 10^{23}$ |
| mol → L (STP) | $V = n \times 22.4$ |
| L (STP) → mol | $n = V / 22.4$ |

### 암기 순위 ⭐⭐⭐
$$F > O > \underbrace{Cl}_{3.2} > \underbrace{N}_{3.0} > C > H$$

> ⚠️ **Cl이 N보다 크다** (3.2 > 3.0) — 시험 함정 포인트!

### 주기율표 경향
- **같은 주기**: 오른쪽으로 갈수록 증가 (Li < Be < B < C < N < O < F)
- **같은 족**: 아래로 갈수록 감소 (F > Cl > Br > I)
- **예외**: 18족(비활성) 전기음성도 **정의 안 함**

## 6. 결합 · 극성 판정 ⭐⭐⭐

### 결합 종류 (전기음성도 차 |Δχ|)
| Δχ 범위 | 결합 종류 | 예 |
|---|---|---|
| 0 ~ 0.4 | **무극성 공유** | H–H, Cl–Cl, C–H (Δ=0.4) |
| 0.4 ~ 1.7 | **극성 공유** | H–Cl (Δ=1.0), O–H (Δ=1.2), N–H (Δ=0.8) |
| 1.7 이상 | **이온 결합** | NaCl (Δ=2.3), MgO (Δ=2.1) |

> 💡 **C–H 결합은 "거의 무극성"** — 대칭이면 CH₄, CCl₄ 같이 분자도 무극성

### 쌍극자 모멘트 (→) 방향
- **작은 전기음성도 → 큰 전기음성도** 쪽으로 화살표
- 화살표 머리(+)는 δ⁺, 꼬리(−)는 δ⁻
- 예: $\overset{\delta+}{H} \rightarrow \overset{\delta-}{Cl}$
- 예: $\overset{\delta+}{H} \rightarrow \overset{\delta-}{O} \leftarrow \overset{\delta+}{H}$

> ⚠️ 한국 교과서 관례: **화살표는 +에서 −로** (국제 IUPAC 관례와 반대)
> 쌍극자 모멘트 기호: ⟶ + 가로 막대 (꼬리 끝에)

|---|---|---|
| 1족 | 1 | H, Li, Na, K |
| 2족 | 2 | Be, Mg, Ca |
| 13족 | 3 | B, Al |
| 14족 | 4 | C, Si |

| 16족 | 6 | O, S |
| 17족 | 7 | F, Cl, Br, I |

### 옥텟 규칙

- 각 결합 = 2개 전자
- 단일결합(1쌍), 이중결합(2쌍), 삼중결합(3쌍)

## 8. VSEPR · 분자 구조 ⭐⭐⭐

### 결합각 암기표
| 분자 모양 | 결합각 | 전자쌍 배치 | 대표 분자 |
|---|---|---|---|
| **선형** | **180°** | 2쌍 (비공유 0) | $\ce{CO2}$, $\ce{BeCl2}$, $\ce{HCN}$ |
| **평면삼각** | **120°** | 3쌍 (비공유 0) | $\ce{BF3}$, $\ce{BCl3}$ |
| **굽은형** (3개 배치) | ~120°보다 작음 | 3쌍 (비공유 1) | $\ce{SO2}$, $\ce{O3}$ |
| **정사면체** | **109.5°** | 4쌍 (비공유 0) | $\ce{CH4}$, $\ce{NH4+}$ |
| **삼각뿔** | **107°** | 4쌍 (비공유 1) | $\ce{NH3}$ |
| **굽은형** (4개 배치) | **104.5°** | 4쌍 (비공유 2) | $\ce{H2O}$ |

### 결합각 감소 순서 ⭐⭐⭐
$$\underbrace{CH_4}_{109.5°} > \underbrace{NH_3}_{107°} > \underbrace{H_2O}_{104.5°}$$

**이유**: 비공유 전자쌍 늘어날수록 결합쌍을 **더 세게 밀어붙임** → 결합각 줄어듦

## 9. 분자 극성 판정 ⭐⭐⭐

### 판정 플로우차트
```
[분자 받기]
     │
     ▼
결합이 모두 무극성? ──Yes──→ 무극성 (예: H₂, Cl₂, O₂)
     │ No
     ▼
분자 구조가 대칭? ──Yes──→ 무극성 (쌍극자 상쇄)
     │ No                   예: CO₂ (선형), CCl₄ (정사면체), BF₃ (평면삼각)
     ▼
극성 분자 (쌍극자 안 상쇄)
예: H₂O (굽은형), NH₃ (삼각뿔), HCl
```

### 극성 vs 무극성 분자 정리
| 극성 분자 ✓ | 무극성 분자 ✗ |
|---|---|
| $\ce{H2O}$ (굽은형) | $\ce{CO2}$ (선형 대칭) |
| $\ce{NH3}$ (삼각뿔) | $\ce{CH4}$ (정사면체 대칭) |
| $\ce{HCl}, \ce{HF}$ | $\ce{CCl4}$ (정사면체 대칭) |
| $\ce{HCN}$ | $\ce{BF3}$ (평면삼각 대칭) |
| $\ce{SO2}$ (굽은형) | $\ce{BeCl2}$ (선형 대칭) |
| $\ce{CH3Cl}$ | $\ce{O2}, \ce{N2}, \ce{H2}, \ce{Cl2}$ |

> 💡 **모양이 같아도 극성이 다름**: $\ce{CO2}$ (선형, 무극성) vs $\ce{H2O}$ (굽은형, 극성)

## 10. 용해성 · 대전체 반응 ⭐⭐⭐

### "같은 끼리 잘 녹는다" (유유상통)
| 용질 | 용매 | 잘 녹나? |
|---|---|---|
| 극성 ($\ce{NaCl}$, 설탕) | 극성 (물) | ✓ |
| 극성 ($\ce{NaCl}$) | 무극성 (헥세인) | ✗ |
| 무극성 ($\ce{I2}$, 기름) | 극성 (물) | ✗ |
| 무극성 (기름) | 무극성 (사염화탄소) | ✓ |

### 대전체(전기장) 실험
- **극성 액체** 줄기: 대전체 가까이 → **휘어짐** ✓ (물, 에탄올)
- **무극성 액체** 줄기: 대전체 가까이 → **안 휘어짐** ✗ (헥세인, 사염화탄소, 벤젠)

### 전기장 속 분자 배열
- **극성 분자**: δ+ 쪽 ⟶ (−)극으로, δ− 쪽 ⟶ (+)극으로 **배열**
- **무극성 분자**: 배열 **안 됨** (무작위 방향 유지)

## 11. 전기분해 ⭐⭐

### 물 전기분해 (황산나트륨 소량 첨가)
$$2\ce{H2O} \xrightarrow{\text{전기}} 2\ce{H2} \uparrow + \ce{O2} \uparrow$$

| 극 | 생성 기체 | 부피비 |
|---|---|---|
| **(−)극 (음극)** | $\ce{H2}$ | **2** |
| **(+)극 (양극)** | $\ce{O2}$ | **1** |

> 💡 암기법: "**음**수소, **양**산소, 수소가 **2**배"

### 염화 나트륨 용융 전기분해 $\ce{NaCl(l)}$
| 극 | 반응 | 생성물 |
|---|---|---|
| **(−)극** | $\ce{Na+ + e- -> Na}$ | 나트륨 (금속) |
| **(+)극** | $2\ce{Cl-} \to \ce{Cl2} + 2e^-$ | 염소 기체 |

## 12. 화학 반응식 계수 · 양적 관계 ⭐⭐⭐

### 화학 반응식에서 계수의 의미
**계수비 = 몰비 = 분자수비 = 부피비 (기체)**
**계수비 ≠ 질량비** ⚠️

예시: $\ce{2H2 + O2 -> 2H2O}$
- 분자 수: 2 : 1 : 2
- 몰 수: 2 : 1 : 2
- 기체 부피: 2 : 1 : 2 (수증기일 때)
- 질량: 4g : 32g : 36g (몰질량 곱해야 함)

### 대표 반응식 (시험 빈출)
| 반응 | 식 |
|---|---|
| 수소 연소 | $2\ce{H2} + \ce{O2} \to 2\ce{H2O}$ |
| 메테인 연소 | $\ce{CH4} + 2\ce{O2} \to \ce{CO2} + 2\ce{H2O}$ |
| 에탄올 연소 | $\ce{C2H5OH} + 3\ce{O2} \to 2\ce{CO2} + 3\ce{H2O}$ |
| **암모니아 합성** (하버법) | $\ce{N2} + 3\ce{H2} \to 2\ce{NH3}$ |
| **Mg + 염산** | $\ce{Mg} + 2\ce{HCl} \to \ce{MgCl2} + \ce{H2}$ |
| **탄산칼슘 분해** | $\ce{CaCO3} \to \ce{CaO} + \ce{CO2}$ |
| 염산+수산화나트륨 | $\ce{HCl} + \ce{NaOH} \to \ce{NaCl} + \ce{H2O}$ |

## 13. 한계반응물 판별법 ⭐⭐⭐

### 3-step 루틴
1. 주어진 물질의 **몰** 구하기 ($n = w/M$)
2. 계수로 나눠서 **"계수당 몰" 비교**
3. **작은 쪽**이 한계반응물 (다 쓰고 나머지 남음)

예제: $\ce{Mg} + 2\ce{HCl} \to \ce{MgCl2} + \ce{H2}$
- Mg 0.3 mol, HCl 0.4 mol 반응
- Mg: 0.3/1 = **0.3**
- HCl: 0.4/2 = **0.2** ← 더 작음
- **한계반응물: HCl**
- 생성 H₂: 0.4 × 1/2 = **0.2 mol**
- 남은 Mg: 0.3 − 0.2 = **0.1 mol**

| $\ce{Na}$ | **은백색** (노란 불꽃) | 고체 | - |
| $\ce{Fe}$ | 은회색 (Fe³⁺은 갈색) | 고체 | - |

## 15. 예외 · 함정 포인트 ⭐⭐⭐

### 시험 함정 TOP 10
1. **Cl(3.2) > N(3.0)** — 전기음성도 순서 헷갈리지 말기
2. **C–H는 Δχ=0.4, 거의 무극성** (대칭이면 분자도 무극성)
3. **CO₂는 무극성** (선형 대칭), **H₂O는 극성** (굽은형) — 결합은 둘 다 극성!
4. **계수비 = 몰비지 질량비 아님** — 항상 몰질량으로 변환
5. **22.4 L는 0℃·1atm 조건** — 다른 온도·압력 조건에서는 바꿔야
6. **H는 18족처럼 2개 채우면 안정** (옥텟 아님)
7. **비공유 전자쌍 많을수록 결합각 작아짐** ($\ce{CH4}$ > $\ce{NH3}$ > $\ce{H2O}$)
8. **쌍극자 모멘트 화살표 방향**: + → − (한국 교과서)
9. **전기분해 물**: (−)극 H₂, (+)극 O₂, 부피비 **2:1**
10. **NaCl 용융 전기분해 vs 수용액 전기분해** 구분 — 용융은 Na(s), 수용액은 H₂

### 자주 틀리는 단위 환산
- **0℃ = 273 K** (273.15이지만 시험용 273)
- 1 atm = 760 mmHg = 101.3 kPa
- 1 L = 1000 mL = 1000 cm³
- 몰농도 M = mol/L
- 퍼센트 농도 = (용질 질량 / 용액 질량) × 100

## 17. 1분 직전 암송 체크리스트 ⭐⭐⭐

시험장 들어가기 직전 머릿속으로 1분 안에 돌리기:

- [ ] 아보가드로수 = **6.02 × 10²³**
- [ ] 0℃·1atm 기체 1몰 = **22.4 L**
- [ ] $n = w/M = N/N_A = V/22.4$
- [ ] 전기음성도: **F(4.0) > O(3.4) > Cl(3.2) > N(3.0) > C(2.6) > H(2.2)**
- [ ] VSEPR: 선형 **180°** / 평면삼각 **120°** / 정사면체 **109.5°** / 삼각뿔 **107°** / 굽은형 **104.5°**
- [ ] 극성 상쇄 분자: $\ce{CO2, CH4, CCl4, BF3, BeCl2}$
- [ ] 극성 분자: $\ce{H2O, NH3, HCl, HCN, SO2}$
- [ ] 물 전기분해: (−)극 H₂ **2**부피, (+)극 O₂ **1**부피
- [ ] 계수비 = 몰비 = 분자수비 = 부피비 ≠ **질량비**
