## A-01. 마음의 달 — 천양희 (현대시, 교과서 p12)

- **갈래**: 자유시, 서정시 / **성격**: 성찰적, 위로적, 역설적
- **주제**: 상처받은 사람들의 마음을 위로 / **화자**: 달을 바라보며 사색하는 '나'
- **핵심 시어**:
  - 달빛·환하다·넘어서다 → **긍정**
  - 가시나무 울타리·걸리다·어둠 → **부정**
  - **꺾이다** → 일상에서 '부정' but 시에서 '**긍정**'(욕망을 버림으로써 자유)
- ⭐ **출제 핵심**: **역설법** ("꺾어지는 것은 무릎이 아니라 마음입니다"), 대조적 시어

## A-02. 꽃 — 이육사 (현대시, 교과서 p28)

- **갈래**: 자유시, 서정시 / **성격**: 의지적, 상징적, 저항적
- **주제**: 일제강점기 암울한 상황에서도 광복을 향한 신념과 의지
- **작가 맥락 ⭐**: 이육사 = 독립운동가, 의열단, 옥고 264번 → 호 '육사'
- **핵심 시어**:
  - **꽃 / 꽃 맹아리** → 강인한 생명력, 광복의 희망
  - **제비 떼·꽃성** → 광복 맞이한 미래 조국
  - **회상의 무리들** → 광복의 기쁨 누릴 미래의 우리 민족
  - 하늘 다 끝나고 / 비 한 방울 안 내림 → 일제의 절망적 상황
  - 북쪽 툰드라·눈 속 → 시련
- ⭐ **출제 핵심**: 작가 맥락 = 저항시, 대립적 이미지(절망 vs 희망), 영탄적 어조

## A-03. 낙화 — 조지훈 (현대시, 교과서 p31)

- **갈래**: 자유시, 서정시 / **성격**: 애상적, 관조적, 전통적
- **주제**: 떨어지는 꽃을 보며 느끼는 삶의 비애와 덧없음
- **작가 맥락 ⭐**: 조지훈 = 1940년대 일제 강점기, 조선어 학회 활동, 청록파
  - "묻혀서 사는 이의 고운 마음" = **은둔하는 지사의 고결한 마음**
- **핵심 시어**:
  - **꽃이 지기로소니 / 바람을 탓하랴** → 운명에 대한 체념적 수용
  - 주렴·성긴 별·귀촉도·머언 산 → 애상적 분위기
  - 촛불·미닫이·꽃 지는 그림자 → 정적·고요함
- ⭐ **꽃(이육사) vs 낙화(조지훈)**:
  - 이육사 꽃 = 미래·희망의 의지 / 조지훈 낙화 = 사라짐·비애

## A-04. 이옥설(理屋說) — 이규보 (고전 수필, 교과서 p58)

- **갈래**: **고전 수필 (설說)** = 교술 갈래
- **성격**: 교훈적, 유추적, 경험적
- **주제**: 잘못을 알고 바로 고치면 큰 해를 입지 않는다 (집 수리 → 사람 → 정치)
- **작가**: 이규보 (1168~1241, 고려), 《동국이상국집》
- ⭐ **핵심 표현 기법**:
  - **유추(類推)**: 행랑채 수리 → 사람의 몸 → 나라의 정치
  - **대조**: 비 샌 지 오래된 것 vs 한 번만 맞은 것
- ⭐ **출제 핵심**: '설(說)' 갈래 특징, 유추, 경험과 깨달음의 구조

## A-05. 찬기파랑가(讚耆婆郞歌) — 충담사 (10구체 향가, 교과서 p160)

- **갈래**: **10구체 향가 = 사뇌가**
- **표기**: **향찰 (한자의 음·뜻 빌려 우리말 표기)**
- **주제**: 기파랑(화랑)의 고매한 인품 예찬 + 추모
- **작가**: 충담사 (통일 신라 경덕왕 때, 승려), 《삼국유사》 수록
- **시상 전개**:
  - 1행~5행: 기파랑을 그리워함 (달, 흰 구름)
  - 6행~8행: 기파랑의 인품을 본받고 싶음 (마음의 갓 좇음)
  - 9행~10행: 기파랑 예찬 (잣나무 가지·눈)
- ⭐ **핵심 시어**:
  - **달·흰 구름** → 기파랑의 고매한 모습
  - **잣나무 가지** → 기파랑의 고매한 인품 (높이·곧음)
  - **눈** → 시련 (눈이라도 덮지 못함 = 좌절시킬 수 없음)
  - **고깔(花判)** → 화랑의 표상
- ⭐ **출제 핵심**:
  - **낙구의 '아야'** → 감탄사, **시조 종장 첫머리에 영향**
  - 향찰 표기 ('生死路隱'에서 '隱'은 음 '은')
  - 10구체 = 정형 시가, 사뇌가

## A-06. 서경별곡(西京別曲) — 작자 미상 (고려 속요, 교과서 p166)

- **갈래**: **고려 속요**
- **성격**: 이별가, 적극적, 직설적
- **주제**: 사랑하는 임과의 이별의 슬픔과 임에 대한 변치 않는 사랑
- **화자**: 임을 떠나보내는 여성 ("질삼 뵈" → 길쌈으로 추정)

### 형식적 특징 ⭐⭐⭐ (출제 100%)
- **연장체**: 14개 연으로 구성
- **후렴구**: '위 두어렁셩 두어렁셩 다링디리' 반복
- **3음보** 율격
- 'a-a-b-a' 구조의 변형

### 시상 전개
| 부분 | 내용 |
|---|---|
| 1~4연 | 이별을 거부, 임에 대한 연모 (서경 좋아도 임 따르겠음) |
| 5~8연 | **변함없는 사랑** (구슬-끈 비유) |
| 9~14연 | 사공 원망 (대동강 건넌 임이 꽃을 꺾을까봐) |

- ⭐ **핵심 시어**:
  - **구슬·바위에 떨어져도 끈은 안 끊어짐** → 변치 않는 사랑 (정석가와 동일)
  - **사공** → 임을 데려간 원망의 대상
  - **대동강 건넌편 꽃** → 다른 여자
- ⭐ **출제 함정**: '구슬-끈' 부분이 **정석가와 동일** → 민간 노래가 궁중에서 공유·편입된 흔적

## A-07. 평시조 3편 (교과서 p172~)

### ① 이조년 「이화에 월백하고~」
- **계절**: **봄밤** / **시각적 시어**: 이화(梨花), 월백, 은한
- **주제**: 봄밤의 애상적 정서, 외로움
- **자규(소쩍새)** = 한(恨)·외로움의 매개
- "다정도 병인 양하여" → **다정 → 병** 역설

### ② 성삼문 「수양산 라보며~」
- **사육신** 작가 → **단종 복위 실패**
- **수양산** = 중의적: ① 백이·숙제의 산 ② **수양 대군** (단종 폐위자)
- **이제(夷齊)** = 백이·숙제 (절의의 인물) → 한(恨)
- **푸새(고사리)도 그것이 누구의 것이냐** → 절의·지조
- **주제**: 굳은 절의와 지조

### ③ 황진이 「어져 내 일이야~」
- **이별 후 후회**, 임을 보낸 자신의 마음
- **'어져'** = 감탄사, 안타까움·후회
- **'제 구여'** = 일부러, 애써 → 자존심
- 종장 갈등: **자존심 vs 임에 대한 연정**
- **주제**: 임을 보낸 후의 회한

## A-08. 사설시조 「창 내고쟈~」 (작자 미상, 교과서 p175)

- **갈래**: 사설시조 (조선 후기, 평시조의 변형)
- **주제**: 답답한 마음을 풀고 싶은 절실한 심정
- **화자**: 일상에서 답답함을 느끼는 평민 (계층 확대)
- ⭐ **표현 특징**:
  - **중장의 길이 확장** (평시조와 다름)
  - **사설(辭說)의 나열**: 고모장지·세살장지·들장지·열장지 등 문 종류 열거
  - **기발한 발상**: 가슴에 창을 내겠다 → 추상을 구체로 형상화
  - **해학·골계미**
- ⭐ 평시조와 차이: 형식 파괴 + 작가 계층 확대 (사대부 → 평민)

## A-09. 상춘곡(賞春曲) — 정극인 (가사, 교과서 p178)

- **갈래**: **가사 (정격 가사)**
- **성격**: 자연 친화, 예찬적, 안빈낙도
- **주제**: 봄날 자연의 아름다움과 안빈낙도하는 삶의 즐거움
- **작가**: 정극인 (조선 초기, 1401~1481)
- **형식**: **4음보 연속체**, 마지막 행이 시조 종장과 같은 형식 → 정격 가사

### 구조 (서사-본사-결사)
| 부분 | 내용 |
|---|---|
| 서사(1~6행) | 자연에 묻혀 사는 풍월주인의 삶 ("홍진에 뭇친 분네...") |
| 본사(7~35행) | 봄의 경치 + 화자의 행위 (시비·정자 → **시냇가** → 봉두) |
| 결사(36~39행) | 안빈낙도 만족 ("단표누항에 흣튼 혜음 아니") |

- ⭐ **핵심 시어**:
  - **홍진(紅塵)** → 속세 (멀리하는 대상)
  - **풍월주인** → 자연의 주인 (화자 자신)
  - **물아일체** → 자연과 하나 됨
  - **단표누항** → 청빈한 선비의 삶
  - **무릉(武陵)** → 이상향 (도화 떠내려옴)
- ⭐ **출제**: 가사 갈래 특징, 4음보, 화자의 공간 이동에 따른 시상 전개

## A-10. 누항사(陋巷詞) — 박인로 (가사, 교과서 p181, 유인물 전문)

- **갈래**: **가사** (조선 후기 가사)
- **성격**: 사실적, 자조적, 향토적
- **주제**: **양반이지만 농사를 지어야 하는 곤궁한 삶**의 묘사 + 안빈낙도 추구
- **작가**: 박인로 (1561~1643), 1611년 작
- **유인물 전문**에서 추가 학습 가능

### 핵심 장면
- 가뭄 후 농사 시기 늦음 → 소를 빌리러 친한 집에 감
- 밤늦게 굳게 닫힌 문 앞 → "헛기침으로 에헴" → 어색함
- "어, 거기 누구신가?" → "염치없는 저옵니다" → **자존심 손상**
- "초경(저녁 7~9시)에 왜 왔는가" → "소 없는 가난한 집..."

### 상춘곡 vs 누항사 비교 ⭐
| 구분 | 상춘곡 (정극인) | 누항사 (박인로) |
|---|---|---|
| 시대 | 조선 초기 | 조선 후기 |
| 분위기 | 봄날 자연 예찬, 풍요 | 가뭄·궁핍, 사실적 |
| 화자 | 자연 즐기는 풍월주인 | 곤궁한 농촌 양반 |
| 소재 | 봄 경치 | 소 빌리러 가는 일 |
| 주제 | 안빈낙도 (이상) | 안빈낙도 (현실 속에서) |

- ⭐ **출제**: 사실적 묘사, 일상 어휘, 양반의 처지 변화 (조선 후기 사회 반영)
