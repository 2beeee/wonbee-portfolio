# TIER 분류 — 2시간 이내 정복 우선순위

## 분류 근거
배재고 문학 시험은 **객관식 100%**, 50분에 25~30문항 추정.
보유 자료의 학습 흔적(빨간펜 필기 양, 학습지 유무, 학력평가 명시 등)을 신호로 삼음.

> **신호 우선순위**: 선생님 필기 > 학력평가 명시 > 교과서 분량 > 사설 빈출

---

## TIER S — 반드시 정복 (5작품, 학습 시간 ~60%)

| 작품 | 갈래 | 근거 |
|---|---|---|
| **자경** (박인로) | 연시조 | ⭐ 시험범위에 "3월 학평 자경 포함" 명시 + 학습지 + 빨간펜 필기 |
| **도산십이곡** (이황) | 연시조 | 유인물 별도 + **카톡 필기 3장 풀세트** (선생님 강조 신호) |
| **만복사저포기** (김시습) | 고전 소설 | 교과서 최대 분량 8쪽(p184–192), 작품 5번 본문 |
| **흥보가** (작자미상) | 판소리 | 교과서 6쪽(p196–204), 판소리 갈래 빈출 + 양반전 비교 |
| **무진기행** (김승옥) | 현대소설 | 학습지 4쪽 + 빨간펜 필기 매우 많음 |

> 무진기행은 시험범위 페이지에는 없으나 별도 학습지로 다룬 흔적이 강함.
> 만약 시험범위가 명시 페이지로만 한정된다면 우선순위가 낮아질 수 있음 — 본 평가는 **학습지 보유 = 출제 가능성**으로 판단.

## TIER A — 압축 학습 (10작품, ~30%)
시험범위 명시 + 단원 핵심 작품. 핵심만 빠르게.

- 마음의 달 (천양희) — 1단원 첫 작품, 정서·미적 기능
- 꽃 (이육사) — 작가 맥락, 일제강점기 저항시
- 낙화 (조지훈) — 작가 맥락 비교 (vs 꽃)
- 이옥설 (이규보) — 교술 갈래 대표, **유추**
- 찬기파랑가 (충담사) — 10구체 향가, 향찰, 낙구 '아야'
- 서경별곡 — 고려 속요, 후렴구, 3음보, 정석가와 동일 부분
- 시조 3편: 이조년·성삼문·황진이 — 평시조 형식
- 사설시조 「창 내고쟈」 — 형식 변화
- 상춘곡 (정극인) — 가사 갈래 대표, 안빈낙도
- 누항사 (박인로) — 가사 변화, 곤궁한 양반 (vs 상춘곡)

## TIER B — 한 줄 요약만 (시간 남으면, ~10%)
**참고/함께 읽기 작품들** — 보기 지문으로 활용될 가능성.

- 메밀꽃 필 무렵 부분 / 달밤 / 그 사람의 손을 보면 (1단원 함께 읽기)
- 헌화가 / 처용가 / 제망매가 / 추야우중 (한문/향가 비교)
- 정석가 / 송인 / 한림별곡 / 공방전 (속요·경기체가·가전 비교)
- 이개·홍랑 시조 (시조 함께 읽기)
- 관동별곡 / 규원가 (가사 함께 읽기)
- 슬견설 / 무소유 / 물방울처럼·유리처럼 (수필 함께 읽기)
- 소대성전 / 이생규장전 / 최척전 (고전소설 비교)
- 양반전 (박지원, 흥보가와 묶임)
