# 작품 S-02: 도산십이곡 — 이황

> ⭐⭐⭐ 유인물 + 카톡 필기 3장 풀세트. 선생님 강조 신호 가장 강함.

## 1. 기본 정보
- **갈래**: 평시조, **연시조 (전 12수)**
- **성격**: 교훈적, 회고적, 예찬적, 관조적
- **제재**: 자연(전6곡) + 학문(후6곡)
- **주제**: 자연을 벗삼아 학문에 정진하며 도(道)에 이르는 삶
- **작가**: 이황 (퇴계, 1501~1570)
- **화자**: 자연에 묻혀 학문을 닦는 선비

## 2. 구조 — 핵심 분류
| 부분 | 명칭 | 내용 | 수록 |
|---|---|---|---|
| **전6곡** | **언지(言志)** | 자연에 대한 감회·뜻 | 제1수~제6수 |
| **후6곡** | **언학(言學)** | 학문 수양의 자세 | 제7수~제12수 |

> ⭐ **출제 100% 확정**: 전6곡=언지 / 후6곡=언학 분류 + 그 의미 묻는 문제

## 3. 각 수 핵심 (필기 기준)
| 수 | 핵심 시어/내용 | 주제 |
|---|---|---|
| 제1수 | 초야우생, 천석고황 | 자연 속 살고픈 본성 |
| 제2수 | 연하·풍월, 태평성대, 병으로 늙어가니 | 자연 동화·은거 |
| 제3수 | 순풍·인성, 영재(英才) | 인성·영재에 대한 신뢰 |
| 제4수 | 유란(幽蘭) 재곡, 백설(白雪) 재산, 피미일인(彼美一人) | 자연 예찬 + 임금 그리움 |
| 제5수 | 산전 유대, 교교백구 | 자연에 대한 예찬 |
| 제6수 | 춘풍 화만산, 추야 월만대, 어약연비 | 사계절 자연의 흥취 |
| 제7수 | 천운대·완락재, 만권생애 | 학문에의 즐거움 |
| 제8수 | 뇌정·파산, 백일·중천, **이목총명** 남자 | 학문에 대한 결연한 의지 |
| 제9수 | 고인(古人)도 날 못 보고, 녀던 길 | 옛 성현의 길을 따름 |
| 제10수 | 당시에 녀던 길, 도라온고 | 학문 정진 의지 회복 |
| **제11수** | **청산(靑山)·유수(流水), 만고상청** | ⭐ **변함없는 학문 정진 (불변)** |
| 제12수 | 우부도 알며, 성인도 못다, 쉽거나 어렵거나 | 학문의 평이성·심오함 |

## 4. ⭐ 핵심 시어 의미
| 시어 | 의미 | 출제 핵심 |
|---|---|---|
| **천석고황(泉石膏肓)** | 자연을 사랑하는 마음이 고질병처럼 깊음 | "자연 동경" 매핑 |
| 초야우생 | 시골에 사는 어리석은 사람 (화자의 겸칭) | 화자의 자기 인식 |
| 연하(煙霞) | 안개와 노을 → 자연 | |
| 피미일인 (彼美一人) | 저 아름다운 한 사람 → **임금** | ⭐ 연군지정 |
| **이목총명(耳目聰明)** | 귀 밝고 눈 밝음 → 사리 분별이 뚜렷한 선비 | "남자로 나서" 강조 |
| **고인(古人)** | 옛 성현 (공자·맹자 등) | 본받아야 할 대상 |
| 녀던 길 | 가던 길 → 학문의 길 | |
| **청산** | 변함없음 → 학문의 영원성 | ⭐ 비교: 자연 = 학문 |
| **유수** | 그치지 않음 → 끊임없는 정진 | |
| **만고상청(萬古常靑)** | 영원히 푸름 → 변치 않는 학문 자세 | ⭐ 출제 1순위 |

## 5. 표현상 특징
- **설의법** ("이런들 엇더 며 뎌런들 엇더하료" / "쉽거나 어렵거나 듕에 늙는 줄을 모랄래라")
- **대구법** ("청산은 엇뎨야 만고에 프르르며 / 유수 엇뎨야 주야에 긋디 아니고")
- **연쇄법** (제9수~제10수 '녀던 길' 반복)
- **의문형 종결** — 화자의 다짐·확신 강조
- **자연물에 인격 부여** — 청산·유수 = 학문 정진의 표상

## 6. ⭐ 선생님 강조 포인트 (카톡 필기)
1. **전6곡 言志 / 후6곡 言學** — 분류와 그 의미 100% 출제
2. **제11수 '청산-만고상청', '유수' 의미** — 가장 빈출
3. 제4수 **'피미일인' = 임금** (연군지정)
4. 제8수 **'이목총명 남자' = 결연한 학문 의지**
5. **고인 = 옛 성현 (공자·맹자)** — 절대 "옛 친구"로 오해 X
6. **연시조의 형식적 안정감** vs 사설시조 (대비)

## 7. 함정/오답 유도 패턴
- ❌ "임에 대한 사랑" → ✗ 학문·자연 (피미일인은 임금)
- ❌ "은둔하며 세상을 비판" → ✗ 적극적 학문·도 추구
- ❌ "연시조 제○수만 단독 작품" → ✗ 12수 전체가 한 작품
- ❌ "제1수가 言學" → ✗ 言志(전6곡)
- ❌ "청산·유수 = 자연 풍경 감상" → ✗ 변함없는 학문 자세 상징

## 8. 자주 묶이는 비교 작품
- **자경** (박인로) — 둘 다 유교적 자기 수양 연시조
- **상춘곡** (정극인) — 자연 속 삶, 안빈낙도 (vs 도산십이곡 = 학문)
- **누항사** (박인로) — 양반의 곤궁 (vs 도산십이곡 = 정신적 충만)
