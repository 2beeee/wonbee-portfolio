# 작품 S-04: 흥보가(興甫歌) — 작자 미상

> ⭐⭐ 교과서 6쪽 분량(p196–204). 판소리 갈래 빈출. 양반전과 묶임 가능.

## 1. 기본 정보
- **갈래**: **판소리** (구전 → 기록)
- **성격**: 풍자적, 해학적, 비판적, 사실적
- **시점**: 전지적 (서술자 개입 多)
- **배경**: 조선 후기 (신분 질서 붕괴, 화폐 경제)
- **제재**: 흥보의 매품팔이
- **주제**: 궁핍한 양반의 처지 + 화폐 경제 중시 + 신분 질서 약화
- **출전**: 판소리 5마당 중 하나

## 2. 구조
| 부분 | 내용 |
|---|---|
| 발단 | 흥보가 환자(곡식)을 얻으러 읍내로 감 |
| 전개 | 호방이 매품팔이(좌수 대신 곤장 맞기)를 권함 |
| 절정 | 흥보가 돈에 흥분 → 마누라가 만류 |
| 결말 | 옆집 꾀수 애비가 먼저 매품 팔아 흥보 헛걸음 |

## 3. ⭐ 판소리 구성 (출제 핵심)
| 구성 요소 | 정의 |
|---|---|
| **창(唱)** | 가락에 맞춰 노래하는 부분 |
| **아니리** | 음률·장단 없이 말로 하는 부분 |
| **너름새/발림** | 몸짓/표정 (이 자료엔 없음) |

### 장단 (느림 → 빠름)
**진양조 → 중모리 → 중중모리 → 자진모리 → 휘모리**

| 장단 | 분위기 | 작품 내 사용 |
|---|---|---|
| **진양조** | 가장 느림, 슬픈/장중한 감정 | "가지 마오 가지 마오" — 마누라의 만류 |
| 중모리 | 보통 | 평화/일상 서술 |
| 중중모리 | 약간 빠름 | 흥겨움 |
| **자진모리** | 빠름 | 흥보 치레 (남루한 행색 묘사) |
| 휘모리 | 가장 빠름 | 긴박한 장면 |

## 4. 핵심 소재·인물
| 요소 | 의미 |
|---|---|
| **환자(還子)** | 봄에 곡식 빌려주고 가을에 이자 붙여 거두던 제도 |
| **매품(매품팔이)** | 돈 받고 대신 곤장 맞는 일 → **궁핍한 양반의 처지** 상징 |
| **돈(엽전)** | 화폐 경제, 양반의 권위 추락 |
| **호방** | 아전, 흥보를 "박 생원"이라 부름 (양반 호칭) |
| 흥보 | 가난한 양반, 매품 팔러 가는 처지 |
| 흥보 마누라 | 매품을 만류 → 가족애 |

## 5. ⭐ 핵심 표현 (자주 출제)
- **돈타령** ("돈 봐라 돈... 삼강오륜이 다 보이고...") → 화폐 만능주의
- **흥보 치레** (자진모리) → 남루한 행색의 사실적·해학적 묘사
- **언어유희**: "곯다니 계란이 곯지, 사람이 곯나"
- **양반의 자조적 어조**: "그래도 양반이라고 여덟팔자걸음"
- **반복**: "가지 마오 가지 마오" (진양조의 슬픔)

## 6. ⭐ 사회적 상황 반영 (출제 3순위)
1. **궁핍한 양반** — 흥보가 양반인데도 매품 팔아야 함
2. **신분 질서 약화** — 호방이 양반에게 품팔이 권유
3. **화폐 경제 중시** — 돈에 대한 흥보의 흥분
4. **매품팔이 풍속** — 실제 존재했던 사회 현상

## 7. 함정/오답 유도 패턴
- ❌ "고전 시가다" → ✗ 판소리(구비문학·서사)
- ❌ "흥보가 매품을 팔았다" → ✗ 옆집 꾀수 애비가 먼저 가서 못 팔았음
- ❌ "흥보가 부자다" → ✗ 매우 가난함
- ❌ "양반의 권위가 강하다" → ✗ 약화됨
- ❌ "진양조 = 빠른 장단" → ✗ 가장 느린 장단
- ❌ "창만으로 구성" → ✗ 창 + 아니리

## 8. 자주 묶이는 비교 작품
- **양반전** (박지원) — 양반의 위선 풍자, 같은 시대 사회 비판
  - 흥보가: 궁핍한 양반의 비극·해학 / 양반전: 양반 신분 매매의 풍자
- 다른 판소리 작품 (춘향가, 심청가 등) — 갈래 비교
