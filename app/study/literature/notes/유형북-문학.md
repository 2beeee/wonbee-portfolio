# 유형북 — 문학 객관식 풀이 알고리즘

> 시험 = **객관식 100점**. 모든 문제는 5~7개 유형 안에 들어간다.
> "이런 문제는 이렇게 푼다" 알고리즘 + **함정 패턴**.

---

## 유형 1. 화자의 정서·태도 ⭐⭐⭐ (출제 1순위)

### 풀이 5단계
1. **화자가 누구인지** 먼저 확정 (성별·신분·상황). 작품 카드에서 확인.
2. 화자가 처한 **상황·시간**을 한 줄로 정리. (예: "임과 이별 후", "자연에 은거")
3. **반복되는 시어·종결어미** 체크 → 정서 단서.
   - "~하리라/~노라" = 의지 / "~하랴/~ㄴ가" = 설의·체념
   - "아!", "어져" = 감탄·후회
4. 보기에서 **반대 정서**부터 소거.
5. **단정적 표현** 주의. "체념적이다 ✓" vs "절망적이다 ✗" 같은 강도 차이.

### 함정 패턴
- ❌ "체념" vs "절망" 강도 혼동
- ❌ "그리움" vs "원망" — 같은 사랑이라도 시점에 따라 다름
- ❌ 표면적 슬픔만 보고 **이면의 의지**를 놓침 (이육사 「꽃」, 윤동주 「서시」)
- ❌ **자연 예찬 ≠ 현실 도피** — 도산십이곡은 도피 아닌 학문 정진

### 빈출 정서 대응표
| 작품 | 화자 정서 | 핵심 단서 |
|---|---|---|
| 자경 | 자기 경계, 다짐 | 명령형, 의지 종결 |
| 도산십이곡 | 학문 정진의 의지 | 만고상청, 청산 |
| 무진기행 | 부끄러움·자기 분열 | 안개, 회귀 |
| 서경별곡 | 적극적 만류, 변치 않는 사랑 | "여히므론", "괴시란" |
| 누항사 | 자조·체념·연민 | "어, 거기 누구신가" |
| 황진이 시조 | 후회·자존심 | "어져 내 일이야" |

## 유형 2. 시어·시구의 함축적 의미 ⭐⭐⭐ (출제 2순위)

### 풀이 5단계
1. 시어를 **밑줄·기호로 분류**: 긍정/부정/중립.
2. 같은 시 안에서 **대립 관계** 확인. (낮 vs 밤, 꽃 vs 가시)
3. **작가 맥락** 적용. (이육사 = 저항시 → 꽃 = 광복)
4. 보기 4개 중 **사전적 의미**만 쓴 보기는 함정 가능성 높음.
5. **시어 ≠ 같은 단어의 일상 의미**. 시 안에서의 역할로 판단.

### 함정 패턴
- ❌ "꽃 = 아름다움" → 이육사에서는 광복 의지
- ❌ "꺾이다 = 부정" → 「마음의 달」에서는 긍정 (역설)
- ❌ "안개 = 신비" → 무진기행에서는 체념·자기 상실
- ❌ "수양산 = 그냥 산" → 성삼문 시조에서는 수양대군 중의

### 자주 묶이는 대립 시어
| 긍정 | 부정 |
|---|---|
| 달빛, 꽃, 청산 | 어둠, 가시, 안개 |
| 만고상청, 잣나무 | 눈(시련), 풍상 |
| 무릉, 풍월주인 | 홍진(속세) |

## 유형 3. 표현상 특징 ⭐⭐⭐ (출제 3순위, 함정 多)

### 풀이 5단계
1. **수사법 체크리스트** 머리에 두고 보기 5개와 매칭.
2. **'~한 표현이 사용되었다'** → 본문에 그 표현이 실제 있는지 확인. 없으면 즉시 소거.
3. **'~한 효과를 거두고 있다'** → 효과까지 정확해야 정답.
4. **연쇄법·반복·대구**는 행 단위로 확인.
5. **모르는 수사법** 보기는 정답 가능성 ↓ (출제자가 자주 쓰는 함정).

### 핵심 수사법 빈출 표
| 수사법 | 정의 | 작품 예 |
|---|---|---|
| **역설** | 표면 모순 → 진실 | "꺾어지는 것은 무릎이 아니라 마음", "다정도 병인 양" |
| **반어** | 의도와 반대로 | "죽어도 아니 눈물" (진달래꽃) |
| **설의** | 의문문 형식의 강조 | "바람을 탓하랴" (낙화) |
| **대구** | 비슷한 구조 짝 | "청산은 만고에 프르며 / 유수는 주야에 긋디 아니" |
| **연쇄** | 앞 구절 끝말 → 다음 시작 | 도산십이곡 9~10수 '녀던 길' |
| **유추** | 비슷한 두 사물 추론 | 이옥설 (행랑채 → 정치) |
| **의인** | 사물 인격화 | "청산", "유수" |
| **영탄** | 감탄 | "아!", "어져" |
| **점층** | 점점 강함 | 사설시조 |
| **언어유희** | 말 재미 | "곯다니 계란이 곯지" (흥보가) |

### 함정 패턴
- ❌ "역설" vs "반어" 혼동 — 역설=표면 모순, 반어=의도 반대
- ❌ "직유법이 사용됨" → '같이/처럼' 없으면 ✗
- ❌ "감각의 전이(공감각)" 함부로 인정 X — 시각→청각 등 명확해야 함
- ❌ "대구" → 단순 반복 ≠ 대구. 구조까지 닮아야 함

## 유형 4. 외적 준거 (작가/시대 맥락) ⭐⭐ (출제 4순위)

### 풀이 5단계
1. **〈보기〉 먼저 정독**. 거기 단서가 다 있다.
2. 〈보기〉의 **키워드** (작가 정보, 시대, 갈래 특성) 동그라미.
3. 보기 5개 중 **〈보기〉를 잘못 적용**한 것을 찾는 문제가 70%.
4. 본문 → 〈보기〉 → 선택지 순서로 일치 검증.
5. **확대 해석** 주의. 〈보기〉에 없는 내용 끌어오면 ✗.

### 자주 등장하는 외적 준거
| 작품 | 외적 준거 키워드 |
|---|---|
| 이육사 「꽃」 | 독립운동가, 의열단, 옥고 17번 |
| 조지훈 「낙화」 | 일제 말, 조선어학회, 청록파, 은둔하는 지사 |
| 김시습 「만복사저포기」 | 생육신, 단종 폐위, 좌절감, 환상으로 위안 |
| 박인로 「누항사」 | 임진왜란 후, 양반의 곤궁한 현실, 조선 후기 사회 변화 |
| 흥보가 | 조선 후기, 신분 질서 약화, 화폐 경제 |
| 도산십이곡 | 이황, 도산서원, 성리학, 학문 수양 |

### 함정 패턴
- ❌ 〈보기〉에 없는 작가 일대기를 끌어와 정답으로 고름
- ❌ "광복을 직접 체험" → 이육사는 광복 직전 옥사. 체험 ✗
- ❌ "조지훈은 적극 저항" → 그는 은둔·내면적 비애

## 유형 5. 작품 간 비교 ⭐⭐ (단골)

### 풀이 5단계
1. 두 작품의 **공통점** 1개 + **차이점** 1개를 빠르게 확정.
2. 보기는 보통 "공통점이다" / "A에만 해당" / "B에만 해당" 형식.
3. **갈래·시대·화자·정서·주제** 5개 축으로 비교.
4. 표 형식의 작품 카드 비교 부분 활용.
5. "둘 다 ~한다" 보기는 한쪽이라도 ✗면 오답.

### 빈출 비교 묶음
| 묶음 | 공통점 | 차이점 |
|---|---|---|
| **꽃(이육사) vs 낙화(조지훈)** | 꽃 소재, 일제강점기 | 의지·희망 vs 비애·체념 |
| **상춘곡 vs 누항사** | 가사 갈래, 안빈낙도 지향 | 풍요·예찬 vs 곤궁·사실 |
| **서경별곡 vs 가시리** | 고려 속요, 이별의 정한 | 적극적 만류 vs 절제된 이별 |
| **흥보가 vs 양반전** | 조선 후기, 양반 비판 | 해학·동정 vs 풍자·비판 |
| **이옥설 vs 슬견설** | 이규보, 설(說), 유추 | 정치 교훈 vs 생명 평등 |
| **만복사저포기 vs 이생규장전** | 김시습, 금오신화, 명혼 | — |
| **자경 vs 도산십이곡** | 유교 수양, 연시조 | 자기 경계 vs 학문 정진 |

## 유형 6. 갈래 특징 묻기 ⭐⭐ (고전에서 빈출)

### 갈래별 핵심 특징표

| 갈래 | 핵심 특징 (출제 포인트) |
|---|---|
| **향가** | 향찰 표기, 4·8·10구체, 10구체=사뇌가, 낙구 '아야' → 시조 종장 |
| **고려 속요** | 분연체(연장체), 후렴구, 3음보, 평민 노래가 궁중 편입 |
| **평시조** | 3장(초·중·종) 6구 45자 내외, 종장 첫구 3자 고정 |
| **사설시조** | 중장 길이 확장, 평민·여성 작가, 해학·골계 |
| **연시조** | 여러 수가 한 작품 (도산십이곡=12수, 어부사시사=40수) |
| **가사** | 4음보 연속체, 정격(종장 시조형) vs 변격, 산문성 |
| **판소리** | 창+아니리+발림, 5마당, 적층문학, 5장단 (진양조→휘모리) |
| **고전 소설(전기)** | 한문, 비현실, 시 삽입, 영웅 일대기 |
| **설(說)** | 교술 산문, 경험→깨달음, 유추 |

### 함정 패턴
- ❌ "고려 속요는 한문 표기" → ✗ 한글 (훈민정음 후 기록)
- ❌ "향가는 한문" → ✗ 향찰
- ❌ "판소리는 창만으로 구성" → ✗ 창+아니리
- ❌ "사설시조는 평시조보다 먼저 등장" → ✗ 조선 후기 변형

## 유형 7. 서술상 특징 (소설) ⭐ (소설 단골)

### 풀이 5단계
1. **시점** 확정. (1인칭 주인공/관찰자, 3인칭 전지/관찰)
2. **서술자 개입** 여부 (판소리·고전 소설은 多).
3. **시간 구성**: 순행/역순행/회상.
4. **공간 이동/인물 갈등** 구조 확인.
5. 보기 5개 중 **본문에 없는 기법** 즉시 소거.

### 작품별 서술 특징 압축
| 작품 | 시점 | 특징 |
|---|---|---|
| 무진기행 | 1인칭 주인공 (윤희중) | 회상·내적 독백, 감각적 문체 |
| 만복사저포기 | 전지적 작가 | 시 삽입, 환상성 |
| 흥보가 | 전지적 (서술자 개입 多) | 창+아니리, 편집자적 논평 |

### 함정 패턴
- ❌ "무진기행은 3인칭" → ✗ 1인칭
- ❌ "전지적 시점이라 인물 심리 모름" → ✗ 다 안다
- ❌ "1인칭 관찰자 = 주인공의 시각" → ✗ 주변 인물 시각

## 유형 8. 시상 전개·구조 ⭐

### 풀이 단계
1. **공간 이동**(상춘곡), **시간 변화**(향수), **시선 이동**(찬기파랑가), **감정 변화** 중 무엇인지 결정.
2. 행/연 단위로 **나누기**.
3. 보기에서 단계를 거꾸로 적은 함정에 주의.

### 빈출 시상 전개
| 작품 | 전개 방식 |
|---|---|
| 상춘곡 | 화자의 공간 이동 (집→정자→시냇가→봉두) |
| 찬기파랑가 | 시선 이동 (달→흰 구름→잣나무) |
| 도산십이곡 | 전6곡 자연 → 후6곡 학문 (병렬) |
| 무진기행 | 서울→무진→서울 (순환) |
| 누항사 | 시간 순행 (가뭄→소 빌리러→귀가) |

## 유형 10. 자료 통합 (학습활동·복합지문) ⭐

### 풀이 단계
1. **〈보기〉 먼저** 읽고 키워드 확보.
2. 본문에서 〈보기〉 키워드와 연결되는 부분 표시.
3. 선택지 5개 검증 — **본문 + 보기 모두 만족**해야 정답.
4. 확대 해석·축소 해석 모두 함정.

## 🚨 시험 직전 체크리스트 30초

1. **갈래 헷갈리지 않기**: 향가/속요/시조/가사/판소리 구별
2. **유사 작품 비교**: 꽃-낙화 / 상춘곡-누항사 / 서경별곡-가시리
3. **작가 맥락 키워드**: 이육사=의열단 / 김시습=생육신 / 조지훈=청록파
4. **수사법 정확히**: 역설 ≠ 반어
5. **함정 단어**: '단정', '주된', '오로지', '항상', '결코' → 거의 ✗
6. **외적 준거**: 〈보기〉에서 벗어난 해석 ✗
7. **이황 도산십이곡**: 전6곡 言志 / 후6곡 言學 / 만고상청 / 청산·유수
8. **자경(박인로)**: 학력평가 명시 → 시험 출제 확률 100%
