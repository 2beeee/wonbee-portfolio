# 작품 S-05: 무진기행(霧津紀行) — 김승옥

> ⭐ 학습지 4쪽 + 빨간펜 필기 매우 많음. 시험범위에는 명시되지 않으나 학습 흔적 강력함 — 출제 가능성 보유.

## 1. 기본 정보
- **갈래**: 현대 단편 소설
- **성격**: 회상적, 자기 고백적, 감각적, 상징적
- **시점**: **1인칭 주인공 시점** (윤희중의 시선)
- **배경**: 1960년대, 무진(霧津) — 가공의 도시
- **제재**: 무진으로의 여행
- **주제**: 일상에서 벗어나 자아를 성찰하지만 결국 현실(서울)로 회귀하는 현대인의 내면
- **작가**: 김승옥 (1941~)
- **출전**: 1964년 발표

## 2. 등장인물
| 인물 | 역할 | 특징 |
|---|---|---|
| **윤희중** | 주인공·서술자 | 제약회사 전무 후보, 무진 출신, 아내(장인의 회사 추천)로 사회적 성공 앞둠 |
| **아내** | 서울에 있음 | 윤희중을 무진으로 보내 휴식 권유 (현실 세계 대표) |
| **박** | 후배 | 무진중학교 교사, 윤희중을 존경 |
| **조** | 동창 | 세무서장, 출세, 속물적 |
| **하인숙(하 선생)** | 음악 교사 | 윤희중의 과거의 자아·이상의 분신, 서울로 가고 싶어 함 |
| **미친 여자** | — | 윤희중의 과거 트라우마와 연결 |

## 3. ⭐ 공간의 상징성 (출제 핵심)
| 공간 | 상징 |
|---|---|
| **무진(霧津)** | 안개의 도시 → **자기 내면, 과거, 무의식, 도피처, 상실/체념의 공간** |
| **안개** | 무진의 명물 → **체념, 자기 상실, 현실 회피, 모호함** |
| **서울** | 일상, 책임, 세속적 성공, 사회적 자아 |
| **버스(여행)** | 무진 ↔ 서울 사이의 이동 → 자아의 두 영역 |
| **방죽길** | 윤희중의 과거 회상이 일어나는 공간 |

## 4. ⭐ 핵심 구조 (서울 → 무진 → 서울 회귀)
1. **무진으로 가는 버스** — 일상에서 벗어나는 시작 (반수면 상태)
2. **무진 도착·후배·동창과 만남** — 과거의 자아와 조우
3. **하인숙과의 만남** — 자기 분신과의 동일시
4. **방죽길의 과거 회상** — 자기 성찰
5. **아내의 전보 → 서울로 귀환** — 현실로의 회귀, 자아 분열의 자각

## 5. ⭐ 빨간펜 필기 강조 포인트
1. **'안개' = 체념/자기 상실**
2. **무진 = 회피의 공간**, but **결국 다시 사회(서울)로 돌아감**
3. 등장인물 = 윤희중의 분신
   - 박 → 과거의 순수했던 자아
   - 조 → 속물화된 현재의 자아 (출세 욕망)
   - 하인숙 → 무진을 떠나고 싶어 하는 욕망의 분신
4. **유행가** = 하인숙이 부르는 무진의 분위기 = 사회의 울림 매개
5. **수면제·반수면 상태** = 윤희중의 자기 마비/회피 상태
6. **사이렌 소리** = 모든 사고를 흡수해 버리는 단조로움
7. **부끄러움** — 무진을 떠날 때 윤희중의 자아 인식

## 6. 표현상 특징
- **감각적 문체** — 시각·청각 이미지 풍부 (안개·바람·소금기·산소)
- **회상 기법** — 과거와 현재 교차 (서울/무진, 어머니 시절/현재)
- **자기 고백적** 1인칭 서술
- **상징적 소재** 다수 (안개·버스·전보)
- **내적 독백·심리 묘사** 비중 큼

## 7. 함정/오답 유도 패턴
- ❌ "전지적 작가 시점" → ✗ 1인칭 주인공
- ❌ "무진은 실재 도시" → ✗ 가공의 도시
- ❌ "무진에 정착한다" → ✗ 결국 서울로 돌아감
- ❌ "안개 = 신비로움" → ✗ 체념·자기 상실
- ❌ "윤희중의 변화 = 적극적 결단" → ✗ 자아 분열의 자각, 회귀
- ❌ "하인숙은 무진의 행복을 상징" → ✗ 떠나고 싶어 하는 욕망
