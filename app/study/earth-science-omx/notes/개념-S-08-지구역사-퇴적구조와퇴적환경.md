# 개념 S-08 — 퇴적 구조·퇴적 환경

## 1. 한 줄 정의
퇴적 구조는 **환경을 역으로 추적하는 단서**다.

## 2. 핵심 메커니즘
- 점이 층리: 아래 굵고 위 가늘다 → 깊은 물
- 사층리: 비스듬한 층리 → 흐름 방향 알 수 있음
- 연흔: 물결 흔적 → 얕은 물
- 건열: 갈라짐 → 지표 노출·건조

## 3. 공식 / 관계식
- 깊은 물 = 점이 층리
- 얕은 물 = 연흔
- 건조 노출 = 건열
- 흐름 방향 = 사층리

## 4. 그래프 / 도표
```text
점이층리 : 아래 굵고 위 가늘다
사층리   : 비스듬한 줄무늬
연흔     : 물결 모양
건열     : 위가 넓은 갈라짐
```

## 5. 자주 나오는 함정
- 점이 층리 위아래 뒤집기
- 연흔과 사층리 혼동
- 건열을 수중 구조로 착각

## 6. ⭐ 선생님 강조 포인트
**사진형은 생김새 키워드부터 외울 것**

## 7. 전형적 문항 형태
- 퇴적 구조 사진 판별
- 환경 연결(사막/삼각주/해빈/호수)
- 지층 상하 판단
