# 우선순위 분류

## TIER S — 반드시 정복
| 개념 | 근거 |
|---|---|
| 해수 수온·염분·밀도·T-S도 | 교사 학습자료 1~5p 전체, 메가/완자 반복, 그래프 최빈출 |
| 표층/심층 순환·수괴 | 지도형 + 용승/침강 + T-S 연결 반복 |
| 중위도 저기압·전선·일기도·위성/레이더 | 자료 해석형 핵심, 보기 ㄱㄴㄷ 빈출 |
| 태풍 구조·위험반원 | 기압/풍속 그래프 빈출 |
| 엘니뇨·라니냐·워커 순환 | 비교표 + 지도 + 원인-결과 연쇄 반복 |
| 지사학 법칙·상대연령 | 단면 해석형 점수 효율 높음 |
| 절대연령·반감기 | 계산형. 틀리면 1문제 통째로 손실 |
| 퇴적 구조·퇴적 환경 | 사진/모식도 판별형 고빈출 |

## TIER A — 핵심만 압축
| 개념 | 근거 |
|---|---|
| 악기상(우박/집중 호우/폭설/황사) | 반복은 많지만 원인 암기형 비중 큼 |
| 기후 변화·복사 평형 | 자료는 자주 나오나 S보다 직접 계산형 비중 낮음 |
| 화석·지질 시대 | 표/매칭형으로 회수 가능 |
| 산소 동위원소·고기후 | 헷갈리기 쉬우나 범위 비중은 제한적 |
| 해양 관측·ARGO | 짧게 외우면 먹히는 정의형 |

## TIER B — 시간 남으면
| 개념 | 이유 |
|---|---|
| 용존 기체 | 핵심 관계 한 줄이면 충분 |
| 퇴적암 생성과 종류 | 구조/환경보다 우선도 낮음 |

## 짧은 판단 근거
- **필기(=교사 학습자료) > 교과서 > 문제집** 순으로 반영
- 문제집 해설 분석상 반복도가 특히 높은 축은
  - 해수 성질
  - 중위도 저기압/전선
  - ENSO
  - 지층 해석
  - 퇴적 구조
- 따라서 밤 2시간은 **S 우선 + A 압축 + B 버리기**로 간다
