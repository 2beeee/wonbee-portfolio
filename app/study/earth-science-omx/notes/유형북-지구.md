## 1. 그래프/도표 해석
**출제 빈도:** ★★★★★

### 풀이 알고리즘
1. 축과 단위를 먼저 본다.
2. 증가/감소 방향을 한 문장으로 말한다.
3. 기울기 급한 구간을 찾는다.
4. 점 비교면 수온 먼저, 염분 다음으로 본다.
5. 보기와 대조한다.

### 함정 패턴
- 혼합층을 변화 큰 층으로 착각
- 적도 표층 염분이 최고라고 오답
- T-S도 밀도 증가 방향 반대로 읽음

### 예제
- 메가 021, 027, 029
- 완자 016, 066

## 2. 해류/순환 지도형
**출제 빈도:** ★★★★★

### 풀이 알고리즘
1. 바람 자료인지 수온·염분 자료인지 구분
2. 난류/한류 방향 표시
3. 침강/용승 지역 표시
4. 기후 영향 붙이기

### 함정 패턴
- 표층 순환 원인과 심층 순환 원인 뒤집기
- 용승 지역을 고수온으로 읽기
- 수괴 위치 혼동

### 예제
- 메가 051~059
- 메가 173~174
- 완자 164~180

## 3. 중위도 저기압/전선 종합형
**출제 빈도:** ★★★★★

### 풀이 알고리즘
1. 저기압 중심과 전선 위치 먼저 표시
2. 한랭/온난 전선 표를 바로 대입
3. 통과 전후 날씨를 읽음
4. 순서 배열이면 정체→파동→발달→폐색

### 함정 패턴
- 한랭/온난 전선 속도 바꾸기
- 기압 변화 방향 반대로 쓰기
- 폐색전선 오판

### 예제
- 메가 087~091
- 완자 106, 120

## 4. 위성/레이더 영상형
**출제 빈도:** ★★★★☆

### 풀이 알고리즘
1. 낮/밤 여부 체크
2. 가시/적외/레이더 분류
3. 구름인지 강수인지 구분
4. 높이/강도까지 연결

### 함정 패턴
- 밝음=강수라고 단정
- 가시 영상을 밤 자료로 선택
- 레이더 영상을 구름 영상으로 착각

### 예제
- 메가 082~083
- 완자 146, 148

## 5. 태풍/악기상 자료형
**출제 빈도:** ★★★★☆

### 풀이 알고리즘
1. 태풍인지 전선성 악기상인지 구분
2. 태풍이면 눈/눈벽/위험반원 판정
3. 악기상이면 발생 조건 키워드 찾기
4. 피해 양상 연결

### 함정 패턴
- 눈에서 풍속 최대라고 답함
- 위험반원 위치 반대로 답함
- 우박/폭설 생성 조건 뒤섞음

### 예제
- 메가 124~131
- 완자 124, 146~149

## 8. 반감기 계산형
**출제 빈도:** ★★★★☆

### 풀이 알고리즘
1. 남은 비율인지 생성물 비율인지 구분
2. 1/2, 1/4, 1/8 꼴로 변환
3. 반감기 횟수 n 계산
4. 반감기 × n

### 함정 패턴
- 생성물/모원소 혼동
- 1/8을 8번 반감기로 계산
- 반감기가 외부 조건에 따라 바뀐다고 선택

### 계산 예시
- 1/8 남음 → 3번 반감기
- 반감기 5000년 → 15000년

### 예제
- 메가 268~272
- 완자 268~272

## 10. 화석/지질 시대 매칭형
**출제 빈도:** ★★★☆☆

### 풀이 알고리즘
1. 표준/시상 화석부터 구분
2. 시대 대표 생물로 매칭

- 표준/시상 역할 뒤집기

### 예제
- 메가 326~352
- 완자 326~338
