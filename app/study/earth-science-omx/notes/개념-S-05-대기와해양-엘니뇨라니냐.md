# 개념 S-05 — 엘니뇨·라니냐·워커 순환

## 1. 한 줄 정의
ENSO는 **무역풍-용승-수온-기압 차**가 같이 움직이는 현상이다.

## 2. 핵심 메커니즘
- 평상시: 무역풍 강함 → 서태평양 고수온 / 동태평양 용승
- 엘니뇨: 무역풍 약화 → 동태평양 수온↑, 용승 약화
- 라니냐: 무역풍 강화 → 동태평양 수온↓, 용승 강화
- 워커 순환 상승/하강 지역도 동서로 이동

## 3. 공식 / 관계식
- 무역풍 약화 = 엘니뇨
- 무역풍 강화 = 라니냐

## 4. 그래프 / 도표
```text
평상시 : 서쪽 warm pool, 동쪽 용승
엘니뇨 : 경사 완만, 동쪽 수온↑
라니냐 : 경사 급함, 동쪽 수온↓
```

## 5. 자주 나오는 함정
- 동태평양 수온 상승을 라니냐로 착각
- 무역풍과 용승 변화를 반대로 연결
- 워커 순환 상승 위치 혼동

## 6. ⭐ 선생님 강조 포인트
**평상시/엘니뇨/라니냐 3장 그림을 세트로 외울 것**

## 7. 전형적 문항 형태
- 3상태 비교표
- 워커 순환 위치 판정
- 지역별 기후 영향 추론
