# 개념 S-03 — 중위도 저기압·전선·위성/레이더 해석

## 1. 한 줄 정의
날씨 문제는 **전선 종류 + 저기압 위치 + 자료 종류**를 동시에 맞히는 문제다.

## 2. 핵심 메커니즘
- 찬 공기와 따뜻한 공기 충돌 → 전선 형성
- 한랭전선: 빠름 / 좁고 강한 비 / 기온↓
- 온난전선: 느림 / 넓고 약한 비 / 기온↑
- 가시 영상 = 햇빛 반사 / 적외 = 구름 꼭대기 온도 / 레이더 = 강수 세기

## 3. 공식 / 관계식
- **한랭전선 통과 후:** 기온↓, 풍향 남서→북서, 기압↑
- **온난전선 통과 후:** 기온↑, 풍향 남동→남서

## 4. 그래프 / 도표
```text
한랭전선: 기울기 급함 / 적운형 / 빠름 / 좁은 비
온난전선: 기울기 완만 / 층운형 / 느림 / 넓은 비
레이더: 강수
가시: 낮의 구름 반사
적외: 밤에도 가능, 구름 꼭대기 높이 판단
```

## 5. 자주 나오는 함정
- 가시 영상과 적외 영상 혼동
- 한랭전선 뒤쪽/앞쪽 강수 위치 혼동
- 기압 변화 방향 반대로 씀

## 6. ⭐ 선생님 강조 포인트
**자료가 2개 이상 나오면 무슨 자료인지부터 판정하고 해석할 것**

## 7. 전형적 문항 형태
- 일기도 + 위성/레이더 종합
- 전선 통과 전후 날씨
- 중위도 저기압 일생 순서 배열
