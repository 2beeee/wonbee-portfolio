# 개념 S-02 — 표층 순환·심층 순환·수괴

## 1. 한 줄 정의
표층은 **바람**, 심층은 **밀도 차**가 움직인다.

## 2. 핵심 메커니즘
- 무역풍·편서풍 → 표층 순환
- 극지방 냉각·결빙 → 밀도↑ → 침강
- 표층+심층 연결 = 전 지구적 컨베이어 벨트
- 용승 지역 = 차고 영양염 풍부

## 3. 공식 / 관계식
- 표층 순환 = 바람
- 심층 순환 = 밀도 차

## 4. 그래프 / 도표
```text
저위도 난류 → 고위도 열 수송
고위도 냉각·결빙 → 침강
동태평양 연안/적도 동부 → 용승
대서양 북부 / 남극 주변 → 심층수 형성
```

## 5. 자주 나오는 함정
- 표층 순환 원인을 밀도 차로 답함
- 용승 지역을 고수온으로 읽음
- 북대서양 심층수 vs 남극 저층수 혼동

## 6. ⭐ 선생님 강조 포인트
**지도에서 차가운 물이 올라오면 수온↓·영양염↑**

## 7. 전형적 문항 형태
- 해류 지도에서 난류/한류 판별
- 수괴 T-S도 연결
- 용승/침강의 기후 영향 추론
