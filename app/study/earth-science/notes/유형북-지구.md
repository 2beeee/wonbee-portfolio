# 유형북 — 2026 1학기 중간 지구과학

각 유형별 **풀이 알고리즘 + 함정 패턴 + 체크리스트**. 시험장에서 머릿속으로 이 순서를 복창하며 푼다.

---

## 유형 1. 연직 수온 분포 그래프 (혼합층/수온약층/심해층)

**풀이 알고리즘**
1. 수면~40m 부근 수온 거의 일정한 층 → **혼합층**
2. 깊어질수록 수온 급감 → **수온약층**
3. 200m 이하 수온 변화 작음 → **심해층**
4. 혼합층 두께 비교 문제면: **바람이 강할수록 두꺼움** (혼합↑)

**함정**
- 여름에 혼합층이 얇고, 겨울에 두꺼움 (여름=일사량↑하지만 바람이 약함, 혼합층 얕음)
- 수온약층은 대류·혼합 억제 → **안정한 층** (대류 X)

**체크리스트**
- [ ] 위도별 혼합층 깊이 비교 (저위도<중위도)
- [ ] 혼합층 두께는 바람 세기에 비례

## 유형 2. T-S도 (수온-염분도)

**풀이 알고리즘**
1. 가로=염분, 세로=수온, **등밀도선은 우하향 대각선**
2. 수괴가 이동해 T 감소 + S 감소해도 밀도가 같을 수 있음
3. 화살표 방향으로 밀도 증가/감소/일정 판별

**함정**
- **밀도는 염분↑ & 수온↓일 때 커짐** (수온이 주된 영향, 단 저위도 표층은 염분 우세)
- 두 수괴 혼합 결과는 T-S도상 **두 점 잇는 직선 위의 한 점**

**체크리스트**
- [ ] 등밀도선 방향 (오른쪽 아래가 고밀도)
- [ ] 수괴 혼합 = 직선상

## 유형 3. 용존기체 분포 곡선

**풀이 알고리즘**
1. **산소**: 표층 최대(광합성+대기교환) → 1000m 최소(분해소비) → 심층 약간 회복
2. **이산화탄소**: 표층 최소 → 깊이 증가에 따라 **단조 증가** (분해·압력↑·수온↓)

**함정**
- 산소 곡선의 1000m 최저점을 "심해층"과 혼동 금지
- 차가운 물이 용존도 ↑ (저위도보다 고위도 표층이 기체 더 녹음)

## 유형 4. 난류 vs 한류 비교

**풀이 알고리즘 (암기표)**

| 항목 | 난류 | 한류 |
| --- | --- | --- |
| 수온 | 높음 | 낮음 |
| 염분 | 높음 | 낮음 |
| 용존산소 | 낮음 | **높음** |
| 영양염 | 낮음 | **높음** |
| 투명도 | 높음 | 낮음 |
| 플랑크톤 | 적음 | 많음 |

**함정**
- "난류가 염분 높다" 외우기. 한류가 염분 낮은 이유: 극지방은 강수·융빙수 영향
- **조경수역** = 난·한류 만나는 곳 → 좋은 어장 (영양염 + 산소)

## 유형 5. 한랭전선 vs 온난전선 비교

**풀이 알고리즘 (암기표)**

| 항목 | 한랭전선 | 온난전선 |
| --- | --- | --- |
| 전선면 기울기 | 급 | 완만 |
| 이동속도 | 빠름 | 느림 |
| 구름 | 적운형 (Cb, Cu) | 층운형 (Ns, As, Cs) |
| 강수범위 | 좁고 뒤쪽 | 넓고 앞쪽 |
| 강수형태 | 소나기·뇌우 | 지속적 약한 비 |
| 통과 후 기온 | 하강 | 상승 |
| 통과 후 기압 | 상승 | 하강 |
| 통과 후 풍향 | 남서→북서 (시계방향) | 남동→남서 (시계방향) |

**함정**
- **온난전선도 풍향이 시계방향**으로 변한다 (둘 다 북반구 저기압 남쪽 반원이기 때문!)
- 강수범위가 전선 "앞이냐 뒤냐": 한랭=뒤, 온난=앞

## 유형 6. 저기압 통과 시 풍향 변화

**규칙**
- **저기압 중심이 북쪽을 지나감** → 관측자는 남쪽 반원 → 풍향 **시계방향**
- **저기압 중심이 남쪽을 지나감** → 관측자는 북쪽 반원 → 풍향 **반시계방향**

**풀이 알고리즘**
1. 시간에 따른 풍향 순서를 본다 (예: SE→S→SW)
2. 나침반 위에서 시계/반시계 판별
3. 규칙에 따라 중심 위치 결정

## 유형 7. 위성 영상 (가시·적외·수증기)

**식별 알고리즘**

| 영상 | 밝음이 의미하는 것 | 키워드 |
| --- | --- | --- |
| **가시** | **두께가 두껍다** (반사율↑) | 밤에는 보이지 않음 |
| **적외** | **온도가 낮다 = 고도가 높다** (방출↓) | 상층운 구분 가능 |
| **수증기** | 상층 수증기 많음 | 맑고 건조한 영역 어둡게 |

**함정**
- 적외영상에서 **낮은 구름(Cu, St)은 어둡게 보임**, 높은 구름(Ci, Cb)은 밝게 보임
- 가시영상은 두꺼운 구름뿐 아니라 **눈·얼음**도 밝게 보임

---

## 유형 8. 태풍 위험반원/가항반원

**풀이 알고리즘**
1. 태풍 진행방향 화살표 확인
2. 진행방향 기준 **오른쪽 = 위험반원**, 왼쪽 = 가항반원
3. 위험반원: 태풍 풍속 + 진행속도 **합** → 풍속↑, 풍향 **시계방향** 회전
4. 가항반원: 태풍 풍속 – 진행속도 → 풍속↓, 풍향 **반시계방향** 회전

**함정**
- "위험반원에서 풍향이 시계방향으로 변한다"만 외우고 나머지는 유도
- 한반도는 태풍이 서해 or 남해로 통과하면 **위험반원에 잘 들어감**

---

## 유형 9. 에크만 수송·용승

**풀이 알고리즘**
1. 바람 방향 식별
2. 북반구 → 바람 기준 **오른쪽 90°**로 해수 수송 (남반구는 왼쪽)
3. 수송 결과 **해수가 밀려 나가면 → 용승** / **모이면 → 침강**

**연안 용승 예시 (페루 연안)**: 남풍 → (남반구이므로 왼쪽 90°) → 외해 방향 수송 → 연안 용승 → 수온↓, 영양염↑

**함정**
- 에크만 수송은 표면 뿐 아니라 **전 수심의 평균** 방향
- 적도 용승은 남북 무역풍이 적도 양쪽으로 수송 → 적도 표층 발산 → 깊은 곳 물 올라옴

## 유형 10. 엘니뇨 vs 라니냐

**풀이 알고리즘 (동태평양 기준)**

| 항목 | 평상시 | 엘니뇨 | 라니냐 |
| --- | --- | --- | --- |
| 무역풍 | 정상 | **약화** | 강화 |
| 동태평양 수온 | 낮음 | **상승** | 더 낮음 |
| 서태평양 수온 | 높음 | 낮아짐 | 더 높음 |
| 동태평양 수온약층 | 얕음 | **깊어짐** | 더 얕음 |
| 동태평양 해수면 | 낮음 | **상승** | 더 낮음 |
| 동태평양 강수 | 적음 | **많음** (홍수) | 적음 (가뭄) |
| 서태평양 강수 | 많음 | 적음 (가뭄) | 많음 (홍수) |
| 용승 (동) | 활발 | **약화** | 강화 |
| 워커순환 | 정상 | 약화/역전 | 강화 |

**함정**
- "엘니뇨=동태평양이 **뜨거워짐**" → 그 결과 모든 변화가 따라옴
- 문제에 "평년과 비교"가 있으면 **편차(이상)** 개념: (+)=평년보다 큼, (-)=작음

## 유형 11. 밀란코비치 3요소

| 요소 | 주기 | 변화 |
| --- | --- | --- |
| **자전축 기울기** | 41,000년 | 22.1° ↔ 24.5° |
| **세차운동** | 26,000년 | 자전축 회전 |
| **이심률** | 10만년 | 궤도 원↔타원 |

**풀이 알고리즘**
- 기울기 ↑ → 계절 차이 ↑
- 세차운동 → 북반구 여름이 근일점 or 원일점에서 일어나는가
- 이심률 ↑ → 계절 차이 (반구별 비대칭) ↑

**함정**
- 세차운동 주기 **26,000년** (2만 6천), 이심률 **10만년**, 기울기 **4만 1천년** — 숫자 혼동 주의
- 현재 북반구는 근일점(1월) = 겨울 → 근일점·원일점 문제 주의

## 유형 12. 지사학 5대 법칙 + 지질 단면도 해석

**순서 결정 알고리즘**
1. **수평퇴적**: 처음에 수평으로 쌓였다
2. **누중**: 아래가 오래됨
3. **관입**: 관입암은 주변보다 젊음
4. **부정합**: 시간적 공백 (경사부정합 주의)

**전형적 단면도 문제 풀이 순서**
1. 퇴적층 쌓인 순서 (아래→위)
2. 관입 / 단층 언제 발생? (자른 층보다 젊음)
3. 부정합면 찾기 (경사 각도 다른 층 사이)
4. 융기·침식 이벤트 끼워 넣기

## 유형 14. 반감기 계산

**공식**
- N = N₀ × (1/2)^n,  n = 경과시간 / 반감기

**모원소 : 자원소 비율 암기표**

| 경과 반감기 수 | 모 : 자 | 모원소 비율 |
| --- | --- | --- |
| 0 | 1 : 0 | 100% |
| 1 | 1 : 1 | 50% |
| 2 | 1 : 3 | 25% |
| 3 | 1 : 7 | 12.5% |
| 4 | 1 : 15 | 6.25% |

**풀이 알고리즘**
1. 모:자 비 → 반감기 수 n
2. 반감기 × n = 암석 나이
3. 암석의 **생성 나이**는 관입/화성암 고화 시점

**함정**
- 퇴적암은 **구성 광물의 나이가 퇴적 시기가 아님** → 퇴적암 연령 측정은 관입·부정합 이용
- 반감기 다른 동위원소 (C-14 / U-238) 적용 범위 차이

## 유형 15. 4대 퇴적구조 + 역전 판단

| 구조 | 형태 | 지층 역전 판단 |
| --- | --- | --- |
| **점이층리** | 아래 굵고 위로 갈수록 세립 | **정상=아래굵음** / 역전=위가 굵음 |
| **사층리** | 경사진 층리 | **정상=경사층 아래가 절단** / 역전=위 |

**풀이 알고리즘**
1. 구조 식별
2. "위쪽 표시"와 현재 지층 방향 비교
3. 불일치하면 **역전 지층**

## 유형 16. 표준화석·시상화석

**표준화석 (시대 지시)**

| 지질시대 | 대표 화석 |
| --- | --- |
| 고생대 | 삼엽충, 방추충(푸줄리나), 갑주어, 필석 |
| 중생대 | 암모나이트, 공룡, 시조새 |
| 신생대 | 화폐석(누물라이트), 매머드 |

**시상화석 (환경 지시)**: 산호=얕고 따뜻한 바다, 고사리=따뜻·습한 육지, 조개=연안

**함정**
- **방추충 = 고생대 페름기**, 신생대 화폐석과 모양 비슷하지만 시대 다름 (둘 다 원생동물이지만 시대 구분 필수)
- 시상화석은 **생존기간 길고 환경제한적**, 표준화석은 **생존기간 짧고 분포 넓음**

## 유형 17. 산소동위원소와 고기후

**원리**
- 물 H₂¹⁶O는 H₂¹⁸O보다 가벼움 → 증발 시 ¹⁶O가 먼저 올라감
- 빙기에는 ¹⁶O가 대륙 빙하에 저장 → 해수는 **¹⁸O/¹⁶O 비 증가**
- 해양 유공충 껍데기 δ¹⁸O ↑ → 빙기 (차가움)
- 빙하 코어 δ¹⁸O ↑ → 간빙기 (따뜻함) **※ 부호 반대**

**풀이 알고리즘**
1. 시료가 **해양 퇴적물**인지 **빙하 코어**인지 확인
2. 해양 퇴적물 δ¹⁸O ↑ = **추웠음**
3. 빙하 코어 δ¹⁸O ↑ = **따뜻했음**

**함정**
- 해양과 빙하의 부호 반대 **반드시 구분**

## 서논술형 대비 — 3줄 정답 템플릿

1. **엘니뇨 서논술**:
   "무역풍이 약화되어 (1) 동태평양 용승이 감소하고 (2) 서태평양 고온 해수가 동쪽으로 이동하여 (3) 동태평양 해수면 온도와 해수면 높이가 상승한다."

2. **한랭전선 통과 시 날씨 변화**:
   "통과 전 남서풍·기온 높음 → 통과 시 적운형 구름과 소나기 → 통과 후 북서풍·기온 하강·기압 상승."

3. **부정합 형성**:
   "수평 퇴적 → 지각 변동으로 융기 → 오랜 침식 → 다시 침강하여 새로운 지층 퇴적. 경사부정합은 중간에 습곡·경동이 추가됨."

4. **밀란코비치 자전축 기울기**:
   "41,000년 주기로 22.1°~24.5° 변화. 기울기가 커지면 여름은 더 덥고 겨울은 더 추워 계절 차이가 심해진다."
